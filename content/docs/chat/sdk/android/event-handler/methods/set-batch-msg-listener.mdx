Use `SetBatchMsgListener` for set batch msg listener in the OpenIM Android SDK.

## Feature

Registers or replaces the listener used to receive set batch msg listener. Register listeners before login when you need login-time synchronization events.

Initialize the SDK before calling user-scoped APIs, and pass a unique `operationID` whenever the core binding signature includes it so client and server logs can be correlated.

## Core Binding Method

| Item             | Description                                        |
| ---------------- | -------------------------------------------------- |
| Method signature | `SetBatchMsgListener(listener OnBatchMsgListener)` |
| Capability       | Event listeners                                    |
| Source file      | `open_im_sdk/listener.go`                          |

## Input Parameters

| Parameter  | Type                                                                            | Description              |
| ---------- | ------------------------------------------------------------------------------- | ------------------------ |
| `listener` | [On Batch Msg Listener](/docs/chat/sdk/v4/android/models/on-batch-msg-listener) | Event listener instance. |



## Return Value

The result is returned through the callback. Success is delivered through `OnSuccess`, failure through `OnError`, and sending or upload methods may also report progress.
