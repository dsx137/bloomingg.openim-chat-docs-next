Use `GetFriendList` for get friend list in the OpenIM Android SDK.

## Feature

Reads get friend list from the local SDK state or OpenIM service according to the method contract.

Initialize the SDK before calling user-scoped APIs, and pass a unique `operationID` whenever the core binding signature includes it so client and server logs can be correlated.

## Core Binding Method

| Item             | Description                                                          |
| ---------------- | -------------------------------------------------------------------- |
| Method signature | `GetFriendList(callback Base, operationID string, filterBlack bool)` |
| Capability       | Users, friends, and blocklist                                        |
| Source file      | `open_im_sdk/relation.go`                                            |

## Android Call Reference

The Java method prototype in Android SDK v3.8.3 is shown below. The core binding signature explains the underlying `open_im_sdk` capability and parameter mapping.

```java showLineNumbers
public void getFriendList(OnBase<List<UserInfo>> base, Boolean filterBlack)
```
