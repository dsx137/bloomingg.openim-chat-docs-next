Use `CreateFileMessageByURL` for create file message by url in the OpenIM Android SDK.

## Feature

Creates the local message payload for create file message by url. Pass the returned message JSON to a sending method when you are ready to send it.

Initialize the SDK before calling user-scoped APIs, and pass a unique `operationID` whenever the core binding signature includes it so client and server logs can be correlated.

| Capability       | Conversations and messages                                               |
