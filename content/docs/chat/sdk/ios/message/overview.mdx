This page covers conversations and messages: conversation lists, conversation settings, typing state, message creation, sending, history, read status, deletion, search, and local messages.

## Main Flow

| Stage | Methods |
| ----- | ------- |
| Create message | Use `Create*Message` methods to create local message objects. |
| Send message | Send with [Send message](/docs/chat/sdk/v4/ios/message/sending-messages/send-message) or [Send message without OSS](/docs/chat/sdk/v4/ios/message/sending-messages/send-message-not-oss). |
| Receive message | Register message listeners from [Event handlers](/docs/chat/sdk/v4/ios/event-handler/overview). |
| Query history | Use history and local search APIs under retrieving messages. |
| Maintain read state | Mark conversation or message IDs as read. |
