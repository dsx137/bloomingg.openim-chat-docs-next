Use **Create a quote message** when your application needs this OpenIM capability from iOS Core SDK v3.8.3. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying core binding.

## Overview

Call `CreateQuoteMessage` through the platform wrapper to create a quote message. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.

## Core Binding Method

| Item             | Description                                                                  |
| ---------------- | ---------------------------------------------------------------------------- |
| Method signature | `CreateQuoteMessage(operationID string, text string, message string) string` |
| Capability       | Conversations and messages                                                   |
| Source location  | `open_im_sdk/conversation_msg.go`                                            |

## iOS Call Reference

iOS Core SDK v3.8.3 exposes the following platform method signature. The core binding signature above explains the underlying capability and parameter mapping.

Source:`Framework/OpenIMCore.xcframework/.../Open_im_sdk.objc.h`.

```objective-c showLineNumbers
FOUNDATION_EXPORT NSString* _Nonnull Open_im_sdkCreateQuoteMessage(NSString* _Nullable operationID, NSString* _Nullable text, NSString* _Nullable message);
```

## Parameters

| Parameter | Type       | Required   | Description                                     |
| --------- | ---------- | ---------- | ----------------------------------------------- |
| text      | NSString\* | As defined | Pass as defined.                                |
| message   | NSString\* | As defined | Message object, message content, or message ID. |

## Return Value

Synchronous creation methods return a value directly. Asynchronous methods report success or failure through the callback. Pass a traceable operationID for each call.

## Code Example

The following example shows the iOS Core SDK v3.8.3 call shape. Replace placeholder values with real business data and add production error handling.
