Use **Create a sound message by URL** when your application needs this OpenIM capability from iOS Core SDK v3.8.3. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying core binding.

## Overview

Call `CreateSoundMessageByURL` through the platform wrapper to create a sound message by url. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.

## Core Binding Method

| Item             | Description                                                                |
| ---------------- | -------------------------------------------------------------------------- |
| Method signature | `CreateSoundMessageByURL(operationID string, soundBaseInfo string) string` |
| Capability       | Conversations and messages                                                 |
| Source location  | `open_im_sdk/conversation_msg.go`                                          |
