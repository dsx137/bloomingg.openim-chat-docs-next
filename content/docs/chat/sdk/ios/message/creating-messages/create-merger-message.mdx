Use **Create Merger Message** when your application needs this OpenIM capability from iOS Core SDK v3.8.3. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying core binding.

## Overview

Call `CreateMergerMessage` through the platform wrapper to create merger message. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.

## Core Binding Method

| Item             | Description                                                                              |
| ---------------- | ---------------------------------------------------------------------------------------- |
| Method signature | `CreateMergerMessage(operationID string, messageList, title, summaryList string) string` |
| Capability       | Conversations and messages                                                               |
| Source location  | `open_im_sdk/conversation_msg.go`                                                        |

## iOS Call Reference

iOS Core SDK v3.8.3 exposes the following platform method signature. The core binding signature above explains the underlying capability and parameter mapping.

FOUNDATION_EXPORT NSString* _Nonnull Open_im_sdkCreateMergerMessage(NSString* _Nullable operationID, NSString* _Nullable messageList, NSString* _Nullable title, NSString* _Nullable summaryList);
