Use **Send message** when your application needs this OpenIM capability from iOS Core SDK v3.8.3. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying core binding.

## Overview

Call `SendMessage` through the platform wrapper to send message. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.

## Core Binding Method

| Item             | Description                                                                                                               |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------- |
| Method signature | `SendMessage(callback SendMsgCallBack, operationID, message, recvID, groupID, offlinePushInfo string, isOnlineOnly bool)` |
| Capability       | Conversations and messages                                                                                                |
| Source location  | `open_im_sdk/conversation_msg.go`                                                                                         |

## iOS Call Reference

iOS Core SDK v3.8.3 exposes the following platform method signature. The core binding signature above explains the underlying capability and parameter mapping.

Source:`Framework/OpenIMCore.xcframework/.../Open_im_sdk.objc.h`.

```objective-c showLineNumbers
FOUNDATION_EXPORT void Open_im_sdkSendMessage(id<Open_im_sdk_callbackSendMsgCallBack> _Nullable callback, NSString* _Nullable operationID, NSString* _Nullable message, NSString* _Nullable recvID, NSString* _Nullable groupID, NSString* _Nullable offlinePushInfo, BOOL isOnlineOnly);
```

## Parameters

| Parameter       | Type                                          | Required   | Description                                                  |
| --------------- | --------------------------------------------- | ---------- | ------------------------------------------------------------ |
| callback        | `id<Open_im_sdk_callbackSendMsgCallBack>` | Yes        | Callback or listener.                                        |
| message         | NSString\*                                    | As defined | Message object, message content, or message ID.              |
| recvID          | NSString\*                                    | As defined | Pass as defined.                                             |
| groupID         | NSString\*                                    | As defined | Group ID or list of group IDs.                               |
| offlinePushInfo | NSString\*                                    | As defined | Request object. Use the fields defined by the platform type. |
| isOnlineOnly    | BOOL                                          | As defined | Pass as defined.                                             |

## Return Value

Synchronous creation methods return a value directly. Asynchronous methods report success or failure through the callback. Pass a traceable operationID for each call.
