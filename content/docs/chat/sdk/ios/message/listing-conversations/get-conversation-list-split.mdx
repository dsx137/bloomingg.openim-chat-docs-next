Use **Get conversations by page** when your application needs this OpenIM capability from iOS Core SDK v3.8.3. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying core binding.

## Overview

Call `GetConversationListSplit` through the platform wrapper to get conversations by page. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.
