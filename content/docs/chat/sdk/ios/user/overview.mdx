This page covers user profile, user command, presence subscription, friend, and blacklist APIs. Each method has its own page for parameters, returns, and usage boundaries.

## User Profile And Presence

| Method | Description |
| ------ | ----------- |
| [Get self user info](/docs/chat/sdk/v4/ios/user/methods/get-self-user-info) | Get the current user's profile for profile pages, startup refresh, or account switching. |
| [Update self info](/docs/chat/sdk/v4/ios/user/methods/set-self-info) | Update the current user's profile, such as nickname, avatar, or extension fields. |
| [Get users info](/docs/chat/sdk/v4/ios/user/methods/get-users-info) | Batch query public user profiles for conversation lists, message sender info, or friend refresh. |
| [Add user command](/docs/chat/sdk/v4/ios/user/methods/add-user-command) | Add a lightweight user-side command or marker. |
| [Delete user command](/docs/chat/sdk/v4/ios/user/methods/delete-user-command) | Delete a user command by type and UUID. |
| [Get all user commands](/docs/chat/sdk/v4/ios/user/methods/get-all-user-commands) | Get all user commands under a specified type. |
| [Subscribe users status](/docs/chat/sdk/v4/ios/user/methods/subscribe-users-status) | Subscribe to online status for a group of users. |
| [Unsubscribe users status](/docs/chat/sdk/v4/ios/user/methods/unsubscribe-users-status) | Stop receiving online status for specified users. |
| [Get subscribed user status](/docs/chat/sdk/v4/ios/user/methods/get-subscribe-users-status) | Get the current status snapshot for subscribed users. |
| [Get user status](/docs/chat/sdk/v4/ios/user/methods/get-user-status) | Query online status for specified users. |
