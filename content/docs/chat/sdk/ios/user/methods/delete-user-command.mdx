Use **Delete user command** when your application needs this OpenIM capability from iOS Core SDK v3.8.3. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying core binding.

## Overview

Call `DeleteUserCommand` through the platform wrapper to delete user command. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.

## Core Binding Method

| Item             | Description                                                                     |
| ---------------- | ------------------------------------------------------------------------------- |
| Method signature | `DeleteUserCommand(callback Base, operationID string, Type int32, uuid string)` |
| Capability       | User                                                                            |
| Source location  | `open_im_sdk/user.go`                                                           |
