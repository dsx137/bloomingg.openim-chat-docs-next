Use **Check friendship** when your application needs this OpenIM capability from uni-app SDK v3.8.3. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying core binding.

## Overview

Call `CheckFriend` through the platform wrapper to check friendship. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.

| ---------------- | ------------------------------------------------------------------- |

| Capability       | Friends                                                             |
