Use **Create a group** when your application needs this OpenIM capability from uni-app SDK v3.8.3. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying core binding.

## Overview

Call `CreateGroup` through the platform wrapper to create a group. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.

| ---------------- | --------------------------------------------------------------------- |

| Capability       | Group                                                                 |

openIM.createGroup(operationID, groupBaseInfo, callback);

Synchronous creation methods return a value directly. Asynchronous methods report success or failure through the callback. Pass a traceable operationID for each call.
