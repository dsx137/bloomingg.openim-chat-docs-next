Use **Get self user info** when your application needs this OpenIM capability from iOS Core SDK 3.8.3-hotfix.14-dynamic.1. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying core binding.

## Overview

Call `GetSelfUserInfo` through the platform wrapper to get self user info. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.
