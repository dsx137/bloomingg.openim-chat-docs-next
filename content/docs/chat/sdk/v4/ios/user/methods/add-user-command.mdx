Use **Add user command** when your application needs this OpenIM capability from iOS Core SDK 3.8.3-hotfix.14-dynamic.1. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying core binding.

## Overview

Call `AddUserCommand` through the platform wrapper to add user command. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.

## Core Binding Method

| Item             | Description                                                                                |
| ---------------- | ------------------------------------------------------------------------------------------ |
| Method signature | `AddUserCommand(callback Base, operationID string, Type int32, uuid string, value string)` |
| Capability       | User                                                                                       |
