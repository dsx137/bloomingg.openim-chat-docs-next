Use **Refuse group application** when your application needs this OpenIM capability from iOS Core SDK 3.8.3-hotfix.14-dynamic.1. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying core binding.

## Overview

Call `RefuseGroupApplication` through the platform wrapper to refuse group application. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.

## Core Binding Method

| Item             | Description                                                                                                      |
| ---------------- | ---------------------------------------------------------------------------------------------------------------- |
| Method signature | `RefuseGroupApplication(callback Base, operationID string, groupID string, fromUserID string, handleMsg string)` |
| Capability       | Group                                                                                                            |
| Source location  | `open_im_sdk/group.go`                                                                                           |
