Use **Join a group** when your application needs this OpenIM capability from iOS Core SDK v3.8.3. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying core binding.

## Overview

Call `JoinGroup` through the platform wrapper to join a group. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.

## Core Binding Method

| Item             | Description                                                                                                |
| ---------------- | ---------------------------------------------------------------------------------------------------------- |
| Method signature | `JoinGroup(callback Base, operationID string, groupID string, reqMsg string, joinSource int32, ex string)` |
| Capability       | Group                                                                                                      |
| Source location  | `open_im_sdk/group.go`                                                                                     |

## iOS Call Reference

Synchronous creation methods return a value directly. Asynchronous methods report success or failure through the callback. Pass a traceable operationID for each call.
