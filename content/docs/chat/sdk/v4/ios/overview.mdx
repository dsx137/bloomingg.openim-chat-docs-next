## Overview

OpenIM iOS SDK `3.8.3-hotfix.12` adds OpenIM messaging to iOS applications. The Objective-C SDK is built on OpenIM SDK Core and exposes `OIMManager` APIs for connection management, login, conversations, friends, groups, messages, and users.

The SDK supports iOS 11.0 and later.

## Install with CocoaPods

Add OpenIMSDK to your `Podfile`:

```ruby
pod 'OpenIMSDK', '3.8.3-hotfix.12'
```

Then install the dependency:

```bash
pod install
```

Import the SDK where it is used:

```objc
@import OpenIMSDK;
```

## Initialize and log in

Initialize the SDK before logging in, and register connection callbacks at initialization time.

```objc
OIMInitConfig *config = [OIMInitConfig new];
config.apiAddr = @"https://your-api-address";
config.wsAddr = @"wss://your-websocket-address";
config.objectStorage = @"";

BOOL initialized = [OIMManager.manager initSDKWithConfig:config
                                            onConnecting:^{
  // The SDK is connecting.
} onConnectFailure:^(NSInteger code, NSString * _Nullable message) {
  // Handle a failed connection.
} onConnectSuccess:^{
  // The SDK is connected.
} onKickedOffline:^{
  // The current session was kicked offline.
} onUserTokenExpired:^{
  // Obtain a new user token and log in again.
}];

if (initialized) {
  [OIMManager.manager login:userID
                      token:token
                  onSuccess:^(NSString * _Nullable data) {
    // The user is logged in.
  } onFailure:^(NSInteger code, NSString * _Nullable message) {
    // Handle login failure.
  }];
}
```

Obtain the `userID` and `token` from a trusted business backend. Do not embed administrator credentials or token-generation secrets in the app.

## Send a text message

Create a message before sending it. For a direct message, set `recvID` and leave `groupID` empty. For a group message, set `groupID` and leave `recvID` empty.

```objc
OIMMessageInfo *message = [OIMMessageInfo createTextMessage:@"Hello, OpenIM!"];

[OIMManager.manager sendMessage:message
                          recvID:receiverUserID
                         groupID:@""
                 offlinePushInfo:nil
                       onSuccess:^(OIMMessageInfo * _Nullable sentMessage) {
  // Replace the local message with sentMessage.
} onProgress:^(NSInteger progress) {
  // Update upload progress when applicable.
} onFailure:^(NSInteger code, NSString * _Nullable error) {
  // Handle send failure.
}];
```

Register the advanced message listener before login so the app can receive incoming messages and message-state callbacks.
