Logger covers writing client logs and uploading SDK logs. Offline push and file upload are documented in separate modules.

## Boundaries

| Scenario | Description |
| -------- | ----------- |
| Log configuration | Configure log level and log directory during SDK initialization. |
| Write client logs | Let the app layer write important IM operations into the SDK log stream. |
| Upload logs | Upload SDK logs for support tickets or production issue analysis, and correlate them with operationID values. |
