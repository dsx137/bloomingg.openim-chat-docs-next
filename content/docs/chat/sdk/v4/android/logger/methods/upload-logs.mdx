Use `UploadLogs` for upload logs in the OpenIM Android SDK.

## Feature

UploadLogs provides the Android SDK operation for upload logs.

Initialize the SDK before calling user-scoped APIs, and pass a unique `operationID` whenever the core binding signature includes it so client and server logs can be correlated.

## Core Binding Method

| Item             | Description                                                                                      |
| ---------------- | ------------------------------------------------------------------------------------------------ |
| Method signature | `UploadLogs(callback Base, operationID string, line int, ex string, progress UploadLogProgress)` |
| Capability       | Logging                                                                                          |
| Source file      | `open_im_sdk/third.go`                                                                           |

## Android Call Reference

The Java method prototype in Android SDK v3.8.3 is shown below. The core binding signature explains the underlying `open_im_sdk` capability and parameter mapping.

```java showLineNumbers
public void uploadLogs(OnBase<String> base, List<String> params, int line, String ex, UploadLogProgress progress)
```

## Input Parameters

| Parameter  | Type                                                                        | Description                                               |
| ---------- | --------------------------------------------------------------------------- | --------------------------------------------------------- |
| `base`     | `OnBase<String>`                                                            | Async callback for success and failure results.           |
| `params`   | `List<String>`                                                              | Request parameters, usually encoded as JSON.              |
| `line`     | int                                                                         | Parameter passed according to the core binding signature. |
| `ex`       | String                                                                      | Parameter passed according to the core binding signature. |
| `progress` | [Upload Log Progress](/docs/chat/sdk/v4/android/models/upload-log-progress) | Parameter passed according to the core binding signature. |



## Return Value

The result is returned through the callback. Success is delivered through `OnSuccess`, failure through `OnError`, and sending or upload methods may also report progress.
