Use `GetSpecifiedGroupMembersInfo` for get specified group members info in the OpenIM Android SDK.

## Feature

Reads get specified group members info from the local SDK state or OpenIM service according to the method contract.

Initialize the SDK before calling user-scoped APIs, and pass a unique `operationID` whenever the core binding signature includes it so client and server logs can be correlated.
