Use `JoinGroup` for join group in the OpenIM Android SDK.

## Feature

JoinGroup provides the Android SDK operation for join group.

Initialize the SDK before calling user-scoped APIs, and pass a unique `operationID` whenever the core binding signature includes it so client and server logs can be correlated.

## Core Binding Method

| Item             | Description                                                                                                |
| ---------------- | ---------------------------------------------------------------------------------------------------------- |
| Method signature | `JoinGroup(callback Base, operationID string, groupID string, reqMsg string, joinSource int32, ex string)` |
| Capability       | Groups                                                                                                     |
| Source file      | `open_im_sdk/group.go`                                                                                     |

## Android Call Reference

The Java method prototype in Android SDK v3.8.3 is shown below. The core binding signature explains the underlying `open_im_sdk` capability and parameter mapping.

```java showLineNumbers
public void joinGroup(OnBase<String> base, String gid, String reason, int joinSource)
public void joinGroup(OnBase<String> base, String gid, String reason, int joinSource, String ex)
```

## Input Parameters

| Parameter    | Type                                                          | Description                                               |
| ------------ | ------------------------------------------------------------- | --------------------------------------------------------- |
| `base`       | [On Base](/docs/chat/sdk/v4/android/models/on-base)`<String>` | Async callback for success and failure results.           |
| `gid`        | `String`                                                      | Parameter passed according to the core binding signature. |
| `reason`     | `String`                                                      | Parameter passed according to the core binding signature. |
| `joinSource` | `int`                                                         | Parameter passed according to the core binding signature. |
| `ex`         | `String`                                                      | Parameter passed according to the core binding signature. |



## Return Value

The result is returned through the callback. Success is delivered through `OnSuccess`, failure through `OnError`, and sending or upload methods may also report progress.
