Use `GetAllConversationList` for get all conversation list in the OpenIM Android SDK.

## Feature

Reads get all conversation list from the local SDK state or OpenIM service according to the method contract.

Initialize the SDK before calling user-scoped APIs, and pass a unique `operationID` whenever the core binding signature includes it so client and server logs can be correlated.
