Use `HideAllConversations` for hide all conversations in the OpenIM Android SDK.

## Feature

HideAllConversations provides the Android SDK operation for hide all conversations.

Initialize the SDK before calling user-scoped APIs, and pass a unique `operationID` whenever the core binding signature includes it so client and server logs can be correlated.

## Core Binding Method

| Item             | Description                                               |
| ---------------- | --------------------------------------------------------- |
| Method signature | `HideAllConversations(callback Base, operationID string)` |
| Capability       | Conversations and messages                                |
| Source file      | `open_im_sdk/conversation_msg.go`                         |

## Input Parameters

| Parameter | Type                                                | Description                                     |
| --------- | --------------------------------------------------- | ----------------------------------------------- |
| `base`    | [On Base](/docs/chat/sdk/v4/android/models/on-base) | Async callback for success and failure results. |



## Return Value

The result is returned through the callback. Success is delivered through `OnSuccess`, failure through `OnError`, and sending or upload methods may also report progress.
