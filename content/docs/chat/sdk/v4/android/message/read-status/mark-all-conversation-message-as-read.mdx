Use `MarkAllConversationMessageAsRead` for mark all conversation message as read in the OpenIM Android SDK.

## Feature

MarkAllConversationMessageAsRead provides the Android SDK operation for mark all conversation message as read.

Initialize the SDK before calling user-scoped APIs, and pass a unique `operationID` whenever the core binding signature includes it so client and server logs can be correlated.
