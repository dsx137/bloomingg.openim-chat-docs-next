Use `CreateCardMessage` for create card message in the OpenIM Android SDK.

## Feature

Creates the local message payload for create card message. Pass the returned message JSON to a sending method when you are ready to send it.

Initialize the SDK before calling user-scoped APIs, and pass a unique `operationID` whenever the core binding signature includes it so client and server logs can be correlated.

## Core Binding Method

| Item             | Description                                                     |
| ---------------- | --------------------------------------------------------------- |
| Method signature | `CreateCardMessage(operationID string, cardInfo string) string` |
| Capability       | Conversations and messages                                      |
| Source file      | `open_im_sdk/conversation_msg.go`                               |
