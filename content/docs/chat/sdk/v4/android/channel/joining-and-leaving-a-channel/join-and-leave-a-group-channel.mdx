## Overview

Use `joinGroup` to request membership and `quitGroup` to leave an OpenIM group.

## How this maps to OpenIM

| Concept | OpenIM API | Notes |
| --- | --- | --- |
| Group channel | `gid` | The group ID. |
| Join | `joinGroup` | Includes a reason and join source. |
| Leave | `quitGroup` | Requires only the group ID. |

## Prerequisites

- An initialized, logged-in client.
- The target group ID.

## Install and configure

Follow the [Android overview](/docs/chat/sdk/v4/android/overview).

## Initialize the SDK

Initialize and log in before calling group APIs.

## Implement the task

### 1. Join a group

```java
OpenIMClient.getInstance().groupManager.joinGroup(callback, groupId, reason, joinSource);
```

The SDK source documents join source values `2` for invitation, `3` for search, and `4` for QR code.

### 2. Leave a group

```java
OpenIMClient.getInstance().groupManager.quitGroup(callback, groupId);
```

### 3. Handle the result

Handle `OnBase<String>.onSuccess` and `onError` for both operations.

## Verify the result

- Refresh the joined group list after a successful operation.
- Confirm group membership from another signed-in client when applicable.

## Troubleshooting

| Case | Cause | Handling |
| --- | --- | --- |
| Join fails | The server did not accept the membership request. | Handle `onError` and verify the group ID and join source. |
