Use `SetUserListener` for set user listener in the OpenIM Android SDK.

## Feature

Registers or replaces the listener used to receive set user listener. Register listeners before login when you need login-time synchronization events.

Initialize the SDK before calling user-scoped APIs, and pass a unique `operationID` whenever the core binding signature includes it so client and server logs can be correlated.

## Core Binding Method

| Item             | Description                                |
| ---------------- | ------------------------------------------ |
| Method signature | `SetUserListener(listener OnUserListener)` |
| Capability       | Event listeners                            |
| Source file      | `open_im_sdk/listener.go`                  |

## Android Call Reference

The Java method prototype in Android SDK v3.8.3 is shown below. The core binding signature explains the underlying `open_im_sdk` capability and parameter mapping.

```java showLineNumbers
public void setOnUserListener(OnUserListener listener)
```

## Input Parameters

| Parameter  | Type                                                                  | Description                                               |
| ---------- | --------------------------------------------------------------------- | --------------------------------------------------------- |
| `listener` | [On User Listener](/docs/chat/sdk/v4/android/models/on-user-listener) | Parameter passed according to the core binding signature. |



## Return Value

The result is returned through the callback. Success is delivered through `OnSuccess`, failure through `OnError`, and sending or upload methods may also report progress.

## Code Example

The following example uses the Android SDK v3.8.3 call style.

```java showLineNumbers
OpenIMClient.getInstance().userInfoManager.setOnUserListener(new OnUserListener() {
    @Override
    public void onSelfInfoUpdated(UserInfo info) {
        // TODO: handle the callback result
    }

    @Override
    public void onUserStatusChanged(UsersOnlineStatus onlineStatus) {
        // TODO: handle the callback result
    }
});
```
