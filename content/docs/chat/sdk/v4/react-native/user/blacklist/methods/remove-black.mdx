Use **Remove from blacklist** when your application needs this OpenIM capability from React Native SDK v3.8.3-patch.12.4. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying core binding.

## Overview

Call `RemoveBlack` through the platform wrapper to remove from blacklist. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.

| Parameter   | Type   | Required | Description                  |

Returns `Promise`. If the call fails, the Promise rejects and the error is wrapped as OpenIMApiError.
