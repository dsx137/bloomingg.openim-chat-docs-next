Use **Set Advanced Msg Listener** when your application needs this OpenIM capability from React Native SDK v3.8.3-patch.12.4. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying core binding.

## Overview

Call `SetAdvancedMsgListener` through the platform wrapper to set advanced msg listener. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.

| Capability       | Event handlers                                           |
