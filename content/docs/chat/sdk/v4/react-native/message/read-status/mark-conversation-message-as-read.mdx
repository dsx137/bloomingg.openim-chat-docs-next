Use **Mark conversation messages as read** when your application needs this OpenIM capability from React Native SDK v3.8.3-patch.12.4. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying core binding.

## Overview

Call `MarkConversationMessageAsRead` through the platform wrapper to mark conversation messages as read. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.

## Core Binding Method

| Item             | Description                                                                               |
| ---------------- | ----------------------------------------------------------------------------------------- |
| Method signature | `MarkConversationMessageAsRead(callback Base, operationID string, conversationID string)` |
| Capability       | Conversations and messages                                                                |
| Source location  | `open_im_sdk/conversation_msg.go`                                                         |

## React Native Call Reference

React Native SDK v3.8.3-patch.12.4 exposes the following platform method signature. The core binding signature above explains the underlying capability and parameter mapping.

Source:`src/sdk.ts`.

```ts showLineNumbers
markConversationMessageAsRead(conversationID: string, operationID: string = id())
```

## Parameters

| Parameter      | Type   | Required | Description                                  |
| -------------- | ------ | -------- | -------------------------------------------- |
| conversationID | string | Yes      | Conversation ID or list of conversation IDs. |

## Return Value

Returns `Promise`. If the call fails, the Promise rejects and the error is wrapped as OpenIMApiError.

## Code Example

The following example shows the React Native SDK v3.8.3-patch.12.4 call shape. Replace placeholder values with real business data and add production error handling.

```ts showLineNumbers
import OpenIMSDK from '@openim/rn-client-sdk';

const result = await OpenIMSDK.markConversationMessageAsRead('conversationID');
```
