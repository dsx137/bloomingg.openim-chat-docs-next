Use **Generate conversation ID** when your application needs this OpenIM capability from React Native SDK v3.8.3-patch.12.4. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying core binding.

## Overview

Call `GetConversationIDBySessionType` through the platform wrapper to generate conversation id. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.

| Item             | Description                                                                                   |
| ---------------- | --------------------------------------------------------------------------------------------- |
| Method signature | `GetConversationIDBySessionType(operationID string, sourceID string, sessionType int) string` |
| Capability       | Conversations and messages                                                                    |
| Source location  | `open_im_sdk/conversation_msg.go`                                                             |
