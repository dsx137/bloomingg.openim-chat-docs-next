## Overview

Use `transferGroupOwner()` when the current group owner hands ownership to another group member.

## OpenIM model

Ownership is a group-member role. The new owner must already be a member of the group.

## Prerequisites

- Current user is the group owner.
- Target user is already in the group.

## API signatures

```ts
transferGroupOwner: (
  data: TransferGroupParams,
  operationID?: string,
) => Promise<WsResponse<unknown>>;
```

## Implementation

```ts
await OpenIM.transferGroupOwner({
  groupID,
  newOwnerUserID: 'user_b',
});
```

## Returned data and events

Refresh group and member state when `CbEvents.OnGroupMemberInfoChanged` or `CbEvents.OnGroupInfoChanged` fires.

## Verify the result

- Confirm the new owner role is visible in the member list.
- Confirm the previous owner no longer has owner-only actions.

## Troubleshooting

| Case | Cause | Handling |
| --- | --- | --- |
| Transfer fails | Current user is not owner or target is not a member | Verify role and target membership before calling. |
