Use **Get login status** when your application needs this OpenIM capability from WASM SDK v3.8.3. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying OpenIM core.

## Overview

Call `GetLoginStatus` through the platform wrapper to get login status. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.

## Underlying Capability

| Item             | Description                              |
| ---------------- | ---------------------------------------- |
| Method signature | `GetLoginStatus(operationID string) int` |
| Capability       | SDK lifecycle                 |
| Source location  | `open_im_sdk/init_login.go`              |
