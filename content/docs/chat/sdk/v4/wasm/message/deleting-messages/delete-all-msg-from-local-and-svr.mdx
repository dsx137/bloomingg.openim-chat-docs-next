Use **Delete All Msg From Local And Svr** when your application needs this OpenIM capability from WASM SDK v3.8.3. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying OpenIM core.

## Overview

Call `DeleteAllMsgFromLocalAndSvr` through the platform wrapper to delete all msg from local and svr. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.
