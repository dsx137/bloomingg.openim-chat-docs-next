## SDK Core Error Codes

### General Errors

| Code | Constant | Default message or source note | Description |
| ---- | -------- | ------------------------------ | ----------- |
| `10000` | `NetworkError` | `Network error` | Network error. |
| `10001` | `NetworkTimeoutError` | `Network timeout error` | Network timeout. |
| `10002` | `ArgsError` | `Invalid input arguments` | Invalid input arguments. |
| `10003` | `CtxDeadlineExceededError` | `Context deadline exceeded` | Context deadline exceeded. |
| `10004` | `ResourceLoadNotCompleteError` | `Resource initialization incomplete` | SDK resources are not fully initialized. |
| `10005` | `UnknownCode` | `Unrecognized code` | Unknown SDK error code. |
| `10006` | `SdkInternalError` | `Internal SDK error` | Internal SDK error. |
| `10007` | `NoUpdateError` | `No updates available` | No update is available. |
