Use **Set Group Listener** when your application needs this OpenIM capability from React Native SDK v3.8.3. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying core binding.

## Overview

Call `SetGroupListener` through the platform wrapper to set group listener. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.

## Core Binding Method

| Item             | Description                                  |
| ---------------- | -------------------------------------------- |
| Method signature | `SetGroupListener(listener OnGroupListener)` |
| Capability       | Event handlers                               |
| Source location  | `open_im_sdk/listener.go`                    |
