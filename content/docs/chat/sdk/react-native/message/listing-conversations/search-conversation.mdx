Use **Search conversations** when your application needs this OpenIM capability from React Native SDK v3.8.3. This page follows the platform SDK reference and keeps the platform call signature visible next to the underlying core binding.

## Overview

Call `SearchConversation` through the platform wrapper to search conversations. The current user usually needs to be initialized and logged in before calling business APIs. Keep `operationID` traceable so client and server logs can be matched during troubleshooting.

## Core Binding Method

| Item             | Description                                                                 |
| ---------------- | --------------------------------------------------------------------------- |
| Method signature | `SearchConversation(callback Base, operationID string, searchParam string)` |
| Capability       | Conversations and messages                                                  |
| Source location  | `open_im_sdk/conversation_msg.go`                                           |

Returns `Promise`. If the call fails, the Promise rejects and the error is wrapped as OpenIMApiError.
