`OnFriendshipListener` 是 React Native SDK 的监听器，完整类名为 `io.openim.react-native.sdk.listener.OnFriendshipListener`。好友、好友申请和黑名单变化监听器。

## 类型说明

好友关系监听

## 回调方法

| 方法                        | 参数                    | 描述             |
| --------------------------- | ----------------------- | ---------------- |
| onBlacklistAdded            | BlacklistInfo u         | 事件触发时回调。 |
| onBlacklistDeleted          | BlacklistInfo u         | 事件触发时回调。 |
| onFriendApplicationAccepted | FriendApplicationInfo u | 事件触发时回调。 |
| onFriendApplicationAdded    | FriendApplicationInfo u | 事件触发时回调。 |
| onFriendApplicationDeleted  | FriendApplicationInfo u | 事件触发时回调。 |
| onFriendApplicationRejected | FriendApplicationInfo u | 事件触发时回调。 |
| onFriendInfoChanged         | FriendInfo u            | 事件触发时回调。 |
| onFriendAdded               | FriendInfo u            | 事件触发时回调。 |
| onFriendDeleted             | FriendInfo u            | 事件触发时回调。 |

## 关联类型

- [好友申请信息](/docs/chat/sdk/v4/react-native/models/friend-application-info)（`FriendApplicationInfo`）：好友申请记录，包含申请人、接收人、处理状态和备注。
- [好友信息](/docs/chat/sdk/v4/react-native/models/friend-info)（`FriendInfo`）：好友资料，包含昵称、头像、备注和添加来源等信息。
- [黑名单用户信息](/docs/chat/sdk/v4/react-native/models/blacklist-info)（`BlacklistInfo`）：黑名单用户资料，用于黑名单列表和黑名单事件回调。
- [用户信息](/docs/chat/sdk/v4/react-native/models/user-info)（`UserInfo`）：用户资料对象。
