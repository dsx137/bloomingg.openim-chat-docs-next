`OnUserListener` 是 React Native SDK 的监听器，完整类名为 `io.openim.react-native.sdk.listener.OnUserListener`。当前用户资料与用户在线状态变化监听器。

## 回调方法

| 方法                | 参数                           | 描述                                                        |
| ------------------- | ------------------------------ | ----------------------------------------------------------- |
| onSelfInfoUpdated   | UserInfo info                  | 当前用户的资料发生了更新 可以在 UI 上更新自己的头像和昵称。 |
| onUserStatusChanged | UsersOnlineStatus onlineStatus | When the user's online status changes, call                 |

## 关联类型

- [用户信息](/docs/chat/sdk/v4/react-native/models/user-info)（`UserInfo`）：用户资料对象。
- [用户在线状态](/docs/chat/sdk/v4/react-native/models/users-online-status)（`UsersOnlineStatus`）：用户在线状态信息。
