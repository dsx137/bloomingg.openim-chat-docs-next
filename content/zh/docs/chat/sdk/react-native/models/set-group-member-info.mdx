`SetGroupMemberInfo` 是 React Native SDK 的数据模型，完整类名为 `io.openim.react-native.sdk.models.SetGroupMemberInfo`。设置群成员资料时使用的请求对象。

## 字段说明

| 字段名称  | TypeScript 类型 | 描述                                     |
| --------- | --------------- | ---------------------------------------- |
| groupID   | String          | 群ID                                     |
| userID    | String          | 群成员 ID                                |
| nickname  | String          | 群成员昵称                               |
| faceURL   | String          | 群成员头像                               |
| roleLevel | Integer         | 群成员角色，1 普通成员；2 群主；3 管理员 |
| ex        | String          | 扩展参数                                 |
