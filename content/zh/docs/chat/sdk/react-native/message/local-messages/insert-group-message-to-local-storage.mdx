向本地群聊会话插入一条消息，适合导入或补写群聊本地消息。

## 功能介绍

向本地群聊会话插入一条消息，适合导入或补写群聊本地消息。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                                                 |
| -------- | -------------------------------------------------------------------------------------------------------------------- |
| 方法签名 | `InsertGroupMessageToLocalStorage(callback Base, operationID string, message string, groupID string, sendID string)` |
| 所属能力 | 会话与消息                                                                                                           |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                                                    |

## React Native 调用参考

React Native SDK v3.8.3 的 TypeScript 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`src/sdk.ts`。

```ts showLineNumbers
insertGroupMessageToLocalStorage(params: InsertGroupMsgParams, operationID: string = id())
```

## 输入参数

| 参数名称 | 参数类型             | 是否必填 | 描述                                   |
| -------- | -------------------- | -------- | -------------------------------------- |
| params   | InsertGroupMsgParams | 是       | 请求对象，字段以对应平台类型定义为准。 |

## 返回结果

返回 `Promise`。调用失败时 Promise reject，错误会被封装为 OpenIMApiError。

## 代码示例

```ts showLineNumbers
import OpenIMSDK from '@openim/rn-client-sdk';

const result = await OpenIMSDK.insertGroupMessageToLocalStorage({});
```
