## 功能介绍

获取当前用户黑名单列表，适合隐私设置页、拦截列表或关系状态恢复。

黑名单新增和移除通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；屏蔽后的业务效果以当前服务端策略为准。

## Core Binding 方法

| 项目     | 说明                                              |
| -------- | ------------------------------------------------- |
| 方法签名 | `GetBlackList(callback Base, operationID string)` |
| 所属能力 | 黑名单                                            |
| 源码位置 | `open_im_sdk/relation.go`                         |
