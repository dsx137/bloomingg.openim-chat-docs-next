退出当前登录用户，断开当前用户态会话并清理登录相关状态。它不等同于销毁 SDK 全局实例。

## 功能介绍

退出当前登录用户，断开当前用户态会话并清理登录相关状态。它不等同于销毁 SDK 全局实例。

初始化类方法应在应用进程内按需调用；用户态方法通常要求 `InitSDK` 已成功执行，并通过 `operationID` 串联日志。

## Core Binding 方法

| 项目     | 说明                                        |
| -------- | ------------------------------------------- |
| 方法签名 | `Logout(callback Base, operationID string)` |
| 所属能力 | SDK 生命周期                                |
| 源码位置 | `open_im_sdk/init_login.go`                 |

## React Native 调用参考

React Native SDK v3.8.3 的 TypeScript 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`src/sdk.ts`。

```ts showLineNumbers
logout(operationID: string = id())
```

## 输入参数

| 参数名称 | 参数类型   | 是否必填   | 描述                 |
| -------- | ---------- | ---------- | -------------------- |
| 参数     | 见方法原型 | 按方法原型 | 按上方调用原型传入。 |

## 返回结果

返回 `Promise`。调用失败时 Promise reject，错误会被封装为 OpenIMApiError。

## 代码示例

以下示例展示 React Native SDK v3.8.3 的调用骨架，请按业务上下文补充真实参数、错误处理和 UI 状态更新。

```ts showLineNumbers
import OpenIMSDK from '@openim/rn-client-sdk';

const result = await OpenIMSDK.logout();
```
