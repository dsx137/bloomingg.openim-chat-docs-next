`UserInfoReq` 是 uni-app SDK 的数据模型，完整类名为 `io.openim.uniapp.sdk.models.UserInfoReq`。更新当前用户资料时使用的请求对象。

## 字段说明

| 字段名称         | JavaScript 类型 | 描述                                               |
| ---------------- | --------------- | -------------------------------------------------- |
| userID           | String          | 用户id                                             |
| nickname         | String          | 用户昵称                                           |
| faceURL          | String          | 用户头像                                           |
| ex               | String          | 扩展参数                                           |
| globalRecvMsgOpt | Integer         | 全局免打扰，1:屏蔽消息; 2:接收消息但不提示; 0:正常 |
