`AdvancedMessage` 是 uni-app SDK 的数据模型，完整类名为 `io.openim.uniapp.sdk.models.AdvancedMessage`。历史消息分页查询返回的结果对象，包含错误信息、分页游标和消息列表。

## 字段说明

| 字段名称    | JavaScript 类型 | 描述                   |
| ----------- | --------------- | ---------------------- |
| errCode     | int             | 错误码。               |
| errMsg      | String          | 错误信息。             |
| lastMinSeq  | int             | SDK 返回或提交的字段。 |
| isEnd       | boolean         | SDK 返回或提交的字段。 |
| messageList | List<Message>   | SDK 返回或提交的字段。 |

## 关联类型

- [消息对象](/docs/chat/sdk/v4/uniapp/models/message)（`Message`）：消息对象，包含消息 ID、发送者、会话、内容元素和状态等信息。
