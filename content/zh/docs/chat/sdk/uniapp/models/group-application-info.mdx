`GroupApplicationInfo` 是 uni-app SDK 的数据模型，完整类名为 `io.openim.uniapp.sdk.models.GroupApplicationInfo`。入群申请记录，包含群资料、申请人、处理状态和来源。

## 字段说明

| 字段名称      | JavaScript 类型 | 描述                                     |
| ------------- | --------------- | ---------------------------------------- |
| groupID       | String          | 群id                                     |
| groupName     | String          | 群名                                     |
| notification  | String          | 群公告                                   |
| introduction  | String          | 群简介                                   |
| groupFaceURL  | String          | 群头像                                   |
| createTime    | long            | 创建时间                                 |
| status        | int             | 状态                                     |
| creatorUserID | String          | 创建者id                                 |
| groupType     | int             | 群类型                                   |
| ownerUserID   | String          | 拥有者id                                 |
| memberCount   | int             | 成员数量                                 |
| userID        | String          | 申请者的id                               |
| nickname      | String          | 申请者的昵称                             |
| userFaceURL   | String          | 申请者的头像                             |
| gender        | int             | 申请者的性别                             |
| handleResult  | int             | REFUSE = -1, AGREE = 1 -1：拒绝，1：同意 |
| reqMsg        | String          | 申请原因                                 |
| handledMsg    | String          | 处理结果描述                             |
| reqTime       | long            | 申请时间                                 |
| handleUserID  | String          | 处理者的id                               |
| handledTime   | long            | 处理时间                                 |
| ex            | String          | 扩展字段                                 |
| joinSource    | int             | 2：通过邀请 3：通过搜索 4：通过二维码    |
| inviterUserID | String          | 邀请进群用户ID                           |
