## SDK core 错误码

### 通用错误

| 错误码 | 常量 | 默认消息或源码说明 | 说明 |
| ------ | ---- | ------------------ | ---- |
| `10000` | `NetworkError` | `Network error` | 网络错误。 |
| `10001` | `NetworkTimeoutError` | `Network timeout error` | 网络超时。 |
| `10002` | `ArgsError` | `Invalid input arguments` | 输入参数无效。 |
| `10003` | `CtxDeadlineExceededError` | `Context deadline exceeded` | 上下文超时。 |
| `10004` | `ResourceLoadNotCompleteError` | `Resource initialization incomplete` | SDK 资源初始化未完成。 |
| `10005` | `UnknownCode` | `Unrecognized code` | 未识别的错误码。 |
| `10006` | `SdkInternalError` | `Internal SDK error` | SDK 内部错误。 |
| `10007` | `NoUpdateError` | `No updates available` | 没有可更新内容。 |
