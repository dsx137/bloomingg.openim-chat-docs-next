使用业务服务端签发的 token 登录指定用户。登录成功后才能可靠调用会话、消息、群组、好友等用户态接口。

## 功能介绍

使用业务服务端签发的 token 登录指定用户。登录成功后才能可靠调用会话、消息、群组、好友等用户态接口。

初始化类方法应在应用进程内按需调用；用户态方法通常要求 `InitSDK` 已成功执行，并通过 `operationID` 串联日志。

## Core Binding 方法

| 项目     | 说明                                                             |
| -------- | ---------------------------------------------------------------- |
| 方法签名 | `Login(callback Base, operationID string, userID, token string)` |
| 所属能力 | SDK 生命周期                                                     |
| 源码位置 | `open_im_sdk/init_login.go`                                      |

## uni-app 调用参考

uni-app SDK v3.8.3 的 JavaScript 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`ios/OpenIMUniPlugin/OpenIMUniPlugin/Classes/OpenIMModule.m`。

```js showLineNumbers
openIM.login(operationID, options, callback);
```

## 输入参数

| 参数名称 | 参数类型 | 是否必填   | 描述                              |
| -------- | -------- | ---------- | --------------------------------- |
| options  | Object   | 按方法原型 | 按 uni-app 原生插件导出方法传入。 |
| callback | Function | 是         | 按 uni-app 原生插件导出方法传入。 |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。

## 代码示例

以下示例展示 uni-app SDK v3.8.3 的调用骨架，请按业务上下文补充真实参数、错误处理和 UI 状态更新。

```js showLineNumbers
const openIM = uni.requireNativePlugin('OpenIMUniPlugin');

openIM.login('operationID', {}, callback);
```
