将全部会话消息标记为已读，适合一键清除全部未读。

## 功能介绍

将全部会话消息标记为已读，适合一键清除全部未读。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                  |
| -------- | --------------------------------------------------------------------- |
| 方法签名 | `MarkAllConversationMessageAsRead(callback Base, operationID string)` |
| 所属能力 | 会话与消息                                                            |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                     |

## uni-app 调用参考

uni-app SDK v3.8.3 的 JavaScript 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

```js showLineNumbers
openIM.markAllConversationMessageAsRead(operationID, /* ... */, callback);
```

## 输入参数

| 参数名称 | 参数类型   | 是否必填   | 描述                 |
| -------- | ---------- | ---------- | -------------------- |
| 参数     | 见方法原型 | 按方法原型 | 按上方调用原型传入。 |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。
