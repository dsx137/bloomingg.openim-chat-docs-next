获取指定用户在线状态。当前源码实现复用订阅状态接口，因此调用后也可能产生订阅效果。

## 功能介绍

获取指定用户在线状态。当前源码实现复用订阅状态接口，因此调用后也可能产生订阅效果。

用户资料和在线状态方法通常要求当前用户已登录；在线状态变化通过 `SetUserListener` 设置的监听器接收。

## Core Binding 方法

| 项目     | 说明                                                               |
| -------- | ------------------------------------------------------------------ |
| 方法签名 | `GetUserStatus(callback Base, operationID string, userIDs string)` |
| 所属能力 | 用户                                                               |
| 源码位置 | `open_im_sdk/online.go`                                            |

## uni-app 调用参考

uni-app SDK v3.8.3 的 JavaScript 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`ios/OpenIMUniPlugin/OpenIMUniPlugin/Classes/OpenIMModule.m`。

```js showLineNumbers
openIM.getUserStatus(operationID, userIDList, callback);
```

## 输入参数

| 参数名称   | 参数类型 | 是否必填   | 描述                              |
| ---------- | -------- | ---------- | --------------------------------- |
| userIDList | Array    | 按方法原型 | 按 uni-app 原生插件导出方法传入。 |
| callback   | Function | 是         | 按 uni-app 原生插件导出方法传入。 |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。
