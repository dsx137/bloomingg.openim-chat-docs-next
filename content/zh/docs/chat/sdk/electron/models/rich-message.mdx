`RichMessage` 是 Electron SDK 的数据模型，完整类名为 `io.openim.react-native.sdk.models.RichMessage`。高级文本消息中的富文本片段。

## 字段说明

| 字段名称 | TypeScript 类型 | 描述                   |
| -------- | --------------- | ---------------------- |
| type     | String          | SDK 返回或提交的字段。 |
| offset   | int             | 分页偏移量。           |
| length   | int             | SDK 返回或提交的字段。 |
| url      | String          | SDK 返回或提交的字段。 |
| info     | String          | SDK 返回或提交的字段。 |
