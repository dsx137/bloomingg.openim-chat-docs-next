`OnBatchMsgListener` 是 Electron SDK 文档中展示 OpenIM core 方法签名时使用的回调类型，完整类名为 `open_im_sdk_callback.OnBatchMsgListener`。OpenIM core 批量消息监听器，用于接收多条新消息和离线消息。

## 回调方法

| 方法                     | 参数                | 描述               |
| ------------------------ | ------------------- | ------------------ |
| OnRecvNewMessages        | messageList: string | 收到多条新消息。   |
| OnRecvOfflineNewMessages | messageList: string | 收到多条离线消息。 |
