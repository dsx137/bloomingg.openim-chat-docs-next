`Platform` 是 Electron SDK 的枚举常量，完整类名为 `io.openim.react-native.sdk.enums.Platform`。客户端平台类型常量。

## 取值说明

| 常量        | 值  | 描述                          |
| ----------- | --- | ----------------------------- |
| IOS         | 1   | IOS 对应的 SDK 常量。         |
| ANDROID     | 2   | ANDROID 对应的 SDK 常量。     |
| WIN         | 3   | WIN 对应的 SDK 常量。         |
| XOS         | 4   | XOS 对应的 SDK 常量。         |
| WEB         | 5   | WEB 对应的 SDK 常量。         |
| MINI_WEB    | 6   | MINI_WEB 对应的 SDK 常量。    |
| LINUX       | 7   | LINUX 对应的 SDK 常量。       |
| ANDROID_PAD | 8   | ANDROID_PAD 对应的 SDK 常量。 |
| IPAD        | 9   | IPAD 对应的 SDK 常量。        |
