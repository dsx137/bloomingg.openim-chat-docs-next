`GroupAtType` 是 Electron SDK 的枚举常量，完整类名为 `io.openim.react-native.sdk.enums.GroupAtType`。群会话强提醒类型常量。

## 取值说明

| 常量               | 值  | 描述           |
| ------------------ | --- | -------------- |
| AT_NORMAL          | 0   | 清除所有       |
| AT_ME              | 1   | at我           |
| AT_ALL             | 2   | at所有人       |
| AT_ALL_AT_ME       | 3   | at所有人，at我 |
| GROUP_NOTIFICATION | 4   | 群公告提示     |
