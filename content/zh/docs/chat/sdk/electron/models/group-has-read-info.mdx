`GroupHasReadInfo` 是 Electron SDK 的数据模型，完整类名为 `io.openim.react-native.sdk.models.GroupHasReadInfo`。群消息已读成员信息。

## 字段说明

| 字段名称     | TypeScript 类型 | 描述   |
| ------------ | --------------- | ------ |
| hasReadCount | int             | 已读数 |
| unreadCount  | int             | 未读数 |
