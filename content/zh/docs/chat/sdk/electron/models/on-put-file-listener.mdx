`OnPutFileListener` 是 Electron SDK 的回调接口，完整类名为 `io.openim.react-native.sdk.listener.OnPutFileListener`。上传文件时使用的阶段和进度回调。

## 回调方法

| 方法               | 参数                              | 描述             |
| ------------------ | --------------------------------- | ---------------- |
| complete           | long size；String url；long type  | 上传完成 - then  |
| hashPartComplete   | String var1；String var2          | 事件触发时回调。 |
| hashPartProgress   | long var1；long var3；String var5 | 事件触发时回调。 |
| open               | long var1                         | 事件触发时回调。 |
| partSize           | long var1；long var3              | 事件触发时回调。 |
| uploadComplete     | long size；long progress；long l2 | 上传进度         |
| uploadID           | String var1                       | 事件触发时回调。 |
| uploadPartComplete | long var1；long var3；String var5 | 事件触发时回调。 |
