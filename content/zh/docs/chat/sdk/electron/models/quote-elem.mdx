`QuoteElem` 是 Electron SDK 的数据模型，完整类名为 `io.openim.react-native.sdk.models.QuoteElem`。引用消息元素。

## 字段说明

| 字段名称     | TypeScript 类型 | 描述         |
| ------------ | --------------- | ------------ |
| text         | String          | 回复内容     |
| quoteMessage | Message         | 被回复的消息 |

## 关联类型

- [消息对象](/docs/chat/sdk/v4/electron/models/message)（`Message`）：消息对象，包含消息 ID、发送者、会话、内容元素和状态等信息。
