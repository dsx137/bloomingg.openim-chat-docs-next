`TypeKeySetResult` 是 Electron SDK 的数据模型，完整类名为 `io.openim.react-native.sdk.models.TypeKeySetResult`。设置消息扩展后的结果信息。

## 字段说明

| 字段名称         | TypeScript 类型 | 描述                   |
| ---------------- | --------------- | ---------------------- |
| errCode          | long            | 错误码。               |
| errMsg           | String          | 错误信息。             |
| typeKey          | String          | SDK 返回或提交的字段。 |
| latestUpdateTime | long            | SDK 返回或提交的字段。 |
| value            | String          | 枚举值。               |
