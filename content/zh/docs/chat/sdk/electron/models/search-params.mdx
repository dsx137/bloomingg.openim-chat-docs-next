`SearchParams` 是 Electron SDK 的数据模型，完整类名为 `io.openim.react-native.sdk.models.SearchParams`。搜索会话或好友时使用的搜索条件。

## 字段说明

| 字段名称        | TypeScript 类型 | 描述                   |
| --------------- | --------------- | ---------------------- |
| conversationID  | String          | SDK 返回或提交的字段。 |
| clientMsgIDList | List<String>    | SDK 返回或提交的字段。 |
