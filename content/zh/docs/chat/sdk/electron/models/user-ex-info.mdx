`UserExInfo` 是 Electron SDK 的数据模型，完整类名为 `io.openim.react-native.sdk.models.UserExInfo`。用户资料中的扩展信息。

## 字段说明

| 字段名称 | TypeScript 类型 | 描述                   |
| -------- | --------------- | ---------------------- |
| userID   | String          | SDK 返回或提交的字段。 |
| ex       | String          | 扩展字段。             |
