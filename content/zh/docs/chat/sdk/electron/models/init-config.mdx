`InitConfig` 是 Electron SDK 的数据模型，完整类名为 `io.openim.react-native.sdk.models.InitConfig`。初始化 SDK 时传入的基础配置。

## 类型说明

apiUrl SDK的API接口地址。如：http:xxx:10000 wsUrl SDK的web socket地址。如： ws:xxx:17778 storageDir 数据存储目录路径 logLevel 日志等级，如：6 isLogStandardOutput 控制台是否输出日志 logFilePath 日志输出的路径

## 字段说明

| 字段名称            | TypeScript 类型 | 描述                   |
| ------------------- | --------------- | ---------------------- |
| platformID          | int             | SDK 返回或提交的字段。 |
| apiAddr             | String          | SDK 返回或提交的字段。 |
| wsAddr              | String          | SDK 返回或提交的字段。 |
| dataDir             | String          | SDK 返回或提交的字段。 |
| logLevel            | int             | SDK 返回或提交的字段。 |
| isLogStandardOutput | boolean         | SDK 返回或提交的字段。 |
| logFilePath         | String          | SDK 返回或提交的字段。 |
