`GetFriendApplicationListAsApplicantReq` 是 Electron SDK 的数据模型，完整类名为 `io.openim.react-native.sdk.models.GetFriendApplicationListAsApplicantReq`。查询当前用户发出的好友申请时使用的分页条件。

## 字段说明

| 字段名称 | TypeScript 类型 | 描述         |
| -------- | --------------- | ------------ |
| offset   | int             | 分页偏移量。 |
| count    | int             | 分页数量。   |
