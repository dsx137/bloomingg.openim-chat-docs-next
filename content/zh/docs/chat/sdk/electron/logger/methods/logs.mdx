## 功能介绍

主动写入一条客户端日志，适合业务层把关键 IM 操作写入 SDK 日志链路。

第三方能力至少要求 SDK 已初始化；日志和文件上传的进度通过对应 progress callback 返回。

## 底层能力

| 项目     | 说明                                                                                                                        |
| -------- | --------------------------------------------------------------------------------------------------------------------------- |
| 方法签名 | `Logs(callback Base, operationID string, logLevel int, file string, line int, msgs string, err string, keyAndValue string)` |
| 所属能力 | 日志 |
| 源码位置 | `open_im_sdk/third.go`                                                                                                      |
