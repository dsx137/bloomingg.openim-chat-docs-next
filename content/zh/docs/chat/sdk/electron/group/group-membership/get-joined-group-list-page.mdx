分页获取当前用户已加入群组，适合群数量较多的账号。

## 功能介绍

分页获取当前用户已加入群组，适合群数量较多的账号。

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。

## 底层能力

| 项目     | 说明                                                                             |
| -------- | -------------------------------------------------------------------------------- |
| 方法签名 | `GetJoinedGroupListPage(callback Base, operationID string, offset, count int32)` |
| 所属能力 | 群组                                                                             |
| 源码位置 | `open_im_sdk/group.go`                                                           |

## Electron 调用参考

Electron SDK v3.8.3 暴露以下平台方法原型。底层能力表用于说明 OpenIM core 能力和参数映射。

源码位置：`src/core/modules/group.ts`。

```ts showLineNumbers
getJoinedGroupListPage(params: OffsetParams, opid = uuidV4())
```

## 输入参数

| 参数名称 | 参数类型     | 是否必填 | 描述                               |
| -------- | ------------ | -------- | ---------------------------------- |
| params   | OffsetParams | 是       | 按平台类型定义传入。               |
| opid     | As defined   | 否       | 链路追踪 ID；不传时 SDK 自动生成。 |

## 返回结果

返回 `Promise` 或注册回调结果。调用失败时 Promise reject；建议在业务层使用 `try/catch` 或 `.catch()` 处理错误。
