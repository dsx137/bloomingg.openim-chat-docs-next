## 功能介绍

更新当前登录用户资料，例如昵称、头像或扩展字段。只应修改当前账号允许自行修改的字段。

用户资料和在线状态方法通常要求当前用户已登录；在线状态变化通过 `SetUserListener` 设置的监听器接收。

## 底层能力

| 项目     | 说明                                                              |
| -------- | ----------------------------------------------------------------- |
| 方法签名 | `SetSelfInfo(callback Base, operationID string, userInfo string)` |
| 所属能力 | 用户                                                              |
| 源码位置 | `open_im_sdk/user.go`                                             |

## Electron 调用参考

Electron SDK v3.8.3 暴露以下平台方法原型。底层能力表用于说明 OpenIM core 能力和参数映射。

源码位置：`src/core/modules/user.ts`。

```ts showLineNumbers
setSelfInfo(params: PartialUserItem, opid = uuidV4())
```

## 输入参数

| 参数名称 | 参数类型        | 是否必填 | 描述                               |
| -------- | --------------- | -------- | ---------------------------------- |
| params   | PartialUserItem | 是       | 按平台类型定义传入。               |
| opid     | As defined      | 否       | 链路追踪 ID；不传时 SDK 自动生成。 |

## 返回结果

返回 `Promise` 或注册回调结果。调用失败时 Promise reject；建议在业务层使用 `try/catch` 或 `.catch()` 处理错误。
