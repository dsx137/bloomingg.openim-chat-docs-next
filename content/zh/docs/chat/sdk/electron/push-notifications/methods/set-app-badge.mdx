## 功能介绍

设置应用角标未读数，适合未读数变化后同步系统角标。

第三方能力至少要求 SDK 已初始化；日志和文件上传的进度通过对应 progress callback 返回。

## 底层能力

| 项目     | 说明                                                                   |
| -------- | ---------------------------------------------------------------------- |
| 方法签名 | `SetAppBadge(callback Base, operationID string, appUnreadCount int32)` |
| 所属能力 | 推送通知                                                       |
| 源码位置 | `open_im_sdk/third.go`                                                 |
