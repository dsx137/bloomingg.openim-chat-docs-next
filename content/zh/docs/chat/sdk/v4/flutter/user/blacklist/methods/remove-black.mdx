将指定用户移出黑名单，恢复与该用户的普通关系能力。

## 功能介绍

将指定用户移出黑名单，恢复与该用户的普通关系能力。

黑名单新增和移除通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；屏蔽后的业务效果以当前服务端策略为准。

## Core Binding 方法

| 项目     | 说明                                                                  |
| -------- | --------------------------------------------------------------------- |
| 方法签名 | `RemoveBlack(callback Base, operationID string, removeUserID string)` |
| 所属能力 | 黑名单                                                                |
| 源码位置 | `open_im_sdk/relation.go`                                             |

## Flutter 调用参考

Flutter SDK 3.8.3+hotfix.12 的 Dart 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`lib/src/manager/im_friendship_manager.dart`。

```dart showLineNumbers
Future<dynamic> removeBlacklist({ required String userID, String? operationID, })
```

## 输入参数

| 参数名称 | 参数类型 | 是否必填 | 描述                     |
| -------- | -------- | -------- | ------------------------ |
| userID   | String   | 是       | 用户 ID 或用户 ID 列表。 |

## 返回结果

返回 `Future<dynamic>`。调用失败时 Future 抛出异常，建议在业务层捕获并记录 operationID。
