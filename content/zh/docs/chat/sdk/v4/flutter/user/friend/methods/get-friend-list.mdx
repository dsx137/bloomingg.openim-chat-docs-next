获取当前用户全部好友列表，适合好友通讯录首屏加载或登录后全量同步。

## 功能介绍

获取当前用户全部好友列表，适合好友通讯录首屏加载或登录后全量同步。

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。

## Core Binding 方法

| 项目     | 说明                                                                 |
| -------- | -------------------------------------------------------------------- |
| 方法签名 | `GetFriendList(callback Base, operationID string, filterBlack bool)` |
| 所属能力 | 好友                                                                 |
| 源码位置 | `open_im_sdk/relation.go`                                            |

## Flutter 调用参考

Flutter SDK 3.8.3+hotfix.12 的 Dart 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`lib/src/manager/im_friendship_manager.dart`。

```dart showLineNumbers
Future<List<FriendInfo>> getFriendList({ String? operationID, bool filterBlack = false, })
```

## 输入参数

| 参数名称    | 参数类型 | 是否必填 | 描述             |
| ----------- | -------- | -------- | ---------------- |
| filterBlack | bool     | 否       | 按方法原型传入。 |

## 返回结果

返回 `Future<List<FriendInfo>>`。调用失败时 Future 抛出异常，建议在业务层捕获并记录 operationID。

## 代码示例

以下示例展示 Flutter SDK 3.8.3+hotfix.12 的调用骨架，请按业务上下文补充真实参数、错误处理和 UI 状态更新。

```dart showLineNumbers
final result = await OpenIM.iMManager.friendshipManager.getFriendList(
  filterBlack: false,

```
