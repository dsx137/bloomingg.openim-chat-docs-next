检查一批用户与当前用户是否为好友关系，适合展示添加好友按钮或发起关系操作前校验。

## 功能介绍

检查一批用户与当前用户是否为好友关系，适合展示添加好友按钮或发起关系操作前校验。

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。

## Core Binding 方法

| 项目     | 说明                                                                |
| -------- | ------------------------------------------------------------------- |
| 方法签名 | `CheckFriend(callback Base, operationID string, userIDList string)` |
| 所属能力 | 好友                                                                |
| 源码位置 | `open_im_sdk/relation.go`                                           |

## Flutter 调用参考

Flutter SDK v3.8.3 的 Dart 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`lib/src/manager/im_friendship_manager.dart`。

```dart showLineNumbers
Future<List<FriendshipInfo>> checkFriend({ required List<String> userIDList, String? operationID, })
```

## 输入参数

| 参数名称   | 参数类型     | 是否必填 | 描述                     |
| ---------- | ------------ | -------- | ------------------------ |
| userIDList | List<String> | 是       | 用户 ID 或用户 ID 列表。 |

## 返回结果

返回 `Future<List<FriendshipInfo>>`。调用失败时 Future 抛出异常，建议在业务层捕获并记录 operationID。
