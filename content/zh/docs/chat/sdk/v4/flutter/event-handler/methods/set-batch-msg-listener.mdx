设置批量消息监听器，适合一次处理多条新消息或多条离线消息，减少高频单条事件处理压力。

## 功能介绍

设置批量消息监听器，适合一次处理多条新消息或多条离线消息，减少高频单条事件处理压力。

`Set*Listener` 会替换当前 SDK 保存的同类监听器；如果多个模块都要消费事件，应在业务层封装统一分发。

## Core Binding 方法

| 项目     | 说明                                               |
| -------- | -------------------------------------------------- |
| 方法签名 | `SetBatchMsgListener(listener OnBatchMsgListener)` |
| 所属能力 | 监听器                                             |
| 源码位置 | `open_im_sdk/listener.go`                          |
