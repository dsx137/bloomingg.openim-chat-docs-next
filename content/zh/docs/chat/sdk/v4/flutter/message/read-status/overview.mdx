已读状态用于进入会话后清理未读、批量清除未读，以及按消息 ID 精确标记已读。

## 方法索引

| 方法                                                                                                    | 说明                                                     |
| ------------------------------------------------------------------------------------------------------- | -------------------------------------------------------- |
| [标记会话已读](/docs/chat/sdk/v4/flutter/message/read-status/mark-conversation-message-as-read)         | 将指定会话中的消息标记为已读，适合进入会话页后清理未读。 |
| [标记全部会话已读](/docs/chat/sdk/v4/flutter/message/read-status/mark-all-conversation-message-as-read) | 将全部会话消息标记为已读，适合一键清除全部未读。         |
| [按消息 ID 标记已读](/docs/chat/sdk/v4/flutter/message/read-status/mark-messages-as-read-by-msg-id)     | 按消息 ID 精确标记已读，适合只确认一部分消息已读的场景。 |

## 相关类型

- [群消息已读回执](/docs/chat/sdk/v4/flutter/models/group-message-receipt)（`GroupMessageReceipt`）：群消息已读回执数据。
- [单聊已读回执信息](/docs/chat/sdk/v4/flutter/models/c2c-read-receipt-info)（`C2CReadReceiptInfo`）：单聊已读回执数据。
