查询消息用于按会话、消息 ID、游标或关键字读取历史和本地消息。

## 方法索引

| 方法                                                                                                                | 说明                                                           |
| ------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------- |
| [查找消息列表](/docs/chat/sdk/v4/flutter/message/retrieving-messages/find-message-list)                             | 按多个会话和消息 ID 条件查找消息，适合定位指定消息集合。       |
| [获取历史消息](/docs/chat/sdk/v4/flutter/message/retrieving-messages/get-advanced-history-message-list)             | 获取指定会话的历史消息，适合聊天页向上翻页或首次加载历史记录。 |
| [反向获取历史消息](/docs/chat/sdk/v4/flutter/message/retrieving-messages/get-advanced-history-message-list-reverse) | 按反向方向获取历史消息，适合围绕游标向另一方向加载。           |
| [搜索本地消息](/docs/chat/sdk/v4/flutter/message/retrieving-messages/search-local-messages)                         | 搜索本地消息，适合聊天记录搜索和按关键字查找历史内容。         |

## 相关类型

- [高级历史消息结果](/docs/chat/sdk/v4/flutter/models/advanced-message)（`AdvancedMessage`）：历史消息分页返回结果。
- [消息搜索结果](/docs/chat/sdk/v4/flutter/models/search-result)（`SearchResult`）：本地消息搜索返回结果。
- [搜索参数](/docs/chat/sdk/v4/flutter/models/search-params)（`SearchParams`）：搜索本地消息时使用的条件对象。
