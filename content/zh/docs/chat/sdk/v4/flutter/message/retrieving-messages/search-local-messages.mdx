搜索本地消息，适合聊天记录搜索和按关键字查找历史内容。

## 功能介绍

搜索本地消息，适合聊天记录搜索和按关键字查找历史内容。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                         |
| -------- | ---------------------------------------------------------------------------- |
| 方法签名 | `SearchLocalMessages(callback Base, operationID string, searchParam string)` |
| 所属能力 | 会话与消息                                                                   |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                            |

## Flutter 调用参考

Flutter SDK 3.8.3+hotfix.12 的 Dart 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`lib/src/manager/im_message_manager.dart`。

```dart showLineNumbers
Future<SearchResult> searchLocalMessages({ String? conversationID, List<String> keywordList = const [], int keywordListMatchType = 0, List<String> senderUserIDList = const [], List<int> messageTypeList = const [], int searchTimePosition = 0, int searchTimePeriod = 0, int pageIndex = 1, int count = 40, String? operationID, })
```

## 输入参数

| 参数名称             | 参数类型     | 是否必填 | 描述                          |
| -------------------- | ------------ | -------- | ----------------------------- |
| conversationID       | String?      | 否       | 会话 ID 或会话 ID 列表。      |
| keywordList          | List<String> | 否       | 按方法原型传入。              |
| keywordListMatchType | int          | 否       | 按方法原型传入。              |
| senderUserIDList     | List<String> | 否       | 用户 ID 或用户 ID 列表。      |
| messageTypeList      | List<int>    | 否       | 消息对象、消息内容或消息 ID。 |
| searchTimePosition   | int          | 否       | 按方法原型传入。              |
| searchTimePeriod     | int          | 否       | 按方法原型传入。              |
| pageIndex            | int          | 否       | 分页参数。                    |
| count                | int          | 否       | 分页参数。                    |

## 返回结果

返回 `Future<SearchResult>`。调用失败时 Future 抛出异常，建议在业务层捕获并记录 operationID。
