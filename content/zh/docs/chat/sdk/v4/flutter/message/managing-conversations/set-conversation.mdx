## 功能介绍

设置会话属性，例如置顶、免打扰、扩展字段等当前结构支持的字段。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                    |
| -------- | --------------------------------------------------------------------------------------- |
| 方法签名 | `SetConversation(callback Base, operationID string, conversationID string, req string)` |
| 所属能力 | 会话与消息                                                                              |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                       |

## Flutter 调用参考

Flutter SDK 3.8.3+hotfix.12 的 Dart 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`lib/src/manager/im_conversation_manager.dart`。

```dart showLineNumbers
Future setConversation(String conversationID, ConversationReq req, { String? operationID, })
```

## 输入参数

| 参数名称       | 参数类型        | 是否必填   | 描述                                   |
| -------------- | --------------- | ---------- | -------------------------------------- |
| conversationID | String          | 按方法原型 | 会话 ID 或会话 ID 列表。               |
| req            | ConversationReq | 按方法原型 | 请求对象，字段以对应平台类型定义为准。 |

## 返回结果

返回 `Future`。调用失败时 Future 抛出异常，建议在业务层捕获并记录 operationID。

final result = await OpenIM.iMManager.conversationManager.setConversation(
