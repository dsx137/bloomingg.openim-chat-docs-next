本地消息用于向本地存储补写消息，常见于导入历史、补写系统消息或只在本地展示的业务消息。

## 方法索引

| 方法                                                                                                        | 说明                                                                   |
| ----------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| [插入单聊本地消息](/docs/chat/sdk/v4/flutter/message/local-messages/insert-single-message-to-local-storage) | 向本地单聊会话插入一条消息，适合导入历史、补写系统消息或本地展示消息。 |
| [插入群聊本地消息](/docs/chat/sdk/v4/flutter/message/local-messages/insert-group-message-to-local-storage)  | 向本地群聊会话插入一条消息，适合导入或补写群聊本地消息。               |

## 相关类型

- [消息对象](/docs/chat/sdk/v4/flutter/models/message)（`Message`）：消息对象，包含消息 ID、发送者、会话、内容元素和状态等信息。
