初始化 SDK 全局实例、日志、长连接管理、本地数据目录和基础资源。应用进程内通常只调用一次。

## 功能介绍

初始化 SDK 全局实例、日志、长连接管理、本地数据目录和基础资源。应用进程内通常只调用一次。

初始化类方法应在应用进程内按需调用；用户态方法通常要求 `InitSDK` 已成功执行，并通过 `operationID` 串联日志。

## Core Binding 方法

| 项目     | 说明                                                                       |
| -------- | -------------------------------------------------------------------------- |
| 方法签名 | `InitSDK(listener OnConnListener, operationID string, config string) bool` |
| 所属能力 | SDK 生命周期                                                               |
| 源码位置 | `open_im_sdk/init_login.go`                                                |

## Flutter 调用参考

Flutter SDK v3.8.3 的 Dart 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`lib/src/manager/im_manager.dart`。

```dart showLineNumbers
Future<dynamic> initSDK({ required int platformID, required String apiAddr, required String wsAddr, required String dataDir, required OnConnectListener listener, int logLevel = 6, bool isNeedEncryption = false, bool isCompression = false, bool isLogStandardOutput = true, String? logFilePath, String? operationID, })
```

## 输入参数

| 参数名称            | 参数类型          | 是否必填 | 描述                                   |
| ------------------- | ----------------- | -------- | -------------------------------------- |
| platformID          | int               | 是       | 按方法原型传入。                       |
| apiAddr             | String            | 是       | 按方法原型传入。                       |
| wsAddr              | String            | 是       | 按方法原型传入。                       |
| dataDir             | String            | 是       | 请求对象，字段以对应平台类型定义为准。 |
| listener            | OnConnectListener | 是       | 回调或监听器。                         |
| logLevel            | int               | 否       | 按方法原型传入。                       |
| isNeedEncryption    | bool              | 否       | 按方法原型传入。                       |
| isCompression       | bool              | 否       | 按方法原型传入。                       |
| isLogStandardOutput | bool              | 否       | 按方法原型传入。                       |
| logFilePath         | String?           | 否       | 按方法原型传入。                       |

## 返回结果

返回 `Future<dynamic>`。调用失败时 Future 抛出异常，建议在业务层捕获并记录 operationID。

## 代码示例

以下示例展示 Flutter SDK v3.8.3 的调用骨架，请按业务上下文补充真实参数、错误处理和 UI 状态更新。

```dart showLineNumbers
final result = await OpenIM.iMManager.initSDK(
  platformID: 0,
  apiAddr: 'apiAddr',
  wsAddr: 'wsAddr',
  dataDir: {},
  listener: [],
  logLevel: 0,
  isNeedEncryption: false,
  isCompression: false,
  isLogStandardOutput: false,
  logFilePath: 'logFilePath',
);
```
