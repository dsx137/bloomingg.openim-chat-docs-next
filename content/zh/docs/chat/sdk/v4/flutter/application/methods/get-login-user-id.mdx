## 功能介绍

获取当前 SDK 实例记录的登录用户 ID，常用于日志、会话归属和账号切换判断。

初始化类方法应在应用进程内按需调用；用户态方法通常要求 `InitSDK` 已成功执行，并通过 `operationID` 串联日志。

## Core Binding 方法

| 项目     | 说明                        |
| -------- | --------------------------- |
| 方法签名 | `GetLoginUserID() string`   |
| 所属能力 | SDK 生命周期                |
| 源码位置 | `open_im_sdk/init_login.go` |

## Flutter 调用参考

Flutter SDK v3.8.3 的 Dart 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`lib/src/manager/im_manager.dart`。

```dart showLineNumbers
Future<String> getLoginUserID()
```

| 参数名称 | 参数类型   | 是否必填   | 描述                 |

## 返回结果

返回 `Future<String>`。调用失败时 Future 抛出异常，建议在业务层捕获并记录 operationID。
