## 功能介绍

获取当前 core SDK 的版本号，适合在启动日志、问题反馈和兼容性判断中记录。

初始化类方法应在应用进程内按需调用；用户态方法通常要求 `InitSDK` 已成功执行，并通过 `operationID` 串联日志。

## Core Binding 方法

| 项目     | 说明                        |
| -------- | --------------------------- |
| 方法签名 | `GetSdkVersion() string`    |
| 所属能力 | SDK 生命周期                |
| 源码位置 | `open_im_sdk/init_login.go` |

## Flutter 调用参考

Flutter SDK v3.8.3 的 Dart 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

```dart showLineNumbers
getSdkVersion(/* ... */)
```

## 返回结果

返回 `Future`。调用失败时 Future 抛出异常，建议在业务层捕获并记录 operationID。
