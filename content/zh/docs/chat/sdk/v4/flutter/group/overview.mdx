OpenIM Flutter SDK 使用群组承载多人聊天能力。本节说明群组创建、资料维护、入群申请、成员管理、入群关系和禁言等能力。

本页是群组方法目录。每个子模块都有独立概述页，每个方法都有独立说明页。

## 事件联动

群组相关变化通过 `SetGroupListener` 注册的 `OnGroupListener` 接收，包括群新增/删除、群资料变化、成员新增/删除、成员资料变化、群申请新增/删除/接受/拒绝等。

## 相关类型

- [群组信息](/docs/chat/sdk/v4/flutter/models/group-info)（`GroupInfo`）：群组资料对象，包含群名、公告、群主、成员数和入群设置。
- [群成员信息](/docs/chat/sdk/v4/flutter/models/group-members-info)（`GroupMembersInfo`）：群成员资料对象，包含角色、入群时间和群昵称等信息。
- [群申请信息](/docs/chat/sdk/v4/flutter/models/group-application-info)（`GroupApplicationInfo`）：入群申请记录，用于申请列表和审核流程。
- [群组监听器](/docs/chat/sdk/v4/flutter/models/on-group-listener)（`OnGroupListener`）：群组、入群申请和群成员变化监听器。
