| 方法                                                                                      | 说明                                               |

| [设置群禁言](/docs/chat/sdk/v4/flutter/group/group-moderation/change-group-mute)          | 开启或关闭全群禁言。适合群管理者控制群内发言状态。 |
| [设置成员禁言](/docs/chat/sdk/v4/flutter/group/group-moderation/change-group-member-mute) | 设置指定群成员禁言时长。适合临时限制单个成员发言。 |
