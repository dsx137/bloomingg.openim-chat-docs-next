## 功能介绍

更新群资料，例如群名称、头像、公告或扩展字段。

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。

## Core Binding 方法

| 项目     | 说明                                                                |
| -------- | ------------------------------------------------------------------- |
| 方法签名 | `SetGroupInfo(callback Base, operationID string, groupInfo string)` |
| 所属能力 | 群组                                                                |
| 源码位置 | `open_im_sdk/group.go`                                              |

## Flutter 调用参考

Flutter SDK v3.8.3 的 Dart 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`lib/src/manager/im_group_manager.dart`。

```dart showLineNumbers
Future<dynamic> setGroupInfo(GroupInfo groupInfo, { String? operationID, })
```

## 输入参数

| 参数名称  | 参数类型                                                 | 是否必填   | 描述                                   |
| --------- | -------------------------------------------------------- | ---------- | -------------------------------------- |
| groupInfo | [GroupInfo](/docs/chat/sdk/v4/flutter/models/group-info) | 按方法原型 | 请求对象，字段以对应平台类型定义为准。 |

## 返回结果

返回 `Future<dynamic>`。调用失败时 Future 抛出异常，建议在业务层捕获并记录 operationID。
