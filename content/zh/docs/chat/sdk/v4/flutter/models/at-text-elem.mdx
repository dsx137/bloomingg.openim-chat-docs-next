`AtTextElem` 是 Flutter SDK 的数据模型，完整类名为 `io.openim.flutter.sdk.models.AtTextElem`。At 文本消息的内容元素，用于保存文本、被提及用户和引用信息。

## 字段说明

| 字段名称     | Dart 类型        | 描述            |
| ------------ | ---------------- | --------------- |
| text         | String           | at 消息内容     |
| atUserList   | List<String>     | 被@的用户id集合 |
| atUsersInfo  | List<AtUserInfo> | 被@的用户信息   |
| isAtSelf     | boolean          | 自己是否被@了   |
| quoteMessage | Message          | 引用消息        |

## 关联类型

- [被提及用户信息](/docs/chat/sdk/v4/flutter/models/at-user-info)（`AtUserInfo`）：被提及用户的 ID 与群昵称映射信息。
- [消息对象](/docs/chat/sdk/v4/flutter/models/message)（`Message`）：消息对象，包含消息 ID、发送者、会话、内容元素和状态等信息。
