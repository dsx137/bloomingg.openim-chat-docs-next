`KeyValue` 是 Flutter SDK 的数据模型，完整类名为 `io.openim.flutter.sdk.models.KeyValue`。消息扩展使用的键值项。

## 字段说明

| 字段名称         | Dart 类型 | 描述                   |
| ---------------- | --------- | ---------------------- |
| typeKey          | String    | SDK 返回或提交的字段。 |
| value            | String    | 枚举值。               |
| latestUpdateTime | long      | SDK 返回或提交的字段。 |
