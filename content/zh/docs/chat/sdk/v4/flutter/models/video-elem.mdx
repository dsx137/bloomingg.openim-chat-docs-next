`VideoElem` 是 Flutter SDK 的数据模型，完整类名为 `io.openim.flutter.sdk.models.VideoElem`。视频消息元素。

## 字段说明

| 字段名称       | Dart 类型 | 描述             |
| -------------- | --------- | ---------------- |
| videoPath      | String    | 视频本地资源地址 |
| videoUUID      | String    | 视频唯一ID       |
| videoUrl       | String    | 视频oss地址      |
| videoType      | String    | 视频类型         |
| videoSize      | long      | 视频大小         |
| duration       | long      | 视频时长         |
| snapshotPath   | String    | 视频快照本地地址 |
| snapshotUUID   | String    | 视频快照唯一ID   |
| snapshotSize   | long      | 视频快照大小     |
| snapshotUrl    | String    | 视频快照oss地址  |
| snapshotWidth  | int       | 视频快照宽度     |
| snapshotHeight | int       | 视频快照高度     |
