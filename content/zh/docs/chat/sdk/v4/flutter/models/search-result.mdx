`SearchResult` 是 Flutter SDK 的数据模型，完整类名为 `io.openim.flutter.sdk.models.SearchResult`。本地消息搜索结果。

## 字段说明

| 字段名称          | Dart 类型              | 描述                   |
| ----------------- | ---------------------- | ---------------------- |
| totalCount        | int                    | 获取到的总的消息数量   |
| searchResultItems | List<SearchResultItem> | SDK 返回或提交的字段。 |
| findResultItems   | List<SearchResultItem> | SDK 返回或提交的字段。 |

## 关联类型

- [消息搜索结果项](/docs/chat/sdk/v4/flutter/models/search-result-item)（`SearchResultItem`）：本地消息搜索结果中的单个会话结果。
