`GroupMessageReadInfo` 是 Flutter SDK 的数据模型，完整类名为 `io.openim.flutter.sdk.models.GroupMessageReadInfo`。群消息已读与未读统计信息。

## 字段说明

| 字段名称     | Dart 类型              | 描述                   |
| ------------ | ---------------------- | ---------------------- |
| clientMsgID  | String                 | SDK 返回或提交的字段。 |
| hasReadCount | int                    | SDK 返回或提交的字段。 |
| unreadCount  | int                    | SDK 返回或提交的字段。 |
| readMembers  | List<GroupMembersInfo> | SDK 返回或提交的字段。 |

## 关联类型

- [群成员信息](/docs/chat/sdk/v4/flutter/models/group-members-info)（`GroupMembersInfo`）：群成员资料，包含群内昵称、角色、入群来源和禁言时间。
