`MessageStatus` 是 Flutter SDK 的枚举常量，完整类名为 `io.openim.flutter.sdk.enums.MessageStatus`。消息发送状态常量。

## 取值说明

| 常量      | 值  | 描述     |
| --------- | --- | -------- |
| SENDING   | 1   | 发送中   |
| SUCCEEDED | 2   | 发送成功 |
| FAILED    | 3   | 发送失败 |
| DELETED   | 4   | 发送取消 |
