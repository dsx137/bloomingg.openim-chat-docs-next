`PutArgs` 是 Flutter SDK 的数据模型，完整类名为 `io.openim.flutter.sdk.models.PutArgs`。文件上传参数。

## 字段说明

| 字段名称 | Dart 类型 | 描述         |
| -------- | --------- | ------------ |
| filepath | String    | 本地文件路径 |
| name     | String    | 文件名       |
| cuase    | String    | 类型         |
