`SearchResultItem` 是 Flutter SDK 的数据模型，完整类名为 `io.openim.flutter.sdk.models.SearchResultItem`。本地消息搜索结果中的单个会话结果。

## 字段说明

| 字段名称       | Dart 类型     | 描述                 |
| -------------- | ------------- | -------------------- |
| conversationID | String        | 会话ID               |
| messageCount   | int           | 这个会话下的消息数量 |
| showName       | String        | 显示名               |
| faceURL        | String        | 头像                 |
| messageList    | List<Message> | Message的列表        |

## 关联类型

- [消息对象](/docs/chat/sdk/v4/flutter/models/message)（`Message`）：消息对象，包含消息 ID、发送者、会话、内容元素和状态等信息。
