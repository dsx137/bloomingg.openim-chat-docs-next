`OfflinePushInfo` 是 Flutter SDK 的数据模型，完整类名为 `io.openim.flutter.sdk.models.OfflinePushInfo`。发送消息时携带的离线推送配置。

## 字段说明

| 字段名称      | Dart 类型 | 描述                   |
| ------------- | --------- | ---------------------- |
| title         | String    | SDK 返回或提交的字段。 |
| desc          | String    | SDK 返回或提交的字段。 |
| ex            | String    | 扩展字段。             |
| iOSPushSound  | String    | SDK 返回或提交的字段。 |
| iOSBadgeCount | boolean   | SDK 返回或提交的字段。 |
