同意指定用户的入群申请。

## 功能介绍

同意指定用户的入群申请。

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。

## Core Binding 方法

| 项目     | 说明                                                                                                             |
| -------- | ---------------------------------------------------------------------------------------------------------------- |
| 方法签名 | `AcceptGroupApplication(callback Base, operationID string, groupID string, fromUserID string, handleMsg string)` |
| 所属能力 | 群组                                                                                                             |
| 源码位置 | `open_im_sdk/group.go`                                                                                           |

## iOS 调用参考

iOS Core SDK v3.8.3 的 Objective-C 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`Framework/OpenIMCore.xcframework/.../Open_im_sdk.objc.h`。

```objective-c showLineNumbers
FOUNDATION_EXPORT void Open_im_sdkAcceptGroupApplication(id<Open_im_sdk_callbackBase> _Nullable callback, NSString* _Nullable operationID, NSString* _Nullable groupID, NSString* _Nullable fromUserID, NSString* _Nullable handleMsg);
```

## 输入参数

| 参数名称   | 参数类型                     | 是否必填   | 描述                          |
| ---------- | ---------------------------- | ---------- | ----------------------------- |
| callback   | id<Open_im_sdk_callbackBase> | 是         | 回调或监听器。                |
| groupID    | NSString\*                   | 按方法原型 | 群组 ID 或群组 ID 列表。      |
| fromUserID | NSString\*                   | 按方法原型 | 用户 ID 或用户 ID 列表。      |
| handleMsg  | NSString\*                   | 按方法原型 | 消息对象、消息内容或消息 ID。 |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。

以下示例展示 iOS Core SDK v3.8.3 的调用骨架，请按业务上下文补充真实参数、错误处理和 UI 状态更新。

Open_im_sdkAcceptGroupApplication(callback, @"operationID", @"groupID", @"fromUserID", @"handleMsg");
