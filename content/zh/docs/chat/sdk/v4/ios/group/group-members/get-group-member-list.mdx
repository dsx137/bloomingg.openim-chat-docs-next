按成员类型筛选并分页获取群成员列表。

## 功能介绍

按成员类型筛选并分页获取群成员列表。

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。

## Core Binding 方法

| 项目     | 说明                                                                                                             |
| -------- | ---------------------------------------------------------------------------------------------------------------- |
| 方法签名 | `GetGroupMemberList(callback Base, operationID string, groupID string, filter int32, offset int32, count int32)` |
| 所属能力 | 群组                                                                                                             |
| 源码位置 | `open_im_sdk/group.go`                                                                                           |

## iOS 调用参考

iOS Core SDK v3.8.3 的 Objective-C 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`Framework/OpenIMCore.xcframework/.../Open_im_sdk.objc.h`。

```objective-c showLineNumbers
FOUNDATION_EXPORT void Open_im_sdkGetGroupMemberList(id<Open_im_sdk_callbackBase> _Nullable callback, NSString* _Nullable operationID, NSString* _Nullable groupID, int32_t filter, int32_t offset, int32_t count);
```

## 输入参数

| 参数名称 | 参数类型                     | 是否必填   | 描述                     |
| -------- | ---------------------------- | ---------- | ------------------------ |
| callback | id<Open_im_sdk_callbackBase> | 是         | 回调或监听器。           |
| groupID  | NSString\*                   | 按方法原型 | 群组 ID 或群组 ID 列表。 |
| filter   | int32_t                      | 按方法原型 | 按方法原型传入。         |
| offset   | int32_t                      | 按方法原型 | 分页参数。               |
| count    | int32_t                      | 按方法原型 | 分页参数。               |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。
