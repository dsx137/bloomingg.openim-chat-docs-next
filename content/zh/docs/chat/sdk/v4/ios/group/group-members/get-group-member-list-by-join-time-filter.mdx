## 功能介绍

按入群时间范围和用户过滤条件获取群成员，适合审计、成员管理和增量查看。

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。

## iOS 调用参考

iOS Core SDK v3.8.3 的 Objective-C 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`Framework/OpenIMCore.xcframework/.../Open_im_sdk.objc.h`。

```objective-c showLineNumbers
FOUNDATION_EXPORT void Open_im_sdkGetGroupMemberListByJoinTimeFilter(id<Open_im_sdk_callbackBase> _Nullable callback, NSString* _Nullable operationID, NSString* _Nullable groupID, int32_t offset, int32_t count, int64_t joinTimeBegin, int64_t joinTimeEnd, NSString* _Nullable filterUserIDList);
```

## 输入参数

| 参数名称         | 参数类型                     | 是否必填   | 描述                     |
| ---------------- | ---------------------------- | ---------- | ------------------------ |
| callback         | id<Open_im_sdk_callbackBase> | 是         | 回调或监听器。           |
| groupID          | NSString\*                   | 按方法原型 | 群组 ID 或群组 ID 列表。 |
| offset           | int32_t                      | 按方法原型 | 分页参数。               |
| count            | int32_t                      | 按方法原型 | 分页参数。               |
| joinTimeBegin    | int64_t                      | 按方法原型 | 按方法原型传入。         |
| joinTimeEnd      | int64_t                      | 按方法原型 | 按方法原型传入。         |
| filterUserIDList | NSString\*                   | 按方法原型 | 用户 ID 或用户 ID 列表。 |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。
