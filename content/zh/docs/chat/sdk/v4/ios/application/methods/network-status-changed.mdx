通知 SDK 网络状态已变化，当前实现会触发长连接关闭并进入后续重连处理。

## 功能介绍

通知 SDK 网络状态已变化，当前实现会触发长连接关闭并进入后续重连处理。

初始化类方法应在应用进程内按需调用；用户态方法通常要求 `InitSDK` 已成功执行，并通过 `operationID` 串联日志。

## Core Binding 方法

| 项目     | 说明                                                      |
| -------- | --------------------------------------------------------- |
| 方法签名 | `NetworkStatusChanged(callback Base, operationID string)` |
| 所属能力 | SDK 生命周期                                              |
| 源码位置 | `open_im_sdk/init_login.go`                               |

## iOS 调用参考

iOS Core SDK v3.8.3 的 Objective-C 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`Framework/OpenIMCore.xcframework/.../Open_im_sdk.objc.h`。

```objective-c showLineNumbers
FOUNDATION_EXPORT void Open_im_sdkNetworkStatusChanged(id<Open_im_sdk_callbackBase> _Nullable callback, NSString* _Nullable operationID);
```

## 输入参数

| 参数名称 | 参数类型                     | 是否必填 | 描述           |
| -------- | ---------------------------- | -------- | -------------- |
| callback | id<Open_im_sdk_callbackBase> | 是       | 回调或监听器。 |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。

## 代码示例

以下示例展示 iOS Core SDK v3.8.3 的调用骨架，请按业务上下文补充真实参数、错误处理和 UI 状态更新。

```objective-c showLineNumbers
Open_im_sdkNetworkStatusChanged(callback, @"operationID");
```
