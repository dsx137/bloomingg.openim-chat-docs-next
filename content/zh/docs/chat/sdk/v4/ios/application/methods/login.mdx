使用业务服务端签发的 token 登录指定用户。登录成功后才能可靠调用会话、消息、群组、好友等用户态接口。

## 功能介绍

使用业务服务端签发的 token 登录指定用户。登录成功后才能可靠调用会话、消息、群组、好友等用户态接口。

初始化类方法应在应用进程内按需调用；用户态方法通常要求 `InitSDK` 已成功执行，并通过 `operationID` 串联日志。

## Core Binding 方法

| 项目     | 说明                                                             |
| -------- | ---------------------------------------------------------------- |
| 方法签名 | `Login(callback Base, operationID string, userID, token string)` |
| 所属能力 | 初始化与登录                                                     |
| 源码位置 | `open_im_sdk/init_login.go`                                      |

## iOS 调用参考

iOS Core SDK 3.8.3-hotfix.14-dynamic.1 的 Objective-C 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`Framework/OpenIMCore.xcframework/.../Open_im_sdk.objc.h`。

```objective-c showLineNumbers
FOUNDATION_EXPORT void Open_im_sdkLogin(id<Open_im_sdk_callbackBase> _Nullable callback, NSString* _Nullable operationID, NSString* _Nullable userID, NSString* _Nullable token);
```

## 输入参数

| 参数名称 | 参数类型                     | 是否必填   | 描述                     |
| -------- | ---------------------------- | ---------- | ------------------------ |
| callback | id<Open_im_sdk_callbackBase> | 是         | 回调或监听器。           |
| userID   | NSString\*                   | 按方法原型 | 用户 ID 或用户 ID 列表。 |
| token    | NSString\*                   | 按方法原型 | 按方法原型传入。         |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。
