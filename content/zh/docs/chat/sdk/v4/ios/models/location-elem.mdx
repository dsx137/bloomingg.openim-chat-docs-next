`LocationElem` 是 iOS Core SDK 的数据模型，完整类名为 `io.openim.ios.sdk.models.LocationElem`。位置消息元素。

| ----------- | ---------------- | -------- |
| description | String           | 位置描述 |
| longitude   | double           | 经度     |
| latitude    | double           | 纬度     |
