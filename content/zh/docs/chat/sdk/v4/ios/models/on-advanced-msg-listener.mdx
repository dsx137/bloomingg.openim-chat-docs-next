`OnAdvancedMsgListener` 是 iOS Core SDK 文档中展示 core binding 方法签名时使用的回调类型，完整类名为 `open_im_sdk_callback.OnAdvancedMsgListener`。core binding 高级消息监听器，iOS 调用层对应 OnAdvanceMsgListener。

## 回调方法

| 方法                    | 参数                   | 描述               |
| ----------------------- | ---------------------- | ------------------ |
| OnRecvNewMessage        | message: string        | 收到单条新消息。   |
| OnRecvC2CReadReceipt    | msgReceiptList: string | 收到单聊已读回执。 |
| OnNewRecvMessageRevoked | messageRevoked: string | 收到消息撤回事件。 |
| OnRecvOfflineNewMessage | message: string        | 收到离线新消息。   |
| OnMsgDeleted            | message: string        | 消息被删除。       |
| OnRecvOnlineOnlyMessage | message: string        | 收到仅在线消息。   |
