`GroupHasReadInfo` 是 iOS Core SDK 的数据模型，完整类名为 `io.openim.ios.sdk.models.GroupHasReadInfo`。群消息已读成员信息。

## 字段说明

| 字段名称     | Objective-C 类型 | 描述   |
| ------------ | ---------------- | ------ |
| hasReadCount | int              | 已读数 |
| unreadCount  | int              | 未读数 |
