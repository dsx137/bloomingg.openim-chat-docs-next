`CustomElem` 是 iOS Core SDK 的数据模型，完整类名为 `io.openim.ios.sdk.models.CustomElem`。自定义消息元素，用于携带业务自定义数据。

## 字段说明

| 字段名称    | Objective-C 类型 | 描述                   |
| ----------- | ---------------- | ---------------------- |
| data        | String           | SDK 返回或提交的字段。 |
| extension   | String           | SDK 返回或提交的字段。 |
| description | String           | SDK 返回或提交的字段。 |
