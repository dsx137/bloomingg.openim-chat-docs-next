`AtUserInfo` 是 iOS Core SDK 的数据模型，完整类名为 `io.openim.ios.sdk.models.AtUserInfo`。被提及用户的 ID 与群昵称映射信息。

## 类型说明

at消息 userID跟nickname映射关系

## 字段说明

| 字段名称      | Objective-C 类型 | 描述             |
| ------------- | ---------------- | ---------------- |
| atUserID      | String           | 被at的用户的id   |
| groupNickname | String           | 被at的用户的昵称 |
