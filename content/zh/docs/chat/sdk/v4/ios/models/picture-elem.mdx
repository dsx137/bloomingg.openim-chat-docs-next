`PictureElem` 是 iOS Core SDK 的数据模型，完整类名为 `io.openim.ios.sdk.models.PictureElem`。图片消息元素，包含图片路径、地址和多尺寸图片信息。

## 字段说明

| 字段名称        | Objective-C 类型 | 描述         |
| --------------- | ---------------- | ------------ |
| sourcePath      | String           | 本地资源地址 |
| sourcePicture   | PictureInfo      | 本地图片详情 |
| bigPicture      | PictureInfo      | 大图详情     |
| snapshotPicture | PictureInfo      | 缩略图详情   |

## 关联类型

- [图片资源信息](/docs/chat/sdk/v4/ios/models/picture-info)（`PictureInfo`）：图片资源信息，包含 UUID、URL、尺寸和大小。
