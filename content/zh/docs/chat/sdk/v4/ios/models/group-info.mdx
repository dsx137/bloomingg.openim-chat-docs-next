`GroupInfo` 是 iOS Core SDK 的数据模型，完整类名为 `io.openim.ios.sdk.models.GroupInfo`。群组资料对象，包含群名、公告、群主、成员数和入群设置。

## 字段说明

| 字段名称               | Objective-C 类型 | 描述                                                |
| ---------------------- | ---------------- | --------------------------------------------------- |
| groupID                | String           | 组ID                                                |
| groupName              | String           | 群名                                                |
| notification           | String           | 群公告                                              |
| introduction           | String           | 群简介                                              |
| faceURL                | String           | 群头像                                              |
| ownerUserID            | String           | 群主id                                              |
| createTime             | Long             | 创建时间                                            |
| memberCount            | Integer          | 群成员数量                                          |
| status                 | Integer          | 群状态： ok = 0 blocked = 1 Dismissed = 2 Muted = 3 |
| creatorUserID          | String           | 创建者id                                            |
| groupType              | Integer          | 群类型                                              |
| ex                     | String           | 扩展字段                                            |
| needVerification       | Integer          | 进群验证方式 GroupVerification                      |
| lookMemberInfo         | Integer          | 不允许通过群获取成员资料 0：关闭，1：打开           |
| applyMemberFriend      | Integer          | 不允许通过群添加好友 0：关闭，1：打开               |
| notificationUpdateTime | Long             | 通知更新时间                                        |
| notificationUserID     | String           | 通知更新人                                          |
