`GroupVerification` 是 iOS Core SDK 的枚举常量，完整类名为 `io.openim.ios.sdk.enums.GroupVerification`。入群验证方式常量。

## 类型说明

进群验证方式

## 取值说明

| 常量                                    | 值  | 描述                                 |
| --------------------------------------- | --- | ------------------------------------ |
| APPLY_NEED_VERIFICATION_INVITE_DIRECTLY | 0   | 申请需要同意 邀请直接进              |
| ALL_NEED_VERIFICATION                   | 1   | 所有人进群需要验证，除了群主管理员邀 |
| DIRECTLY                                | 2   | 直接进群                             |
