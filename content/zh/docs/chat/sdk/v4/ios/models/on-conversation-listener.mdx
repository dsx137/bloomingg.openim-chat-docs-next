`OnConversationListener` 是 iOS Core SDK 的监听器，完整类名为 `io.openim.ios.sdk.listener.OnConversationListener`。会话新增、变更、同步和未读数变化监听器。

## 类型说明

会话监听

## 回调方法

| 方法                                 | 参数                        | 描述                              |
| ------------------------------------ | --------------------------- | --------------------------------- |
| onConversationChanged                | List<ConversationInfo> list | 会话列表有数据更新 可刷新会话列表 |
| onNewConversation                    | List<ConversationInfo> list | 新增会话 可刷新会话列表           |
| onSyncServerFailed                   | boolean reinstalled         | 事件触发时回调。                  |
| onSyncServerFinish                   | boolean reinstalled         | 事件触发时回调。                  |
| onSyncServerStart                    | boolean reinstalled         | 事件触发时回调。                  |
| onSyncServerProgress                 | long progress               | 事件触发时回调。                  |
| onTotalUnreadMessageCountChanged     | int i                       | 未读消息总数改变                  |
| onConversationUserInputStatusChanged | String s                    | 事件触发时回调。                  |

## 关联类型

- [会话信息](/docs/chat/sdk/v4/ios/models/conversation-info)（`ConversationInfo`）：会话资料对象，用于会话列表、会话详情和会话事件。
