`SoundElem` 是 iOS Core SDK 的数据模型，完整类名为 `io.openim.ios.sdk.models.SoundElem`。语音消息元素。

## 字段说明

| 字段名称  | Objective-C 类型 | 描述         |
| --------- | ---------------- | ------------ |
| uuid      | String           | 唯一ID       |
| soundPath | String           | 本地资源地址 |
| sourceUrl | String           | oss地址      |
| dataSize  | long             | 音频大小     |
| duration  | long             | 音频时长 s   |
