`PutArgs` 是 iOS Core SDK 的数据模型，完整类名为 `io.openim.ios.sdk.models.PutArgs`。文件上传参数。

## 字段说明

| 字段名称 | Objective-C 类型 | 描述         |
| -------- | ---------------- | ------------ |
| filepath | String           | 本地文件路径 |
| name     | String           | 文件名       |
| cuase    | String           | 类型         |
