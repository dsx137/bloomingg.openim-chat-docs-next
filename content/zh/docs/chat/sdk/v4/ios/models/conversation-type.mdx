`ConversationType` 是 iOS Core SDK 的枚举常量，完整类名为 `io.openim.ios.sdk.enums.ConversationType`。会话类型常量，用于区分单聊、群聊和通知会话。

## 取值说明

| 常量             | 值  | 描述                           |
| ---------------- | --- | ------------------------------ |
| SINGLE_CHAT      | 1   | 当前会话是单聊                 |
| GROUP_CHAT       | 2   | GROUP_CHAT 对应的 SDK 常量。   |
| SUPER_GROUP_CHAT | 3   | 当前会话是群聊                 |
| NOTIFICATION     | 4   | NOTIFICATION 对应的 SDK 常量。 |
