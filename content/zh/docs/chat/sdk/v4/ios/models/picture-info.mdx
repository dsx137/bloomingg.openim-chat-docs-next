`PictureInfo` 是 iOS Core SDK 的数据模型，完整类名为 `io.openim.ios.sdk.models.PictureInfo`。图片资源信息，包含 UUID、URL、尺寸和大小。

## 字段说明

| 字段名称 | Objective-C 类型 | 描述        |
| -------- | ---------------- | ----------- |
| uuid     | String           | 唯一ID      |
| type     | String           | 图片类型    |
| size     | long             | 图片大小    |
| width    | int              | 图片宽度    |
| height   | int              | 图片高度    |
| url      | String           | 图片oss地址 |
