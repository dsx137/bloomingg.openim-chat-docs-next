`AttachedInfoElem` 是 iOS Core SDK 的数据模型，完整类名为 `io.openim.ios.sdk.models.AttachedInfoElem`。消息附加信息元素，常用于携带群消息已读等扩展数据。

## 字段说明

| 字段名称                  | Objective-C 类型 | 描述                                                             |
| ------------------------- | ---------------- | ---------------------------------------------------------------- |
| groupHasReadInfo          | GroupHasReadInfo | 组信息已读详细                                                   |
| isPrivateChat             | boolean          | 是否是私聊消息（阅后即焚消息）                                   |
| hasReadTime               | long             | 消息已读时间                                                     |
| burnDuration              | int              | 阅读时长 s 即从hasReadTime时间算起，超过了burnDuration秒触发销毁 |
| notSenderNotificationPush | boolean          | SDK 返回或提交的字段。                                           |
| isEncryption              | boolean          | SDK 返回或提交的字段。                                           |
| inEncryptStatus           | boolean          | SDK 返回或提交的字段。                                           |
