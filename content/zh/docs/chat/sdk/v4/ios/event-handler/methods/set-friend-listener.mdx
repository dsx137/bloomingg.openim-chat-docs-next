设置好友关系监听器，用于接收好友申请、好友新增或删除、好友资料变化以及黑名单变化。

## 功能介绍

设置好友关系监听器，用于接收好友申请、好友新增或删除、好友资料变化以及黑名单变化。

`Set*Listener` 会替换当前 SDK 保存的同类监听器；如果多个模块都要消费事件，应在业务层封装统一分发。

## Core Binding 方法

| 项目     | 说明                                               |
| -------- | -------------------------------------------------- |
| 方法签名 | `SetFriendListener(listener OnFriendshipListener)` |
| 所属能力 | 监听器                                             |
| 源码位置 | `open_im_sdk/listener.go`                          |

## iOS 调用参考

iOS Core SDK 3.8.3-hotfix.14-dynamic.1 的 Objective-C 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`Framework/OpenIMCore.xcframework/.../Open_im_sdk.objc.h`。

```objective-c showLineNumbers
FOUNDATION_EXPORT void Open_im_sdkSetFriendListener(id<Open_im_sdk_callbackOnFriendshipListener> _Nullable listener);
```

## 输入参数

| 参数名称 | 参数类型                                     | 是否必填   | 描述           |
| -------- | -------------------------------------------- | ---------- | -------------- |
| listener | id<Open_im_sdk_callbackOnFriendshipListener> | 按方法原型 | 回调或监听器。 |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。
