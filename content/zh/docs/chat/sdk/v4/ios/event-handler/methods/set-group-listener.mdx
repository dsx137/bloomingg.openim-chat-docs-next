设置群组事件监听器，用于接收已加入群变化、群资料变化、群成员变化和群申请处理结果。

## 功能介绍

设置群组事件监听器，用于接收已加入群变化、群资料变化、群成员变化和群申请处理结果。

`Set*Listener` 会替换当前 SDK 保存的同类监听器；如果多个模块都要消费事件，应在业务层封装统一分发。

## Core Binding 方法

| 项目     | 说明                                         |
| -------- | -------------------------------------------- |
| 方法签名 | `SetGroupListener(listener OnGroupListener)` |
| 所属能力 | 监听器                                       |
| 源码位置 | `open_im_sdk/listener.go`                    |

## iOS 调用参考

iOS Core SDK v3.8.3 的 Objective-C 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`Framework/OpenIMCore.xcframework/.../Open_im_sdk.objc.h`。

```objective-c showLineNumbers
FOUNDATION_EXPORT void Open_im_sdkSetGroupListener(id<Open_im_sdk_callbackOnGroupListener> _Nullable listener);
```

## 输入参数

| 参数名称 | 参数类型                                | 是否必填   | 描述           |
| -------- | --------------------------------------- | ---------- | -------------- |
| listener | id<Open_im_sdk_callbackOnGroupListener> | 按方法原型 | 回调或监听器。 |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。
