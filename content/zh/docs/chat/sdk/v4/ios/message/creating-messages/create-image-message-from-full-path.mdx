通过完整图片路径创建图片消息。适合调用方持有绝对路径的场景。

## 功能介绍

通过完整图片路径创建图片消息。适合调用方持有绝对路径的场景。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                              |
| -------- | --------------------------------------------------------------------------------- |
| 方法签名 | `CreateImageMessageFromFullPath(operationID string, imageFullPath string) string` |
| 所属能力 | 会话与消息                                                                        |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                 |

FOUNDATION_EXPORT NSString* _Nonnull Open_im_sdkCreateImageMessageFromFullPath(NSString* _Nullable operationID, NSString* _Nullable imageFullPath);

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。
