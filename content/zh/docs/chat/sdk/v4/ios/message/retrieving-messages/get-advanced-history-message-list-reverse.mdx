## 功能介绍

按反向方向获取历史消息，适合围绕游标向另一方向加载。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                                |
| -------- | --------------------------------------------------------------------------------------------------- |
| 方法签名 | `GetAdvancedHistoryMessageListReverse(callback Base, operationID string, getMessageOptions string)` |
| 所属能力 | 会话与消息                                                                                          |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                                   |

## iOS 调用参考

iOS Core SDK v3.8.3 的 Objective-C 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`Framework/OpenIMCore.xcframework/.../Open_im_sdk.objc.h`。

```objective-c showLineNumbers
FOUNDATION_EXPORT void Open_im_sdkGetAdvancedHistoryMessageListReverse(id<Open_im_sdk_callbackBase> _Nullable callback, NSString* _Nullable operationID, NSString* _Nullable getMessageOptions);
```

## 输入参数

| 参数名称          | 参数类型                     | 是否必填   | 描述                          |
| ----------------- | ---------------------------- | ---------- | ----------------------------- |
| callback          | id<Open_im_sdk_callbackBase> | 是         | 回调或监听器。                |
| getMessageOptions | NSString\*                   | 按方法原型 | 消息对象、消息内容或消息 ID。 |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。
