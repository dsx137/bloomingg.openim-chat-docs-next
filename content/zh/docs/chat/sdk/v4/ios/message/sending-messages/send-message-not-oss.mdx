发送不走 SDK OSS 上传链路的消息。适合业务自定义上传、资源 URL 已经准备好或使用 `Create*ByURL` 方法创建的消息。

## 功能介绍

发送不走 SDK OSS 上传链路的消息。适合业务自定义上传、资源 URL 已经准备好或使用 `Create*ByURL` 方法创建的消息。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                                                                          |
| -------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| 方法签名 | `SendMessageNotOss(callback SendMsgCallBack, operationID string, message, recvID, groupID string, offlinePushInfo string, isOnlineOnly bool)` |
| 所属能力 | 会话与消息                                                                                                                                    |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                                                                             |

## iOS 调用参考

iOS Core SDK v3.8.3 的 Objective-C 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`Framework/OpenIMCore.xcframework/.../Open_im_sdk.objc.h`。

```objective-c showLineNumbers
FOUNDATION_EXPORT void Open_im_sdkSendMessageNotOss(id<Open_im_sdk_callbackSendMsgCallBack> _Nullable callback, NSString* _Nullable operationID, NSString* _Nullable message, NSString* _Nullable recvID, NSString* _Nullable groupID, NSString* _Nullable offlinePushInfo, BOOL isOnlineOnly);
```

## 输入参数

| 参数名称        | 参数类型                                | 是否必填   | 描述                                   |
| --------------- | --------------------------------------- | ---------- | -------------------------------------- |
| callback        | id<Open_im_sdk_callbackSendMsgCallBack> | 是         | 回调或监听器。                         |
| message         | NSString\*                              | 按方法原型 | 消息对象、消息内容或消息 ID。          |
| recvID          | NSString\*                              | 按方法原型 | 按方法原型传入。                       |
| groupID         | NSString\*                              | 按方法原型 | 群组 ID 或群组 ID 列表。               |
| offlinePushInfo | NSString\*                              | 按方法原型 | 请求对象，字段以对应平台类型定义为准。 |
| isOnlineOnly    | BOOL                                    | 按方法原型 | 按方法原型传入。                       |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。
