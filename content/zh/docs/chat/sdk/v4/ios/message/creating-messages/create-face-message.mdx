创建表情消息。适合业务自定义表情包或贴纸消息。

## 功能介绍

创建表情消息。适合业务自定义表情包或贴纸消息。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                   |
| -------- | ---------------------------------------------------------------------- |
| 方法签名 | `CreateFaceMessage(operationID string, index int, data string) string` |
| 所属能力 | 会话与消息                                                             |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                      |

## iOS 调用参考

iOS Core SDK 3.8.3-hotfix.14-dynamic.1 的 Objective-C 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`Framework/OpenIMCore.xcframework/.../Open_im_sdk.objc.h`。

```objective-c showLineNumbers
FOUNDATION_EXPORT NSString* _Nonnull Open_im_sdkCreateFaceMessage(NSString* _Nullable operationID, long index, NSString* _Nullable data);
```

## 输入参数

| 参数名称 | 参数类型   | 是否必填   | 描述                                   |
| -------- | ---------- | ---------- | -------------------------------------- |
| index    | long       | 按方法原型 | 按方法原型传入。                       |
| data     | NSString\* | 按方法原型 | 请求对象，字段以对应平台类型定义为准。 |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。
