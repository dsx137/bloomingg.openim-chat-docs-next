检查一批用户与当前用户是否为好友关系，适合展示添加好友按钮或发起关系操作前校验。

## 功能介绍

检查一批用户与当前用户是否为好友关系，适合展示添加好友按钮或发起关系操作前校验。

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。

## Core Binding 方法

| 项目     | 说明                                                                |
| -------- | ------------------------------------------------------------------- |
| 方法签名 | `CheckFriend(callback Base, operationID string, userIDList string)` |
| 所属能力 | 好友                                                                |
| 源码位置 | `open_im_sdk/relation.go`                                           |

## iOS 调用参考

iOS Core SDK v3.8.3 的 Objective-C 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`Framework/OpenIMCore.xcframework/.../Open_im_sdk.objc.h`。

```objective-c showLineNumbers
FOUNDATION_EXPORT void Open_im_sdkCheckFriend(id<Open_im_sdk_callbackBase> _Nullable callback, NSString* _Nullable operationID, NSString* _Nullable userIDList);
```

## 输入参数

| 参数名称   | 参数类型                     | 是否必填   | 描述                     |
| ---------- | ---------------------------- | ---------- | ------------------------ |
| callback   | id<Open_im_sdk_callbackBase> | 是         | 回调或监听器。           |
| userIDList | NSString\*                   | 按方法原型 | 用户 ID 或用户 ID 列表。 |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。
