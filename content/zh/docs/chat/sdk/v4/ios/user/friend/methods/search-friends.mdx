## 功能介绍

搜索好友，适合按用户 ID、昵称、备注等当前搜索结构支持的字段查找好友。

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。

## Core Binding 方法

| 项目     | 说明                                                                   |
| -------- | ---------------------------------------------------------------------- |
| 方法签名 | `SearchFriends(callback Base, operationID string, searchParam string)` |
| 所属能力 | 好友                                                                   |
| 源码位置 | `open_im_sdk/relation.go`                                              |

## iOS 调用参考

iOS Core SDK 3.8.3-hotfix.14-dynamic.1 的 Objective-C 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`Framework/OpenIMCore.xcframework/.../Open_im_sdk.objc.h`。

```objective-c showLineNumbers
FOUNDATION_EXPORT void Open_im_sdkSearchFriends(id<Open_im_sdk_callbackBase> _Nullable callback, NSString* _Nullable operationID, NSString* _Nullable searchParam);
```

## 输入参数

| 参数名称    | 参数类型                     | 是否必填   | 描述                                   |
| ----------- | ---------------------------- | ---------- | -------------------------------------- |
| callback    | id<Open_im_sdk_callbackBase> | 是         | 回调或监听器。                         |
| searchParam | NSString\*                   | 按方法原型 | 请求对象，字段以对应平台类型定义为准。 |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。
