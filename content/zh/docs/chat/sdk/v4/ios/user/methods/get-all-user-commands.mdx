获取指定类型下的全部用户命令数据，适合恢复收藏列表或同步用户侧命令状态。

## 功能介绍

获取指定类型下的全部用户命令数据，适合恢复收藏列表或同步用户侧命令状态。

用户资料和在线状态方法通常要求当前用户已登录；在线状态变化通过 `SetUserListener` 设置的监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                |
| -------- | ------------------------------------------------------------------- |
| 方法签名 | `GetAllUserCommands(callback Base, operationID string, Type int32)` |
| 所属能力 | 用户                                                                |
| 源码位置 | `open_im_sdk/user.go`                                               |

## iOS 调用参考

iOS Core SDK v3.8.3 的 Objective-C 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`Framework/OpenIMCore.xcframework/.../Open_im_sdk.objc.h`。

```objective-c showLineNumbers
FOUNDATION_EXPORT void Open_im_sdkGetAllUserCommands(id<Open_im_sdk_callbackBase> _Nullable callback, NSString* _Nullable operationID, int32_t Type);
```

## 输入参数

| 参数名称 | 参数类型                     | 是否必填   | 描述             |
| -------- | ---------------------------- | ---------- | ---------------- |
| callback | id<Open_im_sdk_callbackBase> | 是         | 回调或监听器。   |
| Type     | int32_t                      | 按方法原型 | 按方法原型传入。 |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。

以下示例展示 iOS Core SDK v3.8.3 的调用骨架，请按业务上下文补充真实参数、错误处理和 UI 状态更新。

Open_im_sdkGetAllUserCommands(callback, @"operationID", 0);
