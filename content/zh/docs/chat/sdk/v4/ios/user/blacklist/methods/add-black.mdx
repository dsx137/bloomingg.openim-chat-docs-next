## 功能介绍

将指定用户加入黑名单，适合屏蔽对方消息或限制关系互动。具体业务影响由 SDK 和服务端策略决定。

黑名单新增和移除通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；屏蔽后的业务效果以当前服务端策略为准。

## iOS 调用参考

iOS Core SDK 3.8.3-hotfix.14-dynamic.1 的 Objective-C 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`Framework/OpenIMCore.xcframework/.../Open_im_sdk.objc.h`。

```objective-c showLineNumbers
FOUNDATION_EXPORT void Open_im_sdkAddBlack(id<Open_im_sdk_callbackBase> _Nullable callback, NSString* _Nullable operationID, NSString* _Nullable blackUserID, NSString* _Nullable ex);
```

## 输入参数

| 参数名称    | 参数类型                     | 是否必填   | 描述                     |
| ----------- | ---------------------------- | ---------- | ------------------------ |
| callback    | id<Open_im_sdk_callbackBase> | 是         | 回调或监听器。           |
| blackUserID | NSString\*                   | 按方法原型 | 用户 ID 或用户 ID 列表。 |
| ex          | NSString\*                   | 按方法原型 | 按方法原型传入。         |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。
