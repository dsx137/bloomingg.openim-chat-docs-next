获取当前已订阅用户的在线状态快照，适合页面进入时先拉一次状态，再等待事件更新。

## 功能介绍

获取当前已订阅用户的在线状态快照，适合页面进入时先拉一次状态，再等待事件更新。

用户资料和在线状态方法通常要求当前用户已登录；在线状态变化通过 `SetUserListener` 设置的监听器接收。

## Core Binding 方法

| 项目     | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| 方法签名 | `GetSubscribeUsersStatus(callback Base, operationID string)` |
| 所属能力 | 用户                                                         |
| 源码位置 | `open_im_sdk/online.go`                                      |

## iOS 调用参考

iOS Core SDK v3.8.3 的 Objective-C 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`Framework/OpenIMCore.xcframework/.../Open_im_sdk.objc.h`。

```objective-c showLineNumbers
FOUNDATION_EXPORT void Open_im_sdkGetSubscribeUsersStatus(id<Open_im_sdk_callbackBase> _Nullable callback, NSString* _Nullable operationID);
```

## 输入参数

| 参数名称 | 参数类型                     | 是否必填 | 描述           |
| -------- | ---------------------------- | -------- | -------------- |
| callback | id<Open_im_sdk_callbackBase> | 是       | 回调或监听器。 |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。
