会话和消息方法来自 `open_im_sdk/conversation_msg.go`。创建消息类方法同步返回消息对象；发送、查询、删除、已读等方法通过回调返回。

本页是会话与消息方法目录。每个子模块都有独立概述页，每个方法都有独立说明页。
