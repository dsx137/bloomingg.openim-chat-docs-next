解散指定群组，通常要求当前用户具备群主或管理权限。

## 功能介绍

解散指定群组，通常要求当前用户具备群主或管理权限。

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。

## Core Binding 方法

| 项目     | 说明                                                              |
| -------- | ----------------------------------------------------------------- |
| 方法签名 | `DismissGroup(callback Base, operationID string, groupID string)` |
| 所属能力 | 群组                                                              |
| 源码位置 | `open_im_sdk/group.go`                                            |

## uni-app 调用参考

uni-app SDK v3.8.3 的 JavaScript 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`ios/OpenIMUniPlugin/OpenIMUniPlugin/Classes/OpenIMModule.m`。

```js showLineNumbers
openIM.dismissGroup(operationID, groupID, callback);
```

## 输入参数

| 参数名称 | 参数类型 | 是否必填   | 描述                              |
| -------- | -------- | ---------- | --------------------------------- |
| groupID  | string   | 按方法原型 | 按 uni-app 原生插件导出方法传入。 |
| callback | Function | 是         | 按 uni-app 原生插件导出方法传入。 |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。

## 代码示例

以下示例展示 uni-app SDK v3.8.3 的调用骨架，请按业务上下文补充真实参数、错误处理和 UI 状态更新。

```js showLineNumbers
const openIM = uni.requireNativePlugin('OpenIMUniPlugin');

openIM.dismissGroup('operationID', 'groupID', callback);
```
