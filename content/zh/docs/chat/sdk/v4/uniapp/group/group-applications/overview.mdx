入群申请用于处理用户主动申请加入群组，以及群主或管理员审核申请的流程。

## 方法索引

| 方法                                                                                                            | 说明                                                         |
| --------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------ |
| [申请加入群组](/docs/chat/sdk/v4/uniapp/group/group-applications/join-group)                                    | 申请加入指定群组。是否直接入群取决于群设置和服务端审核策略。 |
| [获取收到的入群申请](/docs/chat/sdk/v4/uniapp/group/group-applications/get-group-application-list-as-recipient) | 获取当前用户作为群管理者收到的入群申请列表。                 |
| [获取发出的入群申请](/docs/chat/sdk/v4/uniapp/group/group-applications/get-group-application-list-as-applicant) | 获取当前用户自己提交的入群申请列表。                         |
| [同意入群申请](/docs/chat/sdk/v4/uniapp/group/group-applications/accept-group-application)                      | 同意指定用户的入群申请。                                     |
| [拒绝入群申请](/docs/chat/sdk/v4/uniapp/group/group-applications/refuse-group-application)                      | 拒绝指定用户的入群申请。                                     |
| [获取未处理入群申请数](/docs/chat/sdk/v4/uniapp/group/group-applications/get-group-application-unhandled-count) | 获取未处理入群申请数量，适合管理入口角标和后台同步。         |

## 相关类型

- [群申请信息](/docs/chat/sdk/v4/uniapp/models/group-application-info)（`GroupApplicationInfo`）：入群申请记录，用于申请列表和审核流程。
- [收到的群申请查询条件](/docs/chat/sdk/v4/uniapp/models/get-group-application-list-as-recipient-req)（`GetGroupApplicationListAsRecipientReq`）：查询收到的入群申请时使用的分页条件。
- [发出的群申请查询条件](/docs/chat/sdk/v4/uniapp/models/get-group-application-list-as-applicant-req)（`GetGroupApplicationListAsApplicantReq`）：查询当前用户发出的入群申请时使用的分页条件。
