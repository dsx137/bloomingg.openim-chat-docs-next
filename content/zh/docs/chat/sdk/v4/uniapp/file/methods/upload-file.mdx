## 功能介绍

上传文件到 SDK 配置的对象存储链路，适合发送文件、图片、音视频前先完成资源上传。

第三方能力至少要求 SDK 已初始化；日志和文件上传的进度通过对应 progress callback 返回。

## Core Binding 方法

| 项目     | 说明                                                                                     |
| -------- | ---------------------------------------------------------------------------------------- |
| 方法签名 | `UploadFile(callback Base, operationID string, req string, progress UploadFileCallback)` |
| 所属能力 | 文件上传 |
| 源码位置 | `open_im_sdk/third.go`                                                                   |

## uni-app 调用参考

uni-app SDK v3.8.3 的 JavaScript 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`ios/OpenIMUniPlugin/OpenIMUniPlugin/Classes/OpenIMModule.m`。

```js showLineNumbers
openIM.uploadFile(operationID, reqData, callback);
```

## 输入参数

| 参数名称 | 参数类型 | 是否必填   | 描述                              |
| -------- | -------- | ---------- | --------------------------------- |
| reqData  | string   | 按方法原型 | 按 uni-app 原生插件导出方法传入。 |
| callback | Function | 是         | 按 uni-app 原生插件导出方法传入。 |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。

## 代码示例

以下示例展示 uni-app SDK v3.8.3 的调用骨架，请按业务上下文补充真实参数、错误处理和 UI 状态更新。

```js showLineNumbers
const openIM = uni.requireNativePlugin('OpenIMUniPlugin');

openIM.uploadFile('operationID', {}, callback);
```
