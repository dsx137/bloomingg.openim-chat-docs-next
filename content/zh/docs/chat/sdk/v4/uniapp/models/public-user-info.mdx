`PublicUserInfo` 是 uni-app SDK 的数据模型，完整类名为 `io.openim.uniapp.sdk.models.PublicUserInfo`。公开用户资料。

| 字段名称   | JavaScript 类型 | 描述     |
| ---------- | --------------- | -------- |
| userID     | String          | 用户id   |
| nickname   | String          | 昵称     |
| faceURL    | String          | 头像     |
| ex         | String          | 扩展字段 |
| createTime | Long            | 创建时间 |
