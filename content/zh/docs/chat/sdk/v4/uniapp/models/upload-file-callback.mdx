`UploadFileCallback` 是 uni-app SDK 文档中展示 core binding 方法签名时使用的回调类型，完整类名为 `open_im_sdk_callback.UploadFileCallback`。core binding 文件上传回调，uni-app 调用层通常映射为 OnPutFileListener。

## 回调方法

| 方法               | 参数                                                   | 描述                                |
| ------------------ | ------------------------------------------------------ | ----------------------------------- |
| Open               | size: int64                                            | 文件打开并返回总大小。              |
| PartSize           | partSize: int64；num: int                              | 返回分片大小和分片数量。            |
| HashPartProgress   | index: int；size: int64；partHash: string              | 分片哈希计算进度。                  |
| HashPartComplete   | partsHash: string；fileHash: string                    | 所有分片哈希计算完成。              |
| UploadID           | uploadID: string                                       | 生成并返回上传 ID。                 |
| UploadPartComplete | index: int；partSize: int64；partHash: string          | 单个分片上传完成。                  |
| UploadComplete     | fileSize: int64；streamSize: int64；storageSize: int64 | 文件上传阶段完成并返回大小信息。    |
| Complete           | size: int64；url: string；typ: int                     | 文件上传完成并返回 URL 与文件类型。 |
