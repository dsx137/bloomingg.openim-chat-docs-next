`callback` 是 uni-app SDK 的回调接口，完整类名为 `io.openim.uniapp.sdk.listener.callback`。uni-app SDK 异步方法的通用成功和失败回调。

## 类型说明

api接口回调

## 回调方法

| 方法             | 参数                   | 描述 |
| ---------------- | ---------------------- | ---- |
| error callback   | int code；String error | 失败 |
| success callback | T data                 | 成功 |
