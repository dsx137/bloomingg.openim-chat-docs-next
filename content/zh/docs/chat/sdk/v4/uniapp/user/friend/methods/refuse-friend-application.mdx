拒绝指定好友申请。

## 功能介绍

拒绝指定好友申请。

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。

## Core Binding 方法

| 项目     | 说明                                                                                 |
| -------- | ------------------------------------------------------------------------------------ |
| 方法签名 | `RefuseFriendApplication(callback Base, operationID string, userIDHandleMsg string)` |
| 所属能力 | 好友                                                                                 |
| 源码位置 | `open_im_sdk/relation.go`                                                            |

## uni-app 调用参考

uni-app SDK v3.8.3 的 JavaScript 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`ios/OpenIMUniPlugin/OpenIMUniPlugin/Classes/OpenIMModule.m`。

```js showLineNumbers
openIM.refuseFriendApplication(operationID, userIDHandleMsg, callback);
```

## 输入参数

| 参数名称        | 参数类型 | 是否必填   | 描述                              |
| --------------- | -------- | ---------- | --------------------------------- |
| userIDHandleMsg | string   | 按方法原型 | 按 uni-app 原生插件导出方法传入。 |
| callback        | Function | 是         | 按 uni-app 原生插件导出方法传入。 |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。

## 代码示例

以下示例展示 uni-app SDK v3.8.3 的调用骨架，请按业务上下文补充真实参数、错误处理和 UI 状态更新。

```js showLineNumbers
const openIM = uni.requireNativePlugin('OpenIMUniPlugin');

openIM.refuseFriendApplication('operationID', 'userIDHandleMsg', callback);
```
