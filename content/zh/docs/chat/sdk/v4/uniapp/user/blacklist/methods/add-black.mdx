## 功能介绍

将指定用户加入黑名单，适合屏蔽对方消息或限制关系互动。具体业务影响由 SDK 和服务端策略决定。

黑名单新增和移除通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；屏蔽后的业务效果以当前服务端策略为准。

## Core Binding 方法

| 项目     | 说明                                                                         |
| -------- | ---------------------------------------------------------------------------- |
| 方法签名 | `AddBlack(callback Base, operationID string, blackUserID string, ex string)` |
| 所属能力 | 黑名单                                                                       |
| 源码位置 | `open_im_sdk/relation.go`                                                    |

## uni-app 调用参考

uni-app SDK v3.8.3 的 JavaScript 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`ios/OpenIMUniPlugin/OpenIMUniPlugin/Classes/OpenIMModule.m`。

```js showLineNumbers
openIM.addBlack(operationID, options, callback);
```

## 输入参数

| 参数名称 | 参数类型 | 是否必填   | 描述                              |
| -------- | -------- | ---------- | --------------------------------- |
| options  | Object   | 按方法原型 | 按 uni-app 原生插件导出方法传入。 |
| callback | Function | 是         | 按 uni-app 原生插件导出方法传入。 |

## 返回结果

同步创建类方法直接返回值；异步方法通过 callback 返回成功或失败。建议为每次调用传入可追踪的 operationID。
