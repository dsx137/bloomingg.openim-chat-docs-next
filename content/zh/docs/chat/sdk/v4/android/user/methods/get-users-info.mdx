`GetUsersInfo` 来自 `open_im_sdk/user.go、open_im_sdk/online.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                              |
| -------- | ----------------------------------------------------------------- |
| 方法     | `GetUsersInfo(callback Base, operationID string, userIDs string)` |
| 所属能力 | 用户                                                              |
| 源码位置 | `open_im_sdk/user.go、open_im_sdk/online.go`                      |

## 作用

批量获取指定用户公开资料，常用于会话列表、消息发送者资料补齐和好友资料刷新。

## 参数说明

`userIDs` 是用户 ID 列表 JSON。

## 返回与事件

成功返回用户资料列表 JSON。

## 使用边界

用户资料和在线状态方法通常要求当前用户已登录；在线状态变化通过 `SetUserListener` 设置的监听器接收。
