`GetAllUserCommands` 来自 `open_im_sdk/user.go、open_im_sdk/online.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                |
| -------- | ------------------------------------------------------------------- |
| 方法     | `GetAllUserCommands(callback Base, operationID string, Type int32)` |
| 所属能力 | 用户                                                                |
| 源码位置 | `open_im_sdk/user.go、open_im_sdk/online.go`                        |

## 作用

获取指定类型下的全部用户命令数据，适合恢复收藏列表或同步用户侧命令状态。

## 参数说明

`Type` 指定命令类型。

## 返回与事件

成功返回命令列表 JSON。

## 使用边界

用户资料和在线状态方法通常要求当前用户已登录；在线状态变化通过 `SetUserListener` 设置的监听器接收。
