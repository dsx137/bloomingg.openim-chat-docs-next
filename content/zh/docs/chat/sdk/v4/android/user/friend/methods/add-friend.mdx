`AddFriend` 来自 `open_im_sdk/relation.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                |
| -------- | ------------------------------------------------------------------- |
| 方法     | `AddFriend(callback Base, operationID string, userIDReqMsg string)` |
| 所属能力 | 好友                                                                |
| 源码位置 | `open_im_sdk/relation.go`                                           |

## 作用

向指定用户发送好友申请。是否自动通过取决于服务端策略和对方设置。

## 参数说明

`userIDReqMsg` 是好友申请请求 JSON，通常包含目标用户和申请说明。

## 返回与事件

成功返回申请发送结果，并可能触发好友申请事件。

## 使用边界

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。
