`RemoveBlack` 来自 `open_im_sdk/relation.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                  |
| -------- | --------------------------------------------------------------------- |
| 方法     | `RemoveBlack(callback Base, operationID string, removeUserID string)` |
| 所属能力 | 黑名单                                                                |
| 源码位置 | `open_im_sdk/relation.go`                                             |

## 作用

将指定用户移出黑名单，恢复与该用户的普通关系能力。

## 参数说明

`removeUserID` 是要移出黑名单的用户 ID。

## 返回与事件

成功返回处理结果，并触发黑名单移除事件。

## 使用边界

黑名单新增和移除通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；屏蔽后的业务效果以当前服务端策略为准。
