`SetSelfInfo` 来自 `open_im_sdk/user.go、open_im_sdk/online.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                              |
| -------- | ----------------------------------------------------------------- |
| 方法     | `SetSelfInfo(callback Base, operationID string, userInfo string)` |
| 所属能力 | 用户                                                              |
| 源码位置 | `open_im_sdk/user.go、open_im_sdk/online.go`                      |

## 作用

更新当前登录用户资料，例如昵称、头像或扩展字段。只应修改当前账号允许自行修改的字段。

## 参数说明

`userInfo` 是用户资料 JSON；字段以当前 SDK 用户结构为准。

## 返回与事件

成功返回更新结果，相关变更会触发用户监听。

## 使用边界

用户资料和在线状态方法通常要求当前用户已登录；在线状态变化通过 `SetUserListener` 设置的监听器接收。
