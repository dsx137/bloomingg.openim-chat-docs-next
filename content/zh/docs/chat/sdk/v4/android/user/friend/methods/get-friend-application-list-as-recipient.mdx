获取当前用户收到的好友申请列表，适合好友申请入口和审核页。

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。

## Core Binding 方法

| 项目     | 说明                                                                          |
| -------- | ----------------------------------------------------------------------------- |
| 方法签名 | `GetFriendApplicationListAsRecipient(callback Base, operationID, req string)` |
| 所属能力 | 好友                                                                          |
| 源码位置 | `open_im_sdk/relation.go`                                                     |

## Android 调用参考

Android SDK v3.8.3 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void getRecvFriendApplicationList(OnBase<List<FriendApplicationInfo>> base, GetFriendApplicationListAsRecipientReq recipientReq)
```

## 输入参数

| 参数名称     | 参数类型                                                                                                                                            | 是否必填 | 描述                 |
| ------------ | --------------------------------------------------------------------------------------------------------------------------------------------------- | -------- | -------------------- |
| base         | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)`<List<`[`FriendApplicationInfo`](/docs/chat/sdk/v4/android/models/friend-application-info)`>>` | 是       | 查询结果回调。       |
| recipientReq | [`GetFriendApplicationListAsRecipientReq`](/docs/chat/sdk/v4/android/models/get-friend-application-list-as-recipient-req)                           | 是       | 分页和筛选请求参数。 |

## 返回结果

`onSuccess` 返回当前页 `List<`[`FriendApplicationInfo`](/docs/chat/sdk/v4/android/models/friend-application-info)`>`，包含当前用户收到的且符合处理状态筛选的申请；查询或反序列化失败时调用 `onError`。

## 代码示例

以下示例展示 Android SDK v3.8.3 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().friendshipManager.getRecvFriendApplicationList(
    new OnBase<List<FriendApplicationInfo>>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(List<FriendApplicationInfo> data) {
        // TODO: 处理收到的好友申请列表
    }
},
    recipientReq
);
```
