分页获取好友列表，适合好友数量较多时按页加载。

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。

## Core Binding 方法

| 项目     | 说明                                                                                                |
| -------- | --------------------------------------------------------------------------------------------------- |
| 方法签名 | `GetFriendListPage(callback Base, operationID string, offset int32, count int32, filterBlack bool)` |
| 所属能力 | 好友                                                                                                |
| 源码位置 | `open_im_sdk/relation.go`                                                                           |

## Android 调用参考

Android SDK v3.8.3 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void getFriendListPage(OnBase<List<UserInfo>> base, int offset, int count, Boolean filterBlack)
```

## 输入参数

| 参数名称    | Java 类型                                                                                                                | 是否必填 | 描述                         |
| ----------- | ------------------------------------------------------------------------------------------------------------------------ | -------- | ---------------------------- |
| base        | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)`<List<`[`UserInfo`](/docs/chat/sdk/v4/android/models/user-info)`>>` | 是       | 好友列表分页结果回调。       |
| offset      | `int`                                                                                                                    | 是       | 从零开始的分页偏移量。       |
| count       | `int`                                                                                                                    | 是       | 本次最多返回的好友数量。     |
| filterBlack | `Boolean`                                                                                                                | 否       | 是否过滤已加入黑名单的用户。 |

## 返回结果

| 参数名称 | 参数类型                                                           | 描述     |
| -------- | ------------------------------------------------------------------ | -------- |
| data     | `List<`[`UserInfo`](/docs/chat/sdk/v4/android/models/user-info)`>` | 成功返回 |

## 代码示例

以下示例展示 Android SDK v3.8.3 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().friendshipManager.getFriendListPage(
    new OnBase<List<UserInfo>>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(List<UserInfo> data) {
        // TODO: 处理分页好友列表
    }
},
    offset,
    count,
    filterBlack
);
```
