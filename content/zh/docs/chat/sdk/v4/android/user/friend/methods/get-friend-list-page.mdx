`GetFriendListPage` 来自 `open_im_sdk/relation.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                                |
| -------- | --------------------------------------------------------------------------------------------------- |
| 方法     | `GetFriendListPage(callback Base, operationID string, offset int32, count int32, filterBlack bool)` |
| 所属能力 | 好友                                                                                                |
| 源码位置 | `open_im_sdk/relation.go`                                                                           |

## 作用

分页获取好友列表，适合好友数量较多时按页加载。

## 参数说明

`offset` 和 `count` 控制分页；`filterBlack` 控制是否排除黑名单用户。

## 返回与事件

成功返回当前页好友列表 JSON。

## 使用边界

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。
