`AddBlack` 来自 `open_im_sdk/relation.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                         |
| -------- | ---------------------------------------------------------------------------- |
| 方法     | `AddBlack(callback Base, operationID string, blackUserID string, ex string)` |
| 所属能力 | 黑名单                                                                       |
| 源码位置 | `open_im_sdk/relation.go`                                                    |

## 作用

将指定用户加入黑名单，适合屏蔽对方消息或限制关系互动。具体业务影响由 SDK 和服务端策略决定。

## 参数说明

`blackUserID` 是要加入黑名单的用户 ID；`ex` 是扩展信息。

## 返回与事件

成功返回处理结果，并触发黑名单新增事件。

## 使用边界

黑名单新增和移除通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；屏蔽后的业务效果以当前服务端策略为准。
