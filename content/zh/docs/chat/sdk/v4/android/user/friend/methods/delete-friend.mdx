`DeleteFriend` 来自 `open_im_sdk/relation.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                   |
| -------- | ---------------------------------------------------------------------- |
| 方法     | `DeleteFriend(callback Base, operationID string, friendUserID string)` |
| 所属能力 | 好友                                                                   |
| 源码位置 | `open_im_sdk/relation.go`                                              |

## 作用

删除指定好友关系，适合用户主动解除好友。

## 参数说明

`friendUserID` 是要删除的好友用户 ID。

## 返回与事件

成功返回删除结果，并触发好友删除事件。

## 使用边界

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。
