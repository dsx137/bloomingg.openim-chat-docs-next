删除指定好友关系，适合用户主动解除好友。

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。

## Core Binding 方法

| 项目     | 说明                                                                   |
| -------- | ---------------------------------------------------------------------- |
| 方法签名 | `DeleteFriend(callback Base, operationID string, friendUserID string)` |
| 所属能力 | 好友                                                                   |
| 源码位置 | `open_im_sdk/relation.go`                                              |

## Android 调用参考

Android SDK v3.8.3 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void deleteFriend(OnBase<String> base, String uid)
```

## 输入参数

| 参数名称 | 参数类型                                             | 是否必填 | 描述     |
| -------- | ---------------------------------------------------- | -------- | -------- |
| callBack | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base) | 是       | 回调接口 |
| uid      | String                                               | 是       | 用户 ID  |

## 返回结果

`onSuccess` 表示与目标用户的好友关系已删除并完成本地同步，回调字符串不含业务数据；目标不是好友或服务端请求失败时调用 `onError`。好友删除事件会同步通知业务层。

## 代码示例

以下示例展示 Android SDK v3.8.3 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().friendshipManager.deleteFriend(new OnBase<String>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(String data) {
        // TODO: 处理成功结果
    }
}, uid);
```
