获取当前用户黑名单列表，适合隐私设置页、拦截列表或关系状态恢复。

## 功能介绍

获取当前用户黑名单列表，适合隐私设置页、拦截列表或关系状态恢复。

黑名单新增和移除通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；屏蔽后的业务效果以当前服务端策略为准。

## Core Binding 方法

| 项目     | 说明                                              |
| -------- | ------------------------------------------------- |
| 方法签名 | `GetBlackList(callback Base, operationID string)` |
| 所属能力 | 黑名单                                            |
| 源码位置 | `open_im_sdk/relation.go`                         |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void getBlacklist(OnBase<List<UserInfo>> base)
```

## 输入参数

| 参数名称 | 参数类型                                                                                                                         | 是否必填 | 描述     |
| -------- | -------------------------------------------------------------------------------------------------------------------------------- | -------- | -------- |
| callBack | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)&lt;List&lt;[`UserInfo`](/docs/chat/sdk/v4/android/models/user-info)&gt;&gt; | 是       | 回调接口 |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().friendshipManager.getBlacklist(new OnBase<List<UserInfo>>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
```
