`GetBlackList` 来自 `open_im_sdk/relation.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                              |
| -------- | ------------------------------------------------- |
| 方法     | `GetBlackList(callback Base, operationID string)` |
| 所属能力 | 黑名单                                            |
| 源码位置 | `open_im_sdk/relation.go`                         |

## 作用

获取当前用户黑名单列表，适合隐私设置页、拦截列表或关系状态恢复。

## 参数说明

`operationID` 用于日志追踪；无其他业务参数。

## 返回与事件

成功返回黑名单用户列表 JSON。

## 使用边界

黑名单新增和移除通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；屏蔽后的业务效果以当前服务端策略为准。
