取消订阅指定用户在线状态，适合离开页面、好友列表卸载或减少无用状态推送。

## 功能介绍

取消订阅指定用户在线状态，适合离开页面、好友列表卸载或减少无用状态推送。

用户资料和在线状态方法通常要求当前用户已登录；在线状态变化通过 `SetUserListener` 设置的监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                        |
| -------- | --------------------------------------------------------------------------- |
| 方法签名 | `UnsubscribeUsersStatus(callback Base, operationID string, userIDs string)` |
| 所属能力 | 用户                                                                        |
| 源码位置 | `open_im_sdk/online.go`                                                     |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void unsubscribeOnlineUsersStatus(OnBase<String> callBack, List<String> uid)
```

## 输入参数

| 参数名称 | 参数类型       | 是否必填 | 描述                       |
| -------- | -------------- | -------- | -------------------------- |
| callBack | OnBase<String> | 是       | 取消订阅结果回调。         |
| uid      | List<String>   | 是       | 要取消订阅的用户 ID 列表。 |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().userInfoManager.unsubscribeOnlineUsersStatus(new OnBase<String>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(String data) {
        // TODO: 处理成功结果
    }
}, uidList);
```
