获取当前用户全部好友列表，适合好友通讯录首屏加载或登录后全量同步。

## 功能介绍

获取当前用户全部好友列表，适合好友通讯录首屏加载或登录后全量同步。

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。

## Core Binding 方法

| 项目     | 说明                                                                 |
| -------- | -------------------------------------------------------------------- |
| 方法签名 | `GetFriendList(callback Base, operationID string, filterBlack bool)` |
| 所属能力 | 好友                                                                 |
| 源码位置 | `open_im_sdk/relation.go`                                            |

## Android 调用参考

当前 Android SDK 的方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void getFriendList(OnBase<List<UserInfo>> base, Boolean filterBlack)
```

## 输入参数

| 参数名称    | 参数类型 | 是否必填 | 描述             |
| ----------- | -------- | -------- | ---------------- |
| filterBlack | Boolean  | 否       | 是否过滤掉黑名单 |

## 返回结果

| 参数名称 | 参数类型        | 描述     |
| -------- | --------------- | -------- |
