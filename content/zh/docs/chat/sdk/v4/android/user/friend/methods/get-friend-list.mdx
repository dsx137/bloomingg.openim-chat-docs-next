`GetFriendList` 来自 `open_im_sdk/relation.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                 |
| -------- | -------------------------------------------------------------------- |
| 方法     | `GetFriendList(callback Base, operationID string, filterBlack bool)` |
| 所属能力 | 好友                                                                 |
| 源码位置 | `open_im_sdk/relation.go`                                            |

## 作用

获取当前用户全部好友列表，适合好友通讯录首屏加载或登录后全量同步。

## 参数说明

`filterBlack` 控制是否排除已拉黑用户。

## 返回与事件

成功返回好友列表 JSON。

## 使用边界

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。
