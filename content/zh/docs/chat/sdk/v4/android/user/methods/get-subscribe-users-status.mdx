`GetSubscribeUsersStatus` 来自 `open_im_sdk/user.go、open_im_sdk/online.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| 方法     | `GetSubscribeUsersStatus(callback Base, operationID string)` |
| 所属能力 | 用户                                                         |
| 源码位置 | `open_im_sdk/user.go、open_im_sdk/online.go`                 |

## 作用

获取当前已订阅用户的在线状态快照，适合页面进入时先拉一次状态，再等待事件更新。

## 参数说明

不需要用户列表，读取当前订阅集合。

## 返回与事件

成功返回在线状态 JSON。

## 使用边界

用户资料和在线状态方法通常要求当前用户已登录；在线状态变化通过 `SetUserListener` 设置的监听器接收。
