`GetSelfUserInfo` 来自 `open_im_sdk/user.go、open_im_sdk/online.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                 |
| -------- | ---------------------------------------------------- |
| 方法     | `GetSelfUserInfo(callback Base, operationID string)` |
| 所属能力 | 用户                                                 |
| 源码位置 | `open_im_sdk/user.go、open_im_sdk/online.go`         |

## 作用

获取当前登录用户自己的资料，适合进入个人资料页、启动后刷新当前用户信息或账号切换后校验本地缓存。

## 参数说明

`operationID` 用于追踪；不需要传用户 ID，SDK 使用当前登录用户。

## 返回与事件

成功返回当前用户资料 JSON。

## 使用边界

用户资料和在线状态方法通常要求当前用户已登录；在线状态变化通过 `SetUserListener` 设置的监听器接收。
