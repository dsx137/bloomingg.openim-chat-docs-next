订阅一组用户的在线状态，适合好友列表、会话页和在线状态角标需要持续更新的场景。

## 功能介绍

订阅一组用户的在线状态，适合好友列表、会话页和在线状态角标需要持续更新的场景。

用户资料和在线状态方法通常要求当前用户已登录；在线状态变化通过 `SetUserListener` 设置的监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                      |
| -------- | ------------------------------------------------------------------------- |
| 方法签名 | `SubscribeUsersStatus(callback Base, operationID string, userIDs string)` |
| 所属能力 | 用户                                                                      |
| 源码位置 | `open_im_sdk/online.go`                                                   |

## Android 调用参考

当前 Android SDK 的方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void subscribeUsersOnlineStatus(OnBase<List<UsersOnlineStatus>> callBack, List<String> uid)
```

## 输入参数

| 参数名称 | 参数类型                        | 是否必填 | 描述    |
| -------- | ------------------------------- | -------- | ------- |
| callback | OnBase<List<UsersOnlineStatus>> | 是       | 用户 ID |
| ids      | List<String>                    | 是       | 用户 ID |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。
