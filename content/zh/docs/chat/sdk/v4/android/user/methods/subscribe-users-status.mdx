`SubscribeUsersStatus` 来自 `open_im_sdk/user.go、open_im_sdk/online.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                      |
| -------- | ------------------------------------------------------------------------- |
| 方法     | `SubscribeUsersStatus(callback Base, operationID string, userIDs string)` |
| 所属能力 | 用户                                                                      |
| 源码位置 | `open_im_sdk/user.go、open_im_sdk/online.go`                              |

## 作用

订阅一组用户的在线状态，适合好友列表、会话页和在线状态角标需要持续更新的场景。

## 参数说明

`userIDs` 是用户 ID 列表 JSON。

## 返回与事件

成功返回订阅结果，后续状态变化通过用户监听回调。

## 使用边界

用户资料和在线状态方法通常要求当前用户已登录；在线状态变化通过 `SetUserListener` 设置的监听器接收。
