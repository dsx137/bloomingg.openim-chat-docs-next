好友方法来自 `open_im_sdk/relation.go` 中的好友关系部分。它们负责好友资料读取、好友列表分页、好友搜索、申请处理和好友关系更新。

本页是好友关系方法目录。好友资料、好友列表、好友申请和关系维护都拆成了独立方法页。

## 方法索引

| 方法                                                                                                                            | 说明                                                                             |
| ------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------- |
| [`GetSpecifiedFriendsInfo`](/docs/chat/sdk/v4/android/user/friend/methods/get-specified-friends-info)                           | 批量获取指定好友的关系资料，适合好友详情页、会话资料补齐或局部刷新好友信息。     |
| [`GetFriendList`](/docs/chat/sdk/v4/android/user/friend/methods/get-friend-list)                                                | 获取当前用户全部好友列表，适合好友通讯录首屏加载或登录后全量同步。               |
| [`GetFriendListPage`](/docs/chat/sdk/v4/android/user/friend/methods/get-friend-list-page)                                       | 分页获取好友列表，适合好友数量较多时按页加载。                                   |
| [`SearchFriends`](/docs/chat/sdk/v4/android/user/friend/methods/search-friends)                                                 | 搜索好友，适合按用户 ID、昵称、备注等当前搜索结构支持的字段查找好友。            |
| [`CheckFriend`](/docs/chat/sdk/v4/android/user/friend/methods/check-friend)                                                     | 检查一批用户与当前用户是否为好友关系，适合展示添加好友按钮或发起关系操作前校验。 |
| [`AddFriend`](/docs/chat/sdk/v4/android/user/friend/methods/add-friend)                                                         | 向指定用户发送好友申请。是否自动通过取决于服务端策略和对方设置。                 |
| [`UpdateFriends`](/docs/chat/sdk/v4/android/user/friend/methods/update-friends)                                                 | 更新好友关系资料，例如备注、扩展字段或其他好友关系字段。                         |
| [`DeleteFriend`](/docs/chat/sdk/v4/android/user/friend/methods/delete-friend)                                                   | 删除指定好友关系，适合用户主动解除好友。                                         |
| [`GetFriendApplicationListAsRecipient`](/docs/chat/sdk/v4/android/user/friend/methods/get-friend-application-list-as-recipient) | 获取当前用户收到的好友申请列表，适合好友申请入口和审核页。                       |
| [`GetFriendApplicationListAsApplicant`](/docs/chat/sdk/v4/android/user/friend/methods/get-friend-application-list-as-applicant) | 获取当前用户发出的好友申请列表，适合查看申请进度或历史申请。                     |
| [`AcceptFriendApplication`](/docs/chat/sdk/v4/android/user/friend/methods/accept-friend-application)                            | 同意指定好友申请，建立好友关系。                                                 |
| [`RefuseFriendApplication`](/docs/chat/sdk/v4/android/user/friend/methods/refuse-friend-application)                            | 拒绝指定好友申请。                                                               |
| [`GetFriendApplicationUnhandledCount`](/docs/chat/sdk/v4/android/user/friend/methods/get-friend-application-unhandled-count)    | 获取未处理好友申请数量，适合好友入口角标和后台同步。                             |

## 事件联动

好友申请、新增好友、删除好友和好友资料变化通过 `SetFriendListener` 注册的 `OnFriendshipListener` 接收。
