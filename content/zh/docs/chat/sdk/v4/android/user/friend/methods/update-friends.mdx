`UpdateFriends` 来自 `open_im_sdk/relation.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                           |
| -------- | -------------------------------------------------------------- |
| 方法     | `UpdateFriends(callback Base, operationID string, req string)` |
| 所属能力 | 好友                                                           |
| 源码位置 | `open_im_sdk/relation.go`                                      |

## 作用

更新好友关系资料，例如备注、扩展字段或其他好友关系字段。

## 参数说明

`req` 是好友更新请求 JSON；字段以当前关系结构为准。

## 返回与事件

成功返回更新结果，并触发好友资料变化事件。

## 使用边界

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。
