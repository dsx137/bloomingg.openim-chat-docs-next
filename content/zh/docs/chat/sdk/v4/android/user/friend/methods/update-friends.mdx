## 功能介绍

更新好友关系资料，例如备注、扩展字段或其他好友关系字段。

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。

## Core Binding 方法

| 项目     | 说明                                                           |
| -------- | -------------------------------------------------------------- |
| 方法签名 | `UpdateFriends(callback Base, operationID string, req string)` |
| 所属能力 | 好友                                                           |
| 源码位置 | `open_im_sdk/relation.go`                                      |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void updateFriendsReq(OnBase<String> base, UpdateFriendsReq updateFriendsReq)
```

## 输入参数

| 参数名称         | 参数类型         | 是否必填 | 描述         |
| ---------------- | ---------------- | -------- | ------------ |
| updateFriendsReq | UpdateFriendsReq | 是       | 变更的结构体 |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。
