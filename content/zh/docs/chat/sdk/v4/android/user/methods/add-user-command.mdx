## 功能介绍

新增一条用户命令数据，当前源码注释指向用户收藏类数据，可用于按类型保存用户侧轻量命令或标记。

用户资料和在线状态方法通常要求当前用户已登录；在线状态变化通过 `SetUserListener` 设置的监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                       |
| -------- | ------------------------------------------------------------------------------------------ |
| 方法签名 | `AddUserCommand(callback Base, operationID string, Type int32, uuid string, value string)` |
| 所属能力 | 用户                                                                                       |
| 源码位置 | `open_im_sdk/user.go`                                                                      |

## Android 调用参考

当前 Android SDK v3.8.3.5 未提供该 core 方法对应的公开 Java 调用方法。Core binding 方法签名仅用于说明底层 `open_im_sdk` 能力。

## 输入参数

| 参数名称      | 类型     | 描述                       |
| ------------- | -------- | -------------------------- |
| `callback`    | `Base`   | 异步结果回调。             |
| `operationID` | `string` | 本次操作的追踪 ID。        |
| `Type`        | `int32`  | 按 core binding 签名传入。 |
| `uuid`        | `string` | 按 core binding 签名传入。 |
| `value`       | `string` | 按 core binding 签名传入。 |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。
