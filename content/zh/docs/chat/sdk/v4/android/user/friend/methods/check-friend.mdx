`CheckFriend` 来自 `open_im_sdk/relation.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                |
| -------- | ------------------------------------------------------------------- |
| 方法     | `CheckFriend(callback Base, operationID string, userIDList string)` |
| 所属能力 | 好友                                                                |
| 源码位置 | `open_im_sdk/relation.go`                                           |

## 作用

检查一批用户与当前用户是否为好友关系，适合展示添加好友按钮或发起关系操作前校验。

## 参数说明

`userIDList` 是待检查的用户 ID 列表 JSON。

## 返回与事件

成功返回好友关系检查结果 JSON。

## 使用边界

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。
