检查一批用户与当前用户是否为好友关系，适合展示添加好友按钮或发起关系操作前校验。

## 功能介绍

检查一批用户与当前用户是否为好友关系，适合展示添加好友按钮或发起关系操作前校验。

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。

## Core Binding 方法

| 项目     | 说明                                                                |
| -------- | ------------------------------------------------------------------- |
| 方法签名 | `CheckFriend(callback Base, operationID string, userIDList string)` |
| 所属能力 | 好友                                                                |
| 源码位置 | `open_im_sdk/relation.go`                                           |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void checkFriend(OnBase<List<FriendshipInfo>> base, List<String> uidList)
```

## 输入参数

| 参数名称 | 参数类型                                                                                                                                     | 是否必填 | 描述         |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------- | -------- | ------------ |
| callBack | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)&lt;List&lt;[`FriendshipInfo`](/docs/chat/sdk/v4/android/models/friendship-info)&gt;&gt; | 是       | 回调接口     |
| uidList  | String                                                                                                                                       | 是       | 用户 ID 集合 |

#### FriendInfo

| 字段名称 | 字段类型 | 描述                         |
| -------- | -------- | ---------------------------- |
| userID   | String   | 用户 UserID                  |
| result   | int      | 1 表示好友（并且不是黑名单） |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().friendshipManager.checkFriend(new OnBase<List<FriendshipInfo>>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(List<FriendshipInfo> data) {
        // TODO: 处理好友关系检查结果
    }
}, uidList);
```
