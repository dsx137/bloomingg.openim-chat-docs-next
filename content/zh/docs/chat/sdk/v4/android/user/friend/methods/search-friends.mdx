搜索好友，适合按用户 ID、昵称、备注等当前搜索结构支持的字段查找好友。

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。

## Core Binding 方法

| 项目     | 说明                                                                   |
| -------- | ---------------------------------------------------------------------- |
| 方法签名 | `SearchFriends(callback Base, operationID string, searchParam string)` |
| 所属能力 | 好友                                                                   |
| 源码位置 | `open_im_sdk/relation.go`                                              |

## Android 调用参考

Android SDK v3.8.3 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void searchFriends(OnBase<List<FriendInfo>> base, List<String> keywordList, boolean isSearchUserID, boolean isSearchNickname, boolean isSearchRemark)
```

## 输入参数

| 参数名称         | 参数类型                                                                                                                     | 是否必填 | 描述                                           |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------- | -------- | ---------------------------------------------- |
| callBack         | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)`<List<`[`FriendInfo`](/docs/chat/sdk/v4/android/models/friend-info)`>>` | 是       | 回调接口                                       |
| keywordList      | `List<String>`                                                                                                               | 是       | 搜索关键词，目前仅支持一个关键词搜索，不能为空 |
| isSearchUserID   | Boolean                                                                                                                      | 是       | 是否以关键词搜索 UserID                        |
| isSearchNickname | Boolean                                                                                                                      | 是       | 是否以关键词搜索昵称，默认 false               |
| isSearchRemark   | Boolean                                                                                                                      | 是       | 是否以关键词搜索备注，默认 false               |

## 返回结果

`onSuccess` 返回匹配启用字段的 `List<`[`FriendInfo`](/docs/chat/sdk/v4/android/models/friend-info)`>`；没有匹配项时返回空列表，关键词或本地查询失败时调用 `onError`。

## 代码示例

以下示例展示 Android SDK v3.8.3 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().friendshipManager.searchFriends(
    new OnBase<List<FriendInfo>>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(List<FriendInfo> data) {
        // TODO: 处理好友搜索结果
    }
},
    keywordList,
    isSearchUserID,
    isSearchNickname,
    isSearchRemark
);
```

## 相关类型

- [底层通用回调](/docs/chat/sdk/v4/android/models/base)（`Base`）：core binding 异步结果回调，Android 调用层通常映射为 OnBase。
- [通用回调](/docs/chat/sdk/v4/android/models/on-base)（`OnBase`）：Android SDK 异步方法的通用成功和失败回调。
- [好友关系监听器](/docs/chat/sdk/v4/android/models/on-friendship-listener)（`OnFriendshipListener`）：好友、好友申请和黑名单变化监听器。
- [好友信息](/docs/chat/sdk/v4/android/models/friend-info)（`FriendInfo`）：好友资料，包含昵称、头像、备注和添加来源等信息。
