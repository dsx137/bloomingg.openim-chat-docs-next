`SearchFriends` 来自 `open_im_sdk/relation.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                   |
| -------- | ---------------------------------------------------------------------- |
| 方法     | `SearchFriends(callback Base, operationID string, searchParam string)` |
| 所属能力 | 好友                                                                   |
| 源码位置 | `open_im_sdk/relation.go`                                              |

## 作用

搜索好友，适合按用户 ID、昵称、备注等当前搜索结构支持的字段查找好友。

## 参数说明

`searchParam` 是好友搜索条件 JSON。

## 返回与事件

成功返回好友搜索结果 JSON。

## 使用边界

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。
