同意指定好友申请，建立好友关系。

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。

## Core Binding 方法

| 项目     | 说明                                                                                 |
| -------- | ------------------------------------------------------------------------------------ |
| 方法签名 | `AcceptFriendApplication(callback Base, operationID string, userIDHandleMsg string)` |
| 所属能力 | 好友                                                                                 |
| 源码位置 | `open_im_sdk/relation.go`                                                            |

## Android 调用参考

Android SDK v3.8.3 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void acceptFriendApplication(OnBase<String> base, String uid, String handleMsg)
```

## 输入参数

| 参数名称  | Java 类型                                                      | 是否必填 | 描述                       |
| --------- | -------------------------------------------------------------- | -------- | -------------------------- |
| base      | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)`<String>` | 是       | 好友申请处理结果回调。     |
| uid       | `String`                                                       | 是       | 发起好友申请的用户 ID。    |
| handleMsg | `String`                                                       | 否       | 同意申请时附带的处理说明。 |

## 返回结果

`onSuccess` 表示好友申请已同意并完成本地好友与申请记录同步，回调字符串不含业务数据；申请不存在、已处理或服务端拒绝时调用 `onError`。

## 代码示例

以下示例展示 Android SDK v3.8.3 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().friendshipManager.acceptFriendApplication(new OnBase<String>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(String data) {
        // TODO: 处理成功结果
    }
}, uid, handleMsg);
```
