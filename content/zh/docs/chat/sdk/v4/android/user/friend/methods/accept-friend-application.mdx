`AcceptFriendApplication` 来自 `open_im_sdk/relation.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                 |
| -------- | ------------------------------------------------------------------------------------ |
| 方法     | `AcceptFriendApplication(callback Base, operationID string, userIDHandleMsg string)` |
| 所属能力 | 好友                                                                                 |
| 源码位置 | `open_im_sdk/relation.go`                                                            |

## 作用

同意指定好友申请，建立好友关系。

## 参数说明

`userIDHandleMsg` 是好友申请处理请求 JSON，包含申请人和处理说明等字段。

## 返回与事件

成功返回处理结果，并触发好友新增和申请接受事件。

## 使用边界

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。
