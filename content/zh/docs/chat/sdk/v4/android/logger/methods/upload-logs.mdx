上传 SDK 日志，适合用户反馈、故障排查或线上问题定位。

日志上传能力至少要求 SDK 已初始化；上传进度通过 `UploadLogProgress` 返回。

## Core Binding 方法

| 项目     | 说明                                                                                             |
| -------- | ------------------------------------------------------------------------------------------------ |
| 方法签名 | `UploadLogs(callback Base, operationID string, line int, ex string, progress UploadLogProgress)` |
| 所属能力 | 日志                                                                                             |
| 源码位置 | `open_im_sdk/third.go`                                                                           |

## Android 调用参考

Android SDK v3.8.3 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void uploadLogs(OnBase<String> base, List<String> params, int line, String ex, UploadLogProgress progress)
```

## 输入参数

| 参数名称 | 参数类型                                                                    | 是否必填 | 描述                                                |
| -------- | --------------------------------------------------------------------------- | -------- | --------------------------------------------------- |
| base     | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)`<String>`              | 是       | 上传结果回调。                                      |
| params   | `List<String>`                                                              | 否       | 预留参数，当前 Android SDK 方法签名保留该参数。     |
| line     | int                                                                         | 是       | 上传日志的起始行或限制行数，由业务按 SDK 约定传入。 |
| ex       | String                                                                      | 否       | 扩展信息。                                          |
| progress | [`UploadLogProgress`](/docs/chat/sdk/v4/android/models/upload-log-progress) | 否       | 日志上传进度回调。                                  |

## 返回结果

`onSuccess` 表示日志已打包上传并在服务端登记，回调字符串不含文件 URL 等业务数据；`UploadLogProgress` 报告压缩包已上传字节数和总字节数。日志目录不可读、已有上传任务或网络请求失败时调用 `onError`。

## 代码示例

以下示例展示 Android SDK v3.8.3 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().uploadLogs(
    new OnBase<String>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(String data) {
        // TODO: 处理日志上传结果
    }
},
    params,
    line,
    ex,
    new UploadLogProgress() {
        @Override
        public void onProgress(long current, long size) {
            // TODO: 更新日志上传进度
        }
    }
);
```
