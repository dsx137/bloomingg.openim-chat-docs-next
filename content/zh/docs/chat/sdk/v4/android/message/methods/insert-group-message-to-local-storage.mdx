`InsertGroupMessageToLocalStorage` 来自 `open_im_sdk/conversation_msg.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                                                 |
| -------- | -------------------------------------------------------------------------------------------------------------------- |
| 方法     | `InsertGroupMessageToLocalStorage(callback Base, operationID string, message string, groupID string, sendID string)` |
| 所属能力 | 会话与消息                                                                                                           |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                                                    |

## 作用

向本地群聊会话插入一条消息，适合导入或补写群聊本地消息。

## 参数说明

`message` 是消息 JSON；`groupID` 是群 ID；`sendID` 是发送用户。

## 返回与事件

成功返回插入后的消息 JSON。

## 使用边界

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。
