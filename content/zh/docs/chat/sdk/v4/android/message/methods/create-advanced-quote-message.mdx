创建带实体信息的引用消息。适合引用回复中仍需要链接、富文本实体或业务标记。

## 功能介绍

创建带实体信息的引用消息。适合引用回复中仍需要链接、富文本实体或业务标记。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                                    |
| -------- | ------------------------------------------------------------------------------------------------------- |
| 方法签名 | `CreateAdvancedQuoteMessage(operationID string, text string, message, messageEntityList string) string` |
| 所属能力 | 会话与消息                                                                                              |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                                       |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public Message createAdvancedQuoteMessage(String text, Message message, List<RichMessage> richList)
```

## 输入参数

| 参数名称            | 类型     | 描述                                                        |
| ------------------- | -------- | ----------------------------------------------------------- |
| `operationID`       | `string` | 本次操作的追踪 ID。                                         |
| `text`              | `string` | 按 core binding 签名传入。                                  |
| `message`           | `string` | 通常为 JSON 字符串或上层封装对象，具体结构以 SDK 定义为准。 |
| `messageEntityList` | `string` | 通常为 JSON 字符串或上层封装对象，具体结构以 SDK 定义为准。 |

## 返回结果

同步返回 `string`。

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
Message advancedQuoteMessage = OpenIMClient.getInstance().messageManager.createAdvancedQuoteMessage(text, message, richList);
```
