`GetConversationIDBySessionType` 来自 `open_im_sdk/conversation_msg.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                          |
| -------- | --------------------------------------------------------------------------------------------- |
| 方法     | `GetConversationIDBySessionType(operationID string, sourceID string, sessionType int) string` |
| 所属能力 | 会话与消息                                                                                    |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                             |

## 作用

根据目标 ID 和会话类型生成 OpenIM 会话 ID，适合从用户或群跳转到会话前构造会话标识。

## 参数说明

`sourceID` 是目标用户或群 ID；`sessionType` 是会话类型。

## 返回与事件

同步返回会话 ID 字符串。

## 使用边界

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。
