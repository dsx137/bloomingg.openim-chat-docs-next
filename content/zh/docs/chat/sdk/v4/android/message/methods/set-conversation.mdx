`SetConversation` 来自 `open_im_sdk/conversation_msg.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                    |
| -------- | --------------------------------------------------------------------------------------- |
| 方法     | `SetConversation(callback Base, operationID string, conversationID string, req string)` |
| 所属能力 | 会话与消息                                                                              |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                       |

## 作用

设置会话属性，例如置顶、免打扰、扩展字段等当前结构支持的字段。

## 参数说明

`conversationID` 定位会话；`req` 是会话设置请求 JSON。

## 返回与事件

成功返回更新结果，并可能触发会话变更事件。

## 使用边界

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。
