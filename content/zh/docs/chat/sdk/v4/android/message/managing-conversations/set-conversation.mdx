设置会话属性，例如置顶、免打扰、扩展字段等当前结构支持的字段。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                    |
| -------- | --------------------------------------------------------------------------------------- |
| 方法签名 | `SetConversation(callback Base, operationID string, conversationID string, req string)` |
| 所属能力 | 会话与消息                                                                              |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                       |

## Android 调用参考

Android SDK v3.8.3 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void setConversation(OnBase<String> base, String conversationID, ConversationReq conversationReq)
```

## 输入参数

| 参数名称        | Java 类型                                                              | 是否必填 | 描述                     |
| --------------- | ---------------------------------------------------------------------- | -------- | ------------------------ |
| base            | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)`<String>`         | 是       | 会话设置更新结果回调。   |
| conversationID  | `String`                                                               | 是       | 要更新的会话 ID。        |
| conversationReq | [`ConversationReq`](/docs/chat/sdk/v4/android/models/conversation-req) | 是       | 需要修改的会话设置字段。 |

## 返回结果

`onSuccess` 表示 `ConversationReq` 中提交的会话属性已更新并完成本地同步，回调字符串不含业务数据；会话不存在、字段值无效或服务端拒绝时调用 `onError`。

## 代码示例

以下示例展示 Android SDK v3.8.3 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().conversationManager.setConversation(new OnBase<String>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(String data) {
        // TODO: 处理成功结果
    }
}, conversationID, conversationReq);
```

## 相关类型

- [底层通用回调](/docs/chat/sdk/v4/android/models/base)（`Base`）：core binding 异步结果回调，Android 调用层通常映射为 OnBase。
- [通用回调](/docs/chat/sdk/v4/android/models/on-base)（`OnBase`）：Android SDK 异步方法的通用成功和失败回调。
- [会话设置请求](/docs/chat/sdk/v4/android/models/conversation-req)（`ConversationReq`）：会话设置请求对象，用于设置会话置顶、免打扰、扩展字段等属性。
