搜索本地会话，适合聊天列表搜索入口。

## 功能介绍

搜索本地会话，适合聊天列表搜索入口。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                        |
| -------- | --------------------------------------------------------------------------- |
| 方法签名 | `SearchConversation(callback Base, operationID string, searchParam string)` |
| 所属能力 | 会话与消息                                                                  |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                           |

## Android 调用参考

当前 Android SDK v3.8.3.5 未提供该 core 方法对应的公开 Java 调用方法。Core binding 方法签名仅用于说明底层 `open_im_sdk` 能力。

## 输入参数

| 参数名称      | 类型     | 描述                                                        |
| ------------- | -------- | ----------------------------------------------------------- |
| `callback`    | `Base`   | 异步结果回调。                                              |
| `operationID` | `string` | 本次操作的追踪 ID。                                         |
| `searchParam` | `string` | 通常为 JSON 字符串或上层封装对象，具体结构以 SDK 定义为准。 |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。
