获取指定会话的历史消息，适合聊天页向上翻页或首次加载历史记录。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                         |
| -------- | -------------------------------------------------------------------------------------------- |
| 方法签名 | `GetAdvancedHistoryMessageList(callback Base, operationID string, getMessageOptions string)` |
| 所属能力 | 会话与消息                                                                                   |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                            |

## Android 调用参考

Android SDK v3.8.3 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void getAdvancedHistoryMessageList(OnBase<AdvancedMessage> callBack, String conversationID, Message startMsg, int count, ViewType viewType)
```

## 输入参数

| 参数名称       | 参数类型                                                                                                                         | 是否必填 | 描述                                                                                                           |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------- | -------- | -------------------------------------------------------------------------------------------------------------- |
| callBack       | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)`<`[`AdvancedMessage`](/docs/chat/sdk/v4/android/models/advanced-message)`>` | 是       | 历史消息分页结果回调。                                                                                         |
| conversationID | String                                                                                                                           | 是       | 会话 ID                                                                                                        |
| startMsg       | [`Message`](/docs/chat/sdk/v4/android/models/message)                                                                            | 是       | 从这条消息开始查询[count]条，获取的列表 index==length-1 为最新消息，所以获取下一页历史记录 startMsg=list.first |
| count          | int                                                                                                                              | 是       | 数量                                                                                                           |
| viewType       | [`ViewType`](/docs/chat/sdk/v4/android/models/view-type)                                                                         | 是       | 为History时表示当前在获取历史消息，为Search时表示在执行搜索消息操作                                            |

## 返回结果

`onSuccess` 返回 [`AdvancedMessage`](/docs/chat/sdk/v4/android/models/advanced-message)，其中 `messageList` 是当前页历史消息，`lastMinSeq`、`isEnd` 等字段用于继续分页；会话、游标或本地查询失败时调用 `onError`。

## 代码示例

以下示例展示 Android SDK v3.8.3 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().messageManager.getAdvancedHistoryMessageList(
    new OnBase<AdvancedMessage>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(AdvancedMessage data) {
        // TODO: 处理历史消息
    }
},
    conversationID,
    startMsg,
    count,
    ViewType.History
);
```
