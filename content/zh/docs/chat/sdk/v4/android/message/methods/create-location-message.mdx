`CreateLocationMessage` 来自 `open_im_sdk/conversation_msg.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                                        |
| -------- | ----------------------------------------------------------------------------------------------------------- |
| 方法     | `CreateLocationMessage(operationID string, description string, longitude float64, latitude float64) string` |
| 所属能力 | 会话与消息                                                                                                  |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                                           |

## 作用

创建位置消息。适合发送地图位置、门店位置或业务地点。

## 参数说明

`description` 是位置描述；`longitude` 和 `latitude` 是经纬度。

## 返回与事件

同步返回消息 JSON 字符串。

## 使用边界

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。
