创建位置消息。适合发送地图位置、门店位置或业务地点。

## 功能介绍

创建位置消息。适合发送地图位置、门店位置或业务地点。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                                |
| -------- | --------------------------------------------------------------------------------------------------- |
| 方法签名 | `CreateLocationMessage(operationID string, description string, longitude, latitude float64) string` |
| 所属能力 | 会话与消息                                                                                          |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                                   |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public Message createLocationMessage(double latitude, double longitude, String description)
```

## 输入参数

| 参数名称    | 参数类型 | 是否必填 | 描述 |
| ----------- | -------- | -------- | ---- |
| description | String   | 是       | 描述 |
| latitude    | double   | 是       | 纬度 |
| longitude   | double   | 是       | 经度 |

## 返回结果

| 名称 | 类型                                                  | 描述     |
| ---- | ----------------------------------------------------- | -------- |
| ~    | [`Message`](/docs/chat/sdk/v4/android/models/message) | 成功返回 |

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
Message locationMessage = OpenIMClient.getInstance().messageManager.createLocationMessage(latitude, longitude, description);
```
