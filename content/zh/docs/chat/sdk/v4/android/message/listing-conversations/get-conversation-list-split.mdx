分页获取会话列表，适合会话数量较多时按页加载。

## 功能介绍

分页获取会话列表，适合会话数量较多时按页加载。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                 |
| -------- | ------------------------------------------------------------------------------------ |
| 方法签名 | `GetConversationListSplit(callback Base, operationID string, offset int, count int)` |
| 所属能力 | 会话与消息                                                                           |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                    |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void getConversationListSplit(OnBase<List<ConversationInfo>> base, long offset, long count)
```

## 输入参数

| 参数名称 | 参数类型                                                                                                                                         | 是否必填 | 描述       |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------ | -------- | ---------- |
| base     | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)&lt;List&lt;[`ConversationInfo`](/docs/chat/sdk/v4/android/models/conversation-info)&gt;&gt; | 是       | 回调接口   |
| offset   | int                                                                                                                                              | 是       | 起始偏移量 |
| count    | int                                                                                                                                              | 是       | 数量       |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().conversationManager.getConversationListSplit(
    new OnBase<List<ConversationInfo>>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(List<ConversationInfo> data) {
        // TODO: 处理分页会话列表
    }
},
    offset,
    count
);
```

- [通用回调](/docs/chat/sdk/v4/android/models/on-base)（`OnBase`）：Android SDK 异步方法的通用成功和失败回调。
