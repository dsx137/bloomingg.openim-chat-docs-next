`SendMessageNotOss` 来自 `open_im_sdk/conversation_msg.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                                                                                        |
| -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 方法     | `SendMessageNotOss(callback SendMsgCallBack, operationID string, message string, recvID string, groupID string, offlinePushInfo string, isOnlineOnly bool)` |
| 所属能力 | 会话与消息                                                                                                                                                  |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                                                                                           |

## 作用

发送不走 SDK OSS 上传链路的消息。适合业务自定义上传、资源 URL 已经准备好或使用 `Create*ByURL` 方法创建的消息。

## 参数说明

参数语义与 `SendMessage` 相同，但调用方需要保证消息里的资源信息已经可被接收端访问。

## 返回与事件

成功、进度和失败回调与 `SendMessage` 一致。

## 使用边界

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。
