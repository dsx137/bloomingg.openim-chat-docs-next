发送不走 SDK OSS 上传链路的消息。适合业务自定义上传、资源 URL 已经准备好或使用 `Create*ByURL` 方法创建的消息。

## 功能介绍

发送不走 SDK OSS 上传链路的消息。适合业务自定义上传、资源 URL 已经准备好或使用 `Create*ByURL` 方法创建的消息。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                                                                          |
| -------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| 方法签名 | `SendMessageNotOss(callback SendMsgCallBack, operationID string, message, recvID, groupID string, offlinePushInfo string, isOnlineOnly bool)` |
| 所属能力 | 会话与消息                                                                                                                                    |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                                                                             |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void sendMessageNotOss(OnMsgSendCallback base, Message message, String recvUid,
                              String recvGid, OfflinePushInfo offlinePushInfo)
public void sendMessageNotOss(OnMsgSendCallback base, Message message, String recvUid,
                              String recvGid, OfflinePushInfo offlinePushInfo, boolean isOnlineOnly)
```

## 输入参数

| 参数名称        | 参数类型                                                                     | 是否必填 | 描述                                 |
| --------------- | ---------------------------------------------------------------------------- | -------- | ------------------------------------ |
| base            | [`OnMsgSendCallback`](/docs/chat/sdk/v4/android/models/on-msg-send-callback) | 是       | 发送结果和进度回调。                 |
| message         | [`Message`](/docs/chat/sdk/v4/android/models/message)                        | 是       | 要发送的消息体。                     |
| recvUid         | String                                                                       | 单聊必填 | 单聊接收者用户 ID。                  |
| recvGid         | String                                                                       | 群聊必填 | 群聊接收群 ID。                      |
| offlinePushInfo | [`OfflinePushInfo`](/docs/chat/sdk/v4/android/models/offline-push-info)      | 否       | 离线推送配置。                       |
| isOnlineOnly    | boolean                                                                      | 否       | 是否只发送在线消息，仅重载方法使用。 |

## 返回结果

通过 `OnMsgSendCallback` 返回发送状态：`onProgress` 返回上传/发送进度，`onSuccess` 返回发送成功后的消息，`onError` 返回错误信息。

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().messageManager.sendMessageNotOss(
    new OnMsgSendCallback() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理发送失败
    }

    @Override
    public void onProgress(long progress) {
        // TODO: 更新发送进度
    }

    @Override
    public void onSuccess(Message message) {
        // TODO: 处理发送成功的消息
    }
},
    message,
    recvUid,
    recvGid,
    offlinePushInfo
);
```
