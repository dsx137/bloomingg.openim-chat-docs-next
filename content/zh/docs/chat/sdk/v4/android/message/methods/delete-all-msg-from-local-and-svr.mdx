`DeleteAllMsgFromLocalAndSvr` 来自 `open_im_sdk/conversation_msg.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                             |
| -------- | ---------------------------------------------------------------- |
| 方法     | `DeleteAllMsgFromLocalAndSvr(callback Base, operationID string)` |
| 所属能力 | 会话与消息                                                       |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                |

## 作用

删除当前用户本地和服务端全部消息数据，属于高风险清理操作。

## 参数说明

无业务参数。

## 返回与事件

成功返回清理结果。

## 使用边界

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。
