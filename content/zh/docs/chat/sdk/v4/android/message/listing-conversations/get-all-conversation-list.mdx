获取当前用户的全部本地会话列表，适合聊天列表全量加载或登录后刷新。

## 功能介绍

获取当前用户的全部本地会话列表，适合聊天列表全量加载或登录后刷新。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                        |
| -------- | ----------------------------------------------------------- |
| 方法签名 | `GetAllConversationList(callback Base, operationID string)` |
| 所属能力 | 会话与消息                                                  |
| 源码位置 | `open_im_sdk/conversation_msg.go`                           |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void getAllConversationList(OnBase<List<ConversationInfo>> base)
```

## 输入参数

| 名称     | 类型                                                                                                                                             | 描述 |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------ | ---- |
| callBack | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)&lt;List&lt;[`ConversationInfo`](/docs/chat/sdk/v4/android/models/conversation-info)&gt;&gt; | 是   |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().conversationManager.getAllConversationList(new OnBase<List<ConversationInfo>>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(List<ConversationInfo> data) {
        // TODO: 处理全部会话列表
    }
});
```
