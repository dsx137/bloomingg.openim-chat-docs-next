通过完整视频路径和完整快照路径创建视频消息。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                                                                         |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| 方法签名 | `CreateVideoMessageFromFullPath(operationID string, videoFullPath string, videoType string, duration int64, snapshotFullPath string) string` |
| 所属能力 | 会话与消息                                                                                                                                   |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                                                                            |

## Android 调用参考

Android SDK v3.8.3 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public Message createVideoMessageFromFullPath(String videoPath, String videoType, long duration, String snapshotPath)
```

## 输入参数

| 参数名称     | 参数类型 | 是否必填 | 描述         |
| ------------ | -------- | -------- | ------------ |
| videoPath    | String   | 是       | 绝对路径     |
| videoType    | String   | 是       | 视频类型     |
| duration     | int      | 是       | 时长         |
| snapshotPath | String   | 是       | 视频封面路径 |

## 返回结果

| 名称 | 类型                                                  | 描述     |
| ---- | ----------------------------------------------------- | -------- |
| ~    | [`Message`](/docs/chat/sdk/v4/android/models/message) | 成功返回 |

## 代码示例

以下示例展示 Android SDK v3.8.3 的调用方式。

```java showLineNumbers
Message videoMessage = OpenIMClient.getInstance().messageManager.createVideoMessageFromFullPath(videoPath, videoType, duration, snapshotPath);
```

## 相关类型

- [消息对象](/docs/chat/sdk/v4/android/models/message)（`Message`）：消息对象，包含消息 ID、发送者、会话、内容元素和状态等信息。
