`DeleteMessageFromLocalStorage` 来自 `open_im_sdk/conversation_msg.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                                          |
| -------- | ------------------------------------------------------------------------------------------------------------- |
| 方法     | `DeleteMessageFromLocalStorage(callback Base, operationID string, conversationID string, clientMsgID string)` |
| 所属能力 | 会话与消息                                                                                                    |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                                             |

## 作用

仅从本地存储删除指定消息，不表示服务端消息被删除。

## 参数说明

`conversationID` 和 `clientMsgID` 定位消息。

## 返回与事件

成功返回本地删除结果。

## 使用边界

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。
