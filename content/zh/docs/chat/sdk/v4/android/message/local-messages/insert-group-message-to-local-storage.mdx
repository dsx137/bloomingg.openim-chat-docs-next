向本地群聊会话插入一条消息，适合导入或补写群聊本地消息。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                                                 |
| -------- | -------------------------------------------------------------------------------------------------------------------- |
| 方法签名 | `InsertGroupMessageToLocalStorage(callback Base, operationID string, message string, groupID string, sendID string)` |
| 所属能力 | 会话与消息                                                                                                           |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                                                    |

## Android 调用参考

Android SDK v3.8.3 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void insertGroupMessageToLocalStorage(OnBase<String> base, Message message, String groupID, String senderID)
```

## 输入参数

| 参数名称 | 参数类型                                                       | 是否必填 | 描述               |
| -------- | -------------------------------------------------------------- | -------- | ------------------ |
| callBack | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)`<String>` | 是       | 回调接口           |
| message  | [`Message`](/docs/chat/sdk/v4/android/models/message)          | 是       | 消息体             |
| groupID  | String                                                         | 是       | 接收消息的 groupID |
| senderID | String                                                         | 是       | 发送消息的 userID  |

## 返回结果

`onSuccess` 返回插入后的 [`Message`](/docs/chat/sdk/v4/android/models/message) JSON 字符串；core 会重新生成 `clientMsgID`、发送时间和会话字段，并把状态设为发送成功。参数或本地写入失败时调用 `onError`。

## 代码示例

以下示例展示 Android SDK v3.8.3 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().messageManager.insertGroupMessageToLocalStorage(new OnBase<String>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(String data) {
        // TODO: 处理成功结果
    }
}, message, groupID, senderID);
```

## 相关类型

- [底层通用回调](/docs/chat/sdk/v4/android/models/base)（`Base`）：core binding 异步结果回调，Android 调用层通常映射为 OnBase。
- [通用回调](/docs/chat/sdk/v4/android/models/on-base)（`OnBase`）：Android SDK 异步方法的通用成功和失败回调。
- [消息对象](/docs/chat/sdk/v4/android/models/message)（`Message`）：消息对象，包含消息 ID、发送者、会话、内容元素和状态等信息。
