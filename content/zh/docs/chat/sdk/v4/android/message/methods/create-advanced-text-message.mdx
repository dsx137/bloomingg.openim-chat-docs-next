## 功能介绍

创建带实体信息的文本消息。适合文本中需要携带链接、富文本实体或业务范围标记的场景。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                   |
| -------- | -------------------------------------------------------------------------------------- |
| 方法签名 | `CreateAdvancedTextMessage(operationID string, text, messageEntityList string) string` |
| 所属能力 | 会话与消息                                                                             |
