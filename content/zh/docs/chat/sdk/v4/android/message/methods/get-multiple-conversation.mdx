按会话 ID 批量获取多个会话，适合局部刷新或从推送跳转后补齐会话。

## 功能介绍

按会话 ID 批量获取多个会话，适合局部刷新或从推送跳转后补齐会话。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                    |
| -------- | --------------------------------------------------------------------------------------- |
| 方法签名 | `GetMultipleConversation(callback Base, operationID string, conversationIDList string)` |
| 所属能力 | 会话与消息                                                                              |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                       |

## Android 调用参考

当前 Android SDK 的方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void getMultipleConversation(OnBase<List<ConversationInfo>> base, List<String> conversationIDs)
```

## 输入参数

| 参数名称        | 参数类型                       | 是否必填 | 描述         |
| --------------- | ------------------------------ | -------- | ------------ |
| base            | OnBase<List<ConversationInfo>> | 是       | 回调接口     |
| conversationIDs | List<String>                   | 是       | 会话 ID 列表 |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。

## 代码示例

以下示例展示当前 Android SDK 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().conversationManager.getMultipleConversation(new OnBase<List<ConversationInfo>>() {
            @Override
            public void onError(int code, String error) {

            }

            @Override
            public void onSuccess(List<ConversationInfo> data) {

            }
        },conversationIDs);
```
