`GetMultipleConversation` 来自 `open_im_sdk/conversation_msg.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                    |
| -------- | --------------------------------------------------------------------------------------- |
| 方法     | `GetMultipleConversation(callback Base, operationID string, conversationIDList string)` |
| 所属能力 | 会话与消息                                                                              |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                       |

## 作用

按会话 ID 批量获取多个会话，适合局部刷新或从推送跳转后补齐会话。

## 参数说明

`conversationIDList` 是会话 ID 列表 JSON。

## 返回与事件

成功返回会话列表 JSON。

## 使用边界

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。
