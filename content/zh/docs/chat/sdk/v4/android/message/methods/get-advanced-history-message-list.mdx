`GetAdvancedHistoryMessageList` 来自 `open_im_sdk/conversation_msg.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                         |
| -------- | -------------------------------------------------------------------------------------------- |
| 方法     | `GetAdvancedHistoryMessageList(callback Base, operationID string, getMessageOptions string)` |
| 所属能力 | 会话与消息                                                                                   |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                            |

## 作用

获取指定会话的历史消息，适合聊天页向上翻页或首次加载历史记录。

## 参数说明

`getMessageOptions` 是历史消息分页请求 JSON。

## 返回与事件

成功返回消息列表和是否结束等信息 JSON。

## 使用边界

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。
