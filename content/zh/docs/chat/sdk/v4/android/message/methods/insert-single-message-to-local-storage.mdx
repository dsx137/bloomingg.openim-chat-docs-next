向本地单聊会话插入一条消息，适合导入历史、补写系统消息或本地展示消息。

## 功能介绍

向本地单聊会话插入一条消息，适合导入历史、补写系统消息或本地展示消息。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                                                 |
| -------- | -------------------------------------------------------------------------------------------------------------------- |
| 方法签名 | `InsertSingleMessageToLocalStorage(callback Base, operationID string, message string, recvID string, sendID string)` |
| 所属能力 | 会话与消息                                                                                                           |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                                                    |

## Android 调用参考

当前 Android SDK 的方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void insertSingleMessageToLocalStorage(OnBase<String> base, Message message, String receiver, String sender)
```

## 输入参数

| 参数名称   | 参数类型       | 是否必填 | 描述              |
| ---------- | -------------- | -------- | ----------------- |
| base       | OnBase<String> | 是       | 回调接口          |
| message    | Message        | 是       | 消息体            |
| receiverID | String         | 是       | 接收消息的 userID |
| senderID   | String         | 是       | 发送消息的 userID |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。
