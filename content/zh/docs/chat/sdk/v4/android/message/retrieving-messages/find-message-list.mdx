按多个会话和消息 ID 条件查找消息，适合定位指定消息集合。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                            |
| -------- | ------------------------------------------------------------------------------- |
| 方法签名 | `FindMessageList(callback Base, operationID string, findMessageOptions string)` |
| 所属能力 | 会话与消息                                                                      |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                               |

## Android 调用参考

Android SDK v3.8.3 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void findMessageList(OnBase<SearchResult> base, List<SearchParams> searchParams)
```

## 输入参数

| 参数名称     | Java 类型                                                                                                                  | 是否必填 | 描述                               |
| ------------ | -------------------------------------------------------------------------------------------------------------------------- | -------- | ---------------------------------- |
| base         | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)`<`[`SearchResult`](/docs/chat/sdk/v4/android/models/search-result)`>` | 是       | 消息查找结果回调。                 |
| searchParams | `List<`[`SearchParams`](/docs/chat/sdk/v4/android/models/search-params)`>`                                                 | 是       | 按会话及客户端消息 ID 组织的条件。 |

## 返回结果

结果通过 `base` 返回 [`SearchResult`](/docs/chat/sdk/v4/android/models/search-result)，其中按会话组织命中的消息列表。

## 代码示例

以下示例展示 Android SDK v3.8.3 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().messageManager.findMessageList(new OnBase<SearchResult>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(SearchResult data) {
        // TODO: 处理查询结果
    }
}, searchParams);
```

## 相关类型

- [底层通用回调](/docs/chat/sdk/v4/android/models/base)（`Base`）：core binding 异步结果回调，Android 调用层通常映射为 OnBase。
- [通用回调](/docs/chat/sdk/v4/android/models/on-base)（`OnBase`）：Android SDK 异步方法的通用成功和失败回调。
- [搜索参数](/docs/chat/sdk/v4/android/models/search-params)（`SearchParams`）：搜索会话或好友时使用的搜索条件。
- [消息搜索结果](/docs/chat/sdk/v4/android/models/search-result)（`SearchResult`）：本地消息搜索结果。
