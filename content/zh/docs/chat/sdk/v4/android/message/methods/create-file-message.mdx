`CreateFileMessage` 来自 `open_im_sdk/conversation_msg.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                             |
| -------- | -------------------------------------------------------------------------------- |
| 方法     | `CreateFileMessage(operationID string, filePath string, fileName string) string` |
| 所属能力 | 会话与消息                                                                       |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                |

## 作用

通过文件路径创建普通文件消息。

## 参数说明

`filePath` 是文件路径；`fileName` 是展示文件名。

## 返回与事件

同步返回消息 JSON 字符串。

## 使用边界

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。
