发送由 Create 方法创建的消息，并使用 SDK 内置上传和发送链路。单聊与群聊只能选择一个目标字段。

## 功能介绍

发送由 Create 方法创建的消息，并使用 SDK 内置上传和发送链路。单聊与群聊只能选择一个目标字段。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                                                      |
| -------- | ------------------------------------------------------------------------------------------------------------------------- |
| 方法签名 | `SendMessage(callback SendMsgCallBack, operationID, message, recvID, groupID, offlinePushInfo string, isOnlineOnly bool)` |
| 所属能力 | 会话与消息                                                                                                                |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                                                         |

## Android 调用参考

当前 Android SDK 的方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void sendMessage(OnMsgSendCallback base, Message message, String recvUid, String recvGid, OfflinePushInfo offlinePushInfo)
```

## 输入参数

| 参数名称        | 参数类型          | 是否必填 | 描述                |
| --------------- | ----------------- | -------- | ------------------- |
| base            | OnMsgSendCallback | 是       | callback            |
| message         | Message           | 是       | 消息体              |
| userID          | String            | 否       | 接收消息的 user ID  |
| groupID         | String            | 否       | 接收消息的 group ID |
| offlinePushInfo | OfflinePushInfo   | 否       | 离线消息体          |

## 返回结果

| 名称 | 类型    | 描述     |
| ---- | ------- | -------- |
| ~    | Message | 成功返回 |

## 代码示例

以下示例展示当前 Android SDK 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().messageManager.sendMessage( new OnMsgSendCallback(){

              @Override
              public void onError(int code, String error) {
                  //发送失败
              }

              @Override
              public void onProgress(long progress) {
                    //发送进度
              }

              @Override
              public void onSuccess(Message s) {
                     //发送成功
              }
          },  message,  recvUid,  recvGid,  offlinePushInfo);
```
