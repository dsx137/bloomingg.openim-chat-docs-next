通过图片路径创建图片消息。SDK 会根据本地图片路径生成消息体。

## 功能介绍

通过图片路径创建图片消息。SDK 会根据本地图片路径生成消息体。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                              |
| -------- | ----------------------------------------------------------------- |
| 方法签名 | `CreateImageMessage(operationID string, imagePath string) string` |
| 所属能力 | 会话与消息                                                        |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                 |
