## 功能介绍

通过已上传图片 URL 创建图片消息。适合资源已由业务侧上传完成的场景。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                                                       |
| -------- | -------------------------------------------------------------------------------------------------------------------------- |
| 方法签名 | `CreateImageMessageByURL(operationID string, sourcePath string, sourcePicture, bigPicture, snapshotPicture string) string` |
| 所属能力 | 会话与消息                                                                                                                 |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                                                          |

## Android 调用参考

当前 Android SDK 的方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public Message createImageMessageByURL(PictureInfo sourcePicture, PictureInfo bigPicture, PictureInfo snapshotPicture)
```

## 输入参数

| 参数名称        | 参数类型    | 是否必填 | 描述           |
| --------------- | ----------- | -------- | -------------- |
| sourcePicture   | PictureInfo | 是       | 原图相关信息   |
| bigPicture      | PictureInfo | 是       | 大图相关信息   |
| snapshotPicture | PictureInfo | 是       | 缩略图相关信息 |

## 返回结果

| 名称 | 类型    | 描述     |
| ---- | ------- | -------- |
| ~    | Message | 成功返回 |

以下示例展示当前 Android SDK 的调用方式。
