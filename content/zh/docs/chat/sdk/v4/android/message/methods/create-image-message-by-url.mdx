`CreateImageMessageByURL` 来自 `open_im_sdk/conversation_msg.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                                                                     |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| 方法     | `CreateImageMessageByURL(operationID string, sourcePath string, sourcePicture string, bigPicture string, snapshotPicture string) string` |
| 所属能力 | 会话与消息                                                                                                                               |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                                                                        |

## 作用

通过已上传图片 URL 创建图片消息。适合资源已由业务侧上传完成的场景。

## 参数说明

`sourcePath` 是原始路径或来源标识；`sourcePicture`、`bigPicture`、`snapshotPicture` 是不同规格图片信息 JSON。

## 返回与事件

同步返回消息 JSON 字符串，通常配合不走 OSS 的发送链路。

## 使用边界

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。
