`SetConversationListener` 来自 `open_im_sdk/listener.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                       |
| -------- | ---------------------------------------------------------- |
| 方法     | `SetConversationListener(listener OnConversationListener)` |
| 所属能力 | 监听器                                                     |
| 源码位置 | `open_im_sdk/listener.go`                                  |

## 作用

设置会话事件监听器，用于接收服务端同步进度、新会话、会话变更、总未读数和输入状态变化。

## 参数说明

`listener` 实现 `OnConversationListener`。

## 返回与事件

会话列表和状态变化以 JSON 字符串或基础类型回调。

## 使用边界

`Set*Listener` 会替换当前 SDK 保存的同类监听器；如果多个模块都要消费事件，应在业务层封装统一分发。
