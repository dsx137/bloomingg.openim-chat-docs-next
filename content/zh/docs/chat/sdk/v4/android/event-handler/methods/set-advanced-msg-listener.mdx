设置单条消息高级监听器，用于处理新消息、离线消息、已读回执、消息撤回、消息删除和在线消息。

## 功能介绍

设置单条消息高级监听器，用于处理新消息、离线消息、已读回执、消息撤回、消息删除和在线消息。

`Set*Listener` 会替换当前 SDK 保存的同类监听器；如果多个模块都要消费事件，应在业务层封装统一分发。

## Core Binding 方法

| 项目     | 说明                                                     |
| -------- | -------------------------------------------------------- |
| 方法签名 | `SetAdvancedMsgListener(listener OnAdvancedMsgListener)` |
| 所属能力 | 监听器                                                   |
| 源码位置 | `open_im_sdk/listener.go`                                |

## Android 调用参考

当前 Android SDK 的方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void setAdvancedMsgListener(OnAdvanceMsgListener listener)
```

## 输入参数

| 参数名称 | 参数类型              | 描述 |
| -------- | --------------------- | ---- |
| listener | OnAdvancedMsgListener | 是   |

## 返回结果

无直接返回值。
