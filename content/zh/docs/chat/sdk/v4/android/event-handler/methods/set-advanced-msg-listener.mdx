设置单条消息高级监听器，用于处理新消息、离线消息、已读回执、消息撤回、消息删除和在线消息。

## 功能介绍

设置单条消息高级监听器，用于处理新消息、离线消息、已读回执、消息撤回、消息删除和在线消息。

`Set*Listener` 会替换当前 SDK 保存的同类监听器；如果多个模块都要消费事件，应在业务层封装统一分发。

## Core Binding 方法

| 项目     | 说明                                                     |
| -------- | -------------------------------------------------------- |
| 方法签名 | `SetAdvancedMsgListener(listener OnAdvancedMsgListener)` |
| 所属能力 | 监听器                                                   |
| 源码位置 | `open_im_sdk/listener.go`                                |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void setAdvancedMsgListener(OnAdvanceMsgListener listener)
```

## 输入参数

| 参数名称 | 参数类型                                                                             | 描述 |
| -------- | ------------------------------------------------------------------------------------ | ---- |
| listener | [`OnAdvancedMsgListener`](/docs/chat/sdk/v4/android/models/on-advanced-msg-listener) | 是   |

## 返回结果

无直接返回值。

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().messageManager.setAdvancedMsgListener(new OnAdvanceMsgListener() {
    @Override
    public void onRecvNewMessage(Message msg) {
        // TODO: 处理新消息
    }

    @Override
    public void onRecvC2CReadReceipt(List<C2CReadReceiptInfo> list) {
        // TODO: 处理单聊已读回执
    }

    @Override
    public void onRecvGroupMessageReadReceipt(GroupMessageReceipt groupMessageReceipt) {
        // TODO: 处理群聊已读回执
    }

    @Override
    public void onRecvMessageRevokedV2(RevokedInfo info) {
        // TODO: 处理消息撤回
    }
});
```

## 相关类型

- [底层高级消息监听器](/docs/chat/sdk/v4/android/models/on-advanced-msg-listener)（`OnAdvancedMsgListener`）：core binding 高级消息监听器，Android 调用层对应 OnAdvanceMsgListener。
- [单聊已读回执信息](/docs/chat/sdk/v4/android/models/c2c-read-receipt-info)（`C2CReadReceiptInfo`）：单聊消息已读回执信息。
- [高级消息监听器](/docs/chat/sdk/v4/android/models/on-advance-msg-listener)（`OnAdvanceMsgListener`）：接收新消息、已读回执、撤回和消息扩展事件的监听器。
- [群消息已读回执](/docs/chat/sdk/v4/android/models/group-message-receipt)（`GroupMessageReceipt`）：群消息已读回执。
- [消息撤回信息](/docs/chat/sdk/v4/android/models/revoked-info)（`RevokedInfo`）：消息撤回事件中的撤回详情。
- [消息对象](/docs/chat/sdk/v4/android/models/message)（`Message`）：消息对象，包含消息 ID、发送者、会话、内容元素和状态等信息。
