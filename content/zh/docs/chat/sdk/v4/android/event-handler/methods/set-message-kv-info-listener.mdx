设置消息 KV 信息变化监听器，用于接收消息扩展键值信息变化。

## 功能介绍

设置消息 KV 信息变化监听器，用于接收消息扩展键值信息变化。

`Set*Listener` 会替换当前 SDK 保存的同类监听器；如果多个模块都要消费事件，应在业务层封装统一分发。

## Core Binding 方法

| 项目     | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| 方法签名 | `SetMessageKvInfoListener(listener OnMessageKvInfoListener)` |
| 所属能力 | 监听器                                                       |
| 源码位置 | `open_im_sdk/listener.go`                                    |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void setMessageKvInfoListener(OnMessageKvInfoListener listener)
```

## 输入参数

| 参数名称   | 类型                      | 描述             |
| ---------- | ------------------------- | ---------------- |
| `listener` | `OnMessageKvInfoListener` | 事件监听器实例。 |

## 返回结果

无直接返回值。

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().messageManager.setMessageKvInfoListener(new OnMessageKvInfoListener() {
    @Override
    public void onMessageKvInfoChanged(List<MessageKv> list) {
        // TODO: 处理消息扩展信息变更
    }
});
```
