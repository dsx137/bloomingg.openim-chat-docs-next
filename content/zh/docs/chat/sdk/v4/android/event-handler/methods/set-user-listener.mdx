设置用户事件监听器，用于接收当前用户资料更新、用户在线状态变化和用户命令增删改事件。

## 功能介绍

设置用户事件监听器，用于接收当前用户资料更新、用户在线状态变化和用户命令增删改事件。

`Set*Listener` 会替换当前 SDK 保存的同类监听器；如果多个模块都要消费事件，应在业务层封装统一分发。

## Core Binding 方法

| 项目     | 说明                                       |
| -------- | ------------------------------------------ |
| 方法签名 | `SetUserListener(listener OnUserListener)` |
| 所属能力 | 监听器                                     |
| 源码位置 | `open_im_sdk/listener.go`                  |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void setOnUserListener(OnUserListener listener)
```

## 输入参数

| 参数名称 | 参数类型       | 是否必填               | 描述 |
| -------- | -------------- | ---------------------- | ---- |
| listener | OnUserListener | 用户信息更新的回调函数 |

## 返回结果

无直接返回值。

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().userInfoManager.setOnUserListener(new OnUserListener() {
    @Override
    public void onSelfInfoUpdated(UserInfo info) {
        // TODO: 处理当前用户资料变更
    }

    @Override
    public void onUserStatusChanged(UsersOnlineStatus onlineStatus) {
        // TODO: 处理用户在线状态变更
    }
});
```
