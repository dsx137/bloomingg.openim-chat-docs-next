`SetUserListener` 来自 `open_im_sdk/listener.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                       |
| -------- | ------------------------------------------ |
| 方法     | `SetUserListener(listener OnUserListener)` |
| 所属能力 | 监听器                                     |
| 源码位置 | `open_im_sdk/listener.go`                  |

## 作用

设置用户事件监听器，用于接收当前用户资料更新、用户在线状态变化和用户命令增删改事件。

## 参数说明

`listener` 实现 `OnUserListener`。

## 返回与事件

用户资料、状态和命令以 JSON 字符串回调。

## 使用边界

`Set*Listener` 会替换当前 SDK 保存的同类监听器；如果多个模块都要消费事件，应在业务层封装统一分发。
