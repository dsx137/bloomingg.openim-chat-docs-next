设置群组事件监听器，用于接收已加入群变化、群资料变化、群成员变化和群申请处理结果。

## 功能介绍

设置群组事件监听器，用于接收已加入群变化、群资料变化、群成员变化和群申请处理结果。

`Set*Listener` 会替换当前 SDK 保存的同类监听器；如果多个模块都要消费事件，应在业务层封装统一分发。

## Core Binding 方法

| 项目     | 说明                                         |
| -------- | -------------------------------------------- |
| 方法签名 | `SetGroupListener(listener OnGroupListener)` |
| 所属能力 | 监听器                                       |
| 源码位置 | `open_im_sdk/listener.go`                    |

## Android 调用参考

当前 Android SDK 的方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void setOnGroupListener(OnGroupListener listener)
```

## 输入参数

| 参数名称 | 参数类型        | 是否必填 | 描述     |
| -------- | --------------- | -------- | -------- |
| listener | OnGroupListener | 是       | 回调接口 |

## 返回结果

无直接返回值。
