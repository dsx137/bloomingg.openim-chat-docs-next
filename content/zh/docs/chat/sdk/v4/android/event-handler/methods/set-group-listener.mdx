接收群组相关事件，用于更新群列表、群资料、群成员、入群申请和群权限变化。

## 功能介绍

群组事件适合驱动群聊资料页、群成员列表和入群申请入口。已加入群变化、群资料更新、成员加入或移出、群申请处理结果，都应通过该监听器进入业务层。

该监听器是单实例替换式注册。建议由群组模块统一维护群缓存和成员缓存，再通知群列表、群资料页和聊天页刷新。

## Core Binding 方法

| 项目     | 说明                                         |
| -------- | -------------------------------------------- |
| 方法签名 | `SetGroupListener(listener OnGroupListener)` |
| 所属能力 | 监听器                                       |
| 源码位置 | `open_im_sdk/listener.go`                    |

## Android 调用参考

Android SDK v3.8.3 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void setOnGroupListener(OnGroupListener listener)
```

## 输入参数

| 参数名称 | 参数类型                                                                | 是否必填 | 描述     |
| -------- | ----------------------------------------------------------------------- | -------- | -------- |
| listener | [`OnGroupListener`](/docs/chat/sdk/v4/android/models/on-group-listener) | 是       | 回调接口 |

## 返回结果

无直接返回值。

## 代码示例

以下示例展示 Android SDK v3.8.3 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().groupManager.setOnGroupListener(new OnGroupListener() {
    @Override
    public void onJoinedGroupAdded(GroupInfo info) {
        // TODO: 处理加入群组
    }

    @Override
    public void onGroupInfoChanged(GroupInfo info) {
        // TODO: 处理群资料变更
    }

    @Override
    public void onGroupMemberAdded(GroupMembersInfo info) {
        // TODO: 处理群成员新增
    }
});
```

## 相关类型

- [群成员信息](/docs/chat/sdk/v4/android/models/group-members-info)（`GroupMembersInfo`）：群成员资料，包含群内昵称、角色、入群来源和禁言时间。
- [群组监听器](/docs/chat/sdk/v4/android/models/on-group-listener)（`OnGroupListener`）：群组、入群申请和群成员变化监听器。
- [群组信息](/docs/chat/sdk/v4/android/models/group-info)（`GroupInfo`）：群组资料对象，包含群名、公告、群主、成员数和入群设置。
