`SetCustomBusinessListener` 来自 `open_im_sdk/listener.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                           |
| -------- | -------------------------------------------------------------- |
| 方法     | `SetCustomBusinessListener(listener OnCustomBusinessListener)` |
| 所属能力 | 监听器                                                         |
| 源码位置 | `open_im_sdk/listener.go`                                      |

## 作用

设置自定义业务消息监听器，用于接收业务自定义通道传入的消息。

## 参数说明

`listener` 实现 `OnCustomBusinessListener`。

## 返回与事件

自定义业务消息以 JSON 字符串回调。

## 使用边界

`Set*Listener` 会替换当前 SDK 保存的同类监听器；如果多个模块都要消费事件，应在业务层封装统一分发。
