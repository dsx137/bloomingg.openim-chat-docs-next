`GetGroupMemberListByJoinTimeFilter` 来自 `open_im_sdk/group.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                                                                                                                |
| -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 方法     | `GetGroupMemberListByJoinTimeFilter(callback Base, operationID string, groupID string, offset int32, count int32, joinTimeBegin int64, joinTimeEnd int64, filterUserIDList string)` |
| 所属能力 | 群组                                                                                                                                                                                |
| 源码位置 | `open_im_sdk/group.go`                                                                                                                                                              |

## 作用

按入群时间范围和用户过滤条件获取群成员，适合审计、成员管理和增量查看。

## 参数说明

`joinTimeBegin` 与 `joinTimeEnd` 是时间范围；`filterUserIDList` 是需要过滤的用户 ID 列表 JSON。

## 返回与事件

成功返回群成员列表 JSON。

## 使用边界

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。
