按成员类型筛选并分页获取群成员列表。

## 功能介绍

按成员类型筛选并分页获取群成员列表。

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。

## Core Binding 方法

| 项目     | 说明                                                                                                             |
| -------- | ---------------------------------------------------------------------------------------------------------------- |
| 方法签名 | `GetGroupMemberList(callback Base, operationID string, groupID string, filter int32, offset int32, count int32)` |
| 所属能力 | 群组                                                                                                             |
| 源码位置 | `open_im_sdk/group.go`                                                                                           |

## Android 调用参考

当前 Android SDK 的方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void getGroupMemberList(OnBase<List<GroupMembersInfo>> callBack, String groupId, int filter, int offset, int count)
```

## 输入参数

| 参数名称 | 参数类型                       | 是否必填 | 描述                       |
| -------- | ------------------------------ | -------- | -------------------------- |
| callBack | OnBase<List<GroupMembersInfo>> | 是       | 回调接口                   |
| groupId  | String                         | 是       | 群 ID                      |
| filter   | int                            | 否       | GroupMemberFilter 过滤成员 |
| offset   | int                            | 是       | 偏移量                     |
| count    | int                            | 是       | 每页数量                   |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。

## 代码示例

以下示例展示当前 Android SDK 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().groupManager.getGroupMemberList(new OnBase<List<GroupMembersInfo>>() {
            @Override
            public void onError(int code, String error) {

            }

            @Override
            public void onSuccess(List<GroupMembersInfo> data) {

            }
        }, groupId,  filter,  offset,  count);
```
