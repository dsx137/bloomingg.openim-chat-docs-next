`GetGroupMemberList` 来自 `open_im_sdk/group.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                                             |
| -------- | ---------------------------------------------------------------------------------------------------------------- |
| 方法     | `GetGroupMemberList(callback Base, operationID string, groupID string, filter int32, offset int32, count int32)` |
| 所属能力 | 群组                                                                                                             |
| 源码位置 | `open_im_sdk/group.go`                                                                                           |

## 作用

按成员类型筛选并分页获取群成员列表。

## 参数说明

`filter` 指定成员筛选类型；`offset` 和 `count` 控制分页。

## 返回与事件

成功返回群成员列表 JSON。

## 使用边界

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。
