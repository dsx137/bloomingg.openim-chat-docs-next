`GetSpecifiedGroupsInfo` 来自 `open_im_sdk/group.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                            |
| -------- | ------------------------------------------------------------------------------- |
| 方法     | `GetSpecifiedGroupsInfo(callback Base, operationID string, groupIDList string)` |
| 所属能力 | 群组                                                                            |
| 源码位置 | `open_im_sdk/group.go`                                                          |

## 作用

批量获取指定群组资料，适合会话列表补齐群信息或刷新部分群资料。

## 参数说明

`groupIDList` 是群 ID 列表 JSON。

## 返回与事件

成功返回群资料列表 JSON。

## 使用边界

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。
