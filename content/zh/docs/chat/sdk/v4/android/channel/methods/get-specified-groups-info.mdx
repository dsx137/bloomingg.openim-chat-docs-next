批量获取指定群组资料，适合会话列表补齐群信息或刷新部分群资料。

## 功能介绍

批量获取指定群组资料，适合会话列表补齐群信息或刷新部分群资料。

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。

## Core Binding 方法

| 项目     | 说明                                                                            |
| -------- | ------------------------------------------------------------------------------- |
| 方法签名 | `GetSpecifiedGroupsInfo(callback Base, operationID string, groupIDList string)` |
| 所属能力 | 群组                                                                            |
| 源码位置 | `open_im_sdk/group.go`                                                          |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void getGroupsInfo(OnBase<List<GroupInfo>> base, List<String> gidList)
```

## 输入参数

| 参数名称 | 参数类型                                                                                                                           | 是否必填 | 描述      |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------- | -------- | --------- |
| callBack | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)&lt;List&lt;[`GroupInfo`](/docs/chat/sdk/v4/android/models/group-info)&gt;&gt; | 是       | 回调接口  |
| gidList  | List<String>                                                                                                                       | 是       | 群ID 列表 |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().groupManager.getGroupsInfo(new OnBase<List<GroupInfo>>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(List<GroupInfo> data) {
        // TODO: 处理群资料列表
    }
}, gidList);
```
