OpenIM Android core 使用群组承载多人聊天能力。本节将 channel 场景映射为 OpenIM 群组；当前 core 没有 open channel 的独立导出方法。

本页是群组方法目录。创建、加入、成员、申请、禁言和群管理方法都拆到了独立页面。

## 事件联动

群组相关变化通过 `SetGroupListener` 注册的 `OnGroupListener` 接收，包括群新增/删除、群资料变化、成员新增/删除、成员资料变化、群申请新增/删除/接受/拒绝等。

## 相关类型

- [群组监听器](/docs/chat/sdk/v4/android/models/on-group-listener)（`OnGroupListener`）：群组、入群申请和群成员变化监听器。
