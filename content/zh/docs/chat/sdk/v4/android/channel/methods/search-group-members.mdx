搜索群成员，适合按用户 ID 或群昵称查找成员。

## 功能介绍

搜索群成员，适合按用户 ID 或群昵称查找成员。

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。

## Core Binding 方法

| 项目     | 说明                                                                        |
| -------- | --------------------------------------------------------------------------- |
| 方法签名 | `SearchGroupMembers(callback Base, operationID string, searchParam string)` |
| 所属能力 | 群组                                                                        |
| 源码位置 | `open_im_sdk/group.go`                                                      |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void searchGroupMembers(OnBase<List<GroupMembersInfo>> base, String groupID,
                               List<String> keywordList, boolean isSearchUserID,
                               boolean isSearchMemberNickname, int offset, int count)
```

## 输入参数

| 参数名称               | 参数类型                                                                                                                                          | 是否必填 | 描述           |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- | -------- | -------------- |
| callBack               | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)&lt;List&lt;[`GroupMembersInfo`](/docs/chat/sdk/v4/android/models/group-members-info)&gt;&gt; | 是       | 回调接口       |
| groupID                | String                                                                                                                                            | 是       | 群ID           |
| keywordList            | List<String>                                                                                                                                      | 是       | 关键字         |
| isSearchUserID         | Boolean                                                                                                                                           | 是       | 检索用户 ID    |
| isSearchMemberNickname | Boolean                                                                                                                                           | 是       | 检索用户昵称   |
| offset                 | int                                                                                                                                               | 是       | 起始偏移量     |
| count                  | int                                                                                                                                               | 是       | 每次查询的总数 |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().groupManager.searchGroupMembers(
    new OnBase<List<GroupMembersInfo>>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(List<GroupMembersInfo> data) {
        // TODO: 处理搜索到的群成员
    }
},
    groupID,
    keywordList,
    isSearchUserID,
    isSearchMemberNickname,
    offset,
    count
);
```

## 相关类型

- [底层通用回调](/docs/chat/sdk/v4/android/models/base)（`Base`）：core binding 异步结果回调，Android 调用层通常映射为 OnBase。
- [通用回调](/docs/chat/sdk/v4/android/models/on-base)（`OnBase`）：Android SDK 异步方法的通用成功和失败回调。
- [群成员信息](/docs/chat/sdk/v4/android/models/group-members-info)（`GroupMembersInfo`）：群成员资料，包含群内昵称、角色、入群来源和禁言时间。
