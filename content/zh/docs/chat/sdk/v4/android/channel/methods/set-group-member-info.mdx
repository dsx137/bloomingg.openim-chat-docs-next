`SetGroupMemberInfo` 来自 `open_im_sdk/group.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                            |
| -------- | ------------------------------------------------------------------------------- |
| 方法     | `SetGroupMemberInfo(callback Base, operationID string, groupMemberInfo string)` |
| 所属能力 | 群组                                                                            |
| 源码位置 | `open_im_sdk/group.go`                                                          |

## 作用

更新群成员资料，例如群昵称、角色相关扩展或其他成员字段。

## 参数说明

`groupMemberInfo` 是群成员资料 JSON。

## 返回与事件

成功返回更新结果，并触发群成员资料变化事件。

## 使用边界

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。
