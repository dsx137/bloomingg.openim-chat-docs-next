`KickGroupMember` 来自 `open_im_sdk/group.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                                   |
| -------- | ------------------------------------------------------------------------------------------------------ |
| 方法     | `KickGroupMember(callback Base, operationID string, groupID string, reason string, userIDList string)` |
| 所属能力 | 群组                                                                                                   |
| 源码位置 | `open_im_sdk/group.go`                                                                                 |

## 作用

将一批成员移出群组，通常要求操作者具备管理权限。

## 参数说明

`reason` 是移除原因；`userIDList` 是成员用户 ID 列表 JSON。

## 返回与事件

成功返回移除结果，并触发群成员删除事件。

## 使用边界

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。
