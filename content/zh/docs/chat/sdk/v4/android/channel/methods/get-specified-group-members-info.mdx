获取指定群内指定用户的群成员资料，适合查看成员详情或刷新少量成员信息。

## 功能介绍

获取指定群内指定用户的群成员资料，适合查看成员详情或刷新少量成员信息。

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。

## Core Binding 方法

| 项目     | 说明                                                                                                 |
| -------- | ---------------------------------------------------------------------------------------------------- |
| 方法签名 | `GetSpecifiedGroupMembersInfo(callback Base, operationID string, groupID string, userIDList string)` |
| 所属能力 | 群组                                                                                                 |
| 源码位置 | `open_im_sdk/group.go`                                                                               |

## Android 调用参考

当前 Android SDK 的方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void getGroupMembersInfo(OnBase<List<GroupMembersInfo>> callBack, String groupId, List<String> uidList)
```

## 输入参数

| 参数名称 | 参数类型                       | 是否必填 | 描述         |
| -------- | ------------------------------ | -------- | ------------ |
| callBack | OnBase<List<GroupMembersInfo>> | 是       | 回调接口     |
| groupId  | String                         | 是       | 群ID         |
| uidList  | List<String>                   | 是       | 成员 ID 列表 |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。

            }
        },groupId,uidList)
