`GetSpecifiedGroupMembersInfo` 来自 `open_im_sdk/group.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                                 |
| -------- | ---------------------------------------------------------------------------------------------------- |
| 方法     | `GetSpecifiedGroupMembersInfo(callback Base, operationID string, groupID string, userIDList string)` |
| 所属能力 | 群组                                                                                                 |
| 源码位置 | `open_im_sdk/group.go`                                                                               |

## 作用

获取指定群内指定用户的群成员资料，适合查看成员详情或刷新少量成员信息。

## 参数说明

`groupID` 是群 ID；`userIDList` 是用户 ID 列表 JSON。

## 返回与事件

成功返回群成员资料列表 JSON。

## 使用边界

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。
