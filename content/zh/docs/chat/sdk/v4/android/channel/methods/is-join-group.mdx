判断当前用户是否已经加入指定群组，适合展示入群按钮或发送群消息前校验。

## 功能介绍

判断当前用户是否已经加入指定群组，适合展示入群按钮或发送群消息前校验。

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。

## Core Binding 方法

| 项目     | 说明                                                             |
| -------- | ---------------------------------------------------------------- |
| 方法签名 | `IsJoinGroup(callback Base, operationID string, groupID string)` |
| 所属能力 | 群组                                                             |
| 源码位置 | `open_im_sdk/group.go`                                           |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void isJoinGroup(String gid, OnBase<Boolean> callBack)
```

## 输入参数

| 参数名称 | 参数类型                                                            | 是否必填 | 描述                                   |
| -------- | ------------------------------------------------------------------- | -------- | -------------------------------------- |
| gid      | String                                                              | 是       | 群 ID。                                |
| callBack | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)&lt;Boolean&gt; | 是       | 查询结果回调，成功返回是否已加入该群。 |

## 返回结果

| 参数名称 | 参数类型 | 描述                     |
| -------- | -------- | ------------------------ |
| data     | Boolean  | 当前用户是否已加入该群。 |

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().groupManager.isJoinGroup(gid, new OnBase<Boolean>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(Boolean data) {
        // TODO: 根据 data 判断当前用户是否已加入该群
    }
});
```
