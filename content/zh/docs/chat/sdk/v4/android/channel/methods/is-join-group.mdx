`IsJoinGroup` 来自 `open_im_sdk/group.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                             |
| -------- | ---------------------------------------------------------------- |
| 方法     | `IsJoinGroup(callback Base, operationID string, groupID string)` |
| 所属能力 | 群组                                                             |
| 源码位置 | `open_im_sdk/group.go`                                           |

## 作用

判断当前用户是否已经加入指定群组，适合展示入群按钮或发送群消息前校验。

## 参数说明

`groupID` 是目标群 ID。

## 返回与事件

成功返回是否已入群的结果 JSON。

## 使用边界

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。
