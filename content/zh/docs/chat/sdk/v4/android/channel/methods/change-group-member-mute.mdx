`ChangeGroupMemberMute` 来自 `open_im_sdk/group.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                                        |
| -------- | ----------------------------------------------------------------------------------------------------------- |
| 方法     | `ChangeGroupMemberMute(callback Base, operationID string, groupID string, userID string, mutedSeconds int)` |
| 所属能力 | 群组                                                                                                        |
| 源码位置 | `open_im_sdk/group.go`                                                                                      |

## 作用

设置指定群成员禁言时长。适合临时限制单个成员发言。

## 参数说明

`userID` 是被禁言成员；`mutedSeconds` 是禁言秒数，通常 `0` 表示取消禁言。

## 返回与事件

成功返回处理结果，并触发群成员资料变化事件。

## 使用边界

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。
