`GetJoinedGroupListPage` 来自 `open_im_sdk/group.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                   |
| -------- | -------------------------------------------------------------------------------------- |
| 方法     | `GetJoinedGroupListPage(callback Base, operationID string, offset int32, count int32)` |
| 所属能力 | 群组                                                                                   |
| 源码位置 | `open_im_sdk/group.go`                                                                 |

## 作用

分页获取当前用户已加入群组，适合群数量较多的账号。

## 参数说明

`offset` 和 `count` 控制分页。

## 返回与事件

成功返回当前页群列表 JSON。

## 使用边界

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。
