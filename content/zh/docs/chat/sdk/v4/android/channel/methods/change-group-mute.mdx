`ChangeGroupMute` 来自 `open_im_sdk/group.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                              |
| -------- | --------------------------------------------------------------------------------- |
| 方法     | `ChangeGroupMute(callback Base, operationID string, groupID string, isMute bool)` |
| 所属能力 | 群组                                                                              |
| 源码位置 | `open_im_sdk/group.go`                                                            |

## 作用

开启或关闭全群禁言。适合群管理者控制群内发言状态。

## 参数说明

`groupID` 是目标群 ID；`isMute` 为 `true` 表示开启全群禁言。

## 返回与事件

成功返回处理结果，并触发群资料变化事件。

## 使用边界

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。
