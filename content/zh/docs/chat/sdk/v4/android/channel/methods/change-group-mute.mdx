开启或关闭全群禁言。适合群管理者控制群内发言状态。

## 功能介绍

开启或关闭全群禁言。适合群管理者控制群内发言状态。

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。

## Core Binding 方法

| 项目     | 说明                                                                              |
| -------- | --------------------------------------------------------------------------------- |
| 方法签名 | `ChangeGroupMute(callback Base, operationID string, groupID string, isMute bool)` |
| 所属能力 | 群组                                                                              |
| 源码位置 | `open_im_sdk/group.go`                                                            |

## Android 调用参考

当前 Android SDK 的方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void changeGroupMute(OnBase<String> callBack, String gid, boolean mute)
```

## 输入参数

| 参数名称 | 参数类型 | 是否必填 | 描述                     |
| -------- | -------- | -------- | ------------------------ |
| callBack | OnBase   | 是       | 回调接口                 |
| gid      | String   | 是       | 群ID                     |
| mute     | Boolean  | 是       | true 禁言 false 取消禁言 |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。

## 代码示例

以下示例展示当前 Android SDK 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().groupManager.changeGroupMute(new OnBase<String>() {
            @Override
            public void onError(int code, String error) {

            }

            @Override
            public void onSuccess(String data) {

            }
        },gid,mute)
```
