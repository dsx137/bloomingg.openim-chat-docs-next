`SetGroupInfo` 来自 `open_im_sdk/group.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                |
| -------- | ------------------------------------------------------------------- |
| 方法     | `SetGroupInfo(callback Base, operationID string, groupInfo string)` |
| 所属能力 | 群组                                                                |
| 源码位置 | `open_im_sdk/group.go`                                              |

## 作用

更新群资料，例如群名称、头像、公告或扩展字段。

## 参数说明

`groupInfo` 是群资料 JSON。

## 返回与事件

成功返回更新结果，并触发群资料变化事件。

## 使用边界

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。
