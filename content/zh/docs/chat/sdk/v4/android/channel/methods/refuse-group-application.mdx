`RefuseGroupApplication` 来自 `open_im_sdk/group.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                                             |
| -------- | ---------------------------------------------------------------------------------------------------------------- |
| 方法     | `RefuseGroupApplication(callback Base, operationID string, groupID string, fromUserID string, handleMsg string)` |
| 所属能力 | 群组                                                                                                             |
| 源码位置 | `open_im_sdk/group.go`                                                                                           |

## 作用

拒绝指定用户的入群申请。

## 参数说明

`groupID` 是群 ID；`fromUserID` 是申请人；`handleMsg` 是处理说明。

## 返回与事件

成功返回处理结果，并触发申请拒绝事件。

## 使用边界

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。
