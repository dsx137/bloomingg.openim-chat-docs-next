`SetAppBadge` 来自 `open_im_sdk/third.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                   |
| -------- | ---------------------------------------------------------------------- |
| 方法     | `SetAppBadge(callback Base, operationID string, appUnreadCount int32)` |
| 所属能力 | 推送、日志与文件                                                       |
| 源码位置 | `open_im_sdk/third.go`                                                 |

## 作用

设置应用角标未读数，适合未读数变化后同步系统角标。

## 参数说明

`appUnreadCount` 是要展示的未读数量。

## 返回与事件

成功返回设置结果。

## 使用边界

第三方能力至少要求 SDK 已初始化；日志和文件上传的进度通过对应 progress callback 返回。
