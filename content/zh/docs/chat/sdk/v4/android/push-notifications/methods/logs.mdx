## 功能介绍

主动写入一条客户端日志，适合业务层把关键 IM 操作写入 SDK 日志链路。

第三方能力至少要求 SDK 已初始化；日志和文件上传的进度通过对应 progress callback 返回。

## Core Binding 方法

| 项目     | 说明                                                                                                                        |
| -------- | --------------------------------------------------------------------------------------------------------------------------- |
| 方法签名 | `Logs(callback Base, operationID string, logLevel int, file string, line int, msgs string, err string, keyAndValue string)` |
| 所属能力 | 推送、日志与文件                                                                                                            |
| 源码位置 | `open_im_sdk/third.go`                                                                                                      |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void logs(@LogLevel int logLevel, String currentClassName, int currentLineNum, @NonNull String msgStr, @Nullable String errStr, @Nullable String... extra)
```

## 输入参数

| 参数名称         | 参数类型  | 是否必填 | 描述                                                                           |
| ---------------- | --------- | -------- | ------------------------------------------------------------------------------ |
| logLevel         | int       | 是       | 日志级别，使用 [`LogLevel`](/docs/chat/sdk/v4/android/models/log-level) 常量。 |
| currentClassName | String    | 是       | 产生日志的类名或文件名。                                                       |
| currentLineNum   | int       | 是       | 产生日志的代码行号。                                                           |
| msgStr           | String    | 是       | 日志正文。                                                                     |
| errStr           | String    | 否       | 错误详情；Warn、Error、Fatal 等级建议传入。                                    |
| extra            | String... | 否       | 额外键值对，按 key、value 成对传入。                                           |

## 返回结果

无直接返回值。SDK 会将日志写入内部日志系统。

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().logs(
    LogLevel.Info,
    "ChatActivity",
    120,
    "send message",
    null,
    "conversationID",
    conversationID
);
```
