## 功能介绍

上传 SDK 日志，适合用户反馈、故障排查或线上问题定位。

第三方能力至少要求 SDK 已初始化；日志和文件上传的进度通过对应 progress callback 返回。

## Core Binding 方法

| 项目     | 说明                                                                                             |
| -------- | ------------------------------------------------------------------------------------------------ |
| 方法签名 | `UploadLogs(callback Base, operationID string, line int, ex string, progress UploadLogProgress)` |
| 所属能力 | 推送、日志与文件                                                                                 |
| 源码位置 | `open_im_sdk/third.go`                                                                           |

## Android 调用参考

当前 Android SDK 的方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void getSelfUserInfo(OnBase<UserInfo> callBack)
```

## 输入参数

| 参数名称 | 参数类型         | 是否必填 | 描述     |
| -------- | ---------------- | -------- | -------- |
| callBack | OnBase<UserInfo> | 是       | 回调接口 |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。

## 代码示例

以下示例展示当前 Android SDK 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().friendshipManager.getSelfUserInfo(new OnBase<UserInfo>(){…})
```
