`UploadFile` 来自 `open_im_sdk/third.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                     |
| -------- | ---------------------------------------------------------------------------------------- |
| 方法     | `UploadFile(callback Base, operationID string, req string, progress UploadFileCallback)` |
| 所属能力 | 推送、日志与文件                                                                         |
| 源码位置 | `open_im_sdk/third.go`                                                                   |

## 作用

上传文件到 SDK 配置的对象存储链路，适合发送文件、图片、音视频前先完成资源上传。

## 参数说明

`req` 是上传请求 JSON；`progress` 接收打开、分片、哈希、上传和完成事件。

## 返回与事件

成功返回文件上传结果，详细过程走 `UploadFileCallback`。

## 使用边界

第三方能力至少要求 SDK 已初始化；日志和文件上传的进度通过对应 progress callback 返回。
