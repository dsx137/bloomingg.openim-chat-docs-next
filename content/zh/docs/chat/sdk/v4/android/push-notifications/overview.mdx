第三方能力来自 `open_im_sdk/third.go`。当前 Android core 只暴露 FCM token、应用角标、日志上传、日志写入和文件上传；HMS、多设备推送模板和推送测试器当前不是 core 的独立导出方法。

本页是推送、日志与文件方法目录。FCM token、角标、日志和文件上传方法都拆成独立页面。

## 方法索引

| 方法                                                                                    | 说明                                                                            |
| --------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------- |
| [更新 FCM Token](/docs/chat/sdk/v4/android/push-notifications/methods/update-fcm-token) | 更新当前设备的 FCM token，用于服务端向 Android 设备发送离线推送。               |
| [设置应用角标](/docs/chat/sdk/v4/android/push-notifications/methods/set-app-badge)      | 设置应用角标未读数，适合未读数变化后同步系统角标。                              |
| [上传日志](/docs/chat/sdk/v4/android/push-notifications/methods/upload-logs)            | 上传 SDK 日志，适合用户反馈、故障排查或线上问题定位。                           |
| [写入客户端日志](/docs/chat/sdk/v4/android/push-notifications/methods/logs)             | 主动写入一条客户端日志，适合业务层把关键 IM 操作写入 SDK 日志链路。             |
| [上传文件](/docs/chat/sdk/v4/android/push-notifications/methods/upload-file)            | 上传文件到 SDK 配置的对象存储链路，适合发送文件、图片、音视频前先完成资源上传。 |

## 上传回调

| 回调                                                                                     | 说明                   |
| ---------------------------------------------------------------------------------------- | ---------------------- |
| `UploadLogProgress.OnProgress(current int64, size int64)`                                | 日志上传进度。         |
| `UploadFileCallback.Open(size int64)`                                                    | 文件打开并得到总大小。 |
| `UploadFileCallback.PartSize(partSize int64, num int)`                                   | 分片大小和分片数量。   |
| `UploadFileCallback.HashPartProgress(index int, size int64, partHash string)`            | 分片哈希进度。         |
| `UploadFileCallback.HashPartComplete(partsHash string, fileHash string)`                 | 所有分片哈希完成。     |
| `UploadFileCallback.UploadID(uploadID string)`                                           | 获取上传 ID。          |
| `UploadFileCallback.UploadPartComplete(index int, partSize int64, partHash string)`      | 单个分片上传完成。     |
| `UploadFileCallback.UploadComplete(fileSize int64, streamSize int64, storageSize int64)` | 文件上传完成阶段信息。 |
| `UploadFileCallback.Complete(size int64, url string, typ int)`                           | 文件完成并返回 URL。   |
