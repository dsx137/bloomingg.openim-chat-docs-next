`UpdateFcmToken` 来自 `open_im_sdk/third.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                                   |
| -------- | -------------------------------------------------------------------------------------- |
| 方法     | `UpdateFcmToken(callback Base, operationID string, fcmToken string, expireTime int64)` |
| 所属能力 | 推送、日志与文件                                                                       |
| 源码位置 | `open_im_sdk/third.go`                                                                 |

## 作用

更新当前设备的 FCM token，用于服务端向 Android 设备发送离线推送。

## 参数说明

`fcmToken` 是 Firebase 返回的 token；`expireTime` 是过期时间。

## 返回与事件

成功返回更新结果。

## 使用边界

第三方能力至少要求 SDK 已初始化；日志和文件上传的进度通过对应 progress callback 返回。
