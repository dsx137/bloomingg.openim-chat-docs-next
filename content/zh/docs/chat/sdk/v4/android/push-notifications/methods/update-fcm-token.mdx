更新当前设备的 FCM token，用于服务端向 Android 设备发送离线推送。

## 功能介绍

更新当前设备的 FCM token，用于服务端向 Android 设备发送离线推送。

第三方能力至少要求 SDK 已初始化；日志和文件上传的进度通过对应 progress callback 返回。

## Core Binding 方法

| 项目     | 说明                                                                            |
| -------- | ------------------------------------------------------------------------------- |
| 方法签名 | `UpdateFcmToken(callback Base, operationID, fcmToken string, expireTime int64)` |
| 所属能力 | 推送、日志与文件                                                                |
| 源码位置 | `open_im_sdk/third.go`                                                          |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void updateFcmToken(OnBase<String> base, String fcmToken, long expireTime)
```

## 输入参数

| 参数名称      | 类型     | 描述                       |
| ------------- | -------- | -------------------------- |
| `callback`    | `Base`   | 异步结果回调。             |
| `operationID` | `string` | 本次操作的追踪 ID。        |
| `fcmToken`    | `string` | 按 core binding 签名传入。 |
| `expireTime`  | `int64`  | 按 core binding 签名传入。 |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().updateFcmToken(new OnBase<String>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(String data) {
        // TODO: 处理成功结果
    }
}, fcmToken, expireTime);
```
