初始化 SDK 全局实例、日志、长连接管理、本地数据目录和基础资源。应用进程内通常只调用一次。

## 功能介绍

初始化 SDK 全局实例、日志、长连接管理、本地数据目录和基础资源。应用进程内通常只调用一次。

初始化类方法应在应用进程内按需调用；用户态方法通常要求 `InitSDK` 已成功执行，并通过 `operationID` 串联日志。

## Core Binding 方法

| 项目     | 说明                                                                       |
| -------- | -------------------------------------------------------------------------- |
| 方法签名 | `InitSDK(listener OnConnListener, operationID string, config string) bool` |
| 所属能力 | 初始化与登录                                                               |
| 源码位置 | `open_im_sdk/init_login.go`                                                |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public boolean initSDK(Application application, InitConfig initConfig, @NotNull OnConnListener listener)
```

## 输入参数

| 参数名称      | 类型                                                                  | 描述                       |
| ------------- | --------------------------------------------------------------------- | -------------------------- |
| `listener`    | [`OnConnListener`](/docs/chat/sdk/v4/android/models/on-conn-listener) | 事件监听器实例。           |
| `operationID` | `string`                                                              | 本次操作的追踪 ID。        |
| `config`      | `string`                                                              | 按 core binding 签名传入。 |

## 返回结果

同步返回 `bool`。

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
boolean initialized = OpenIMClient.getInstance().initSDK(application, initConfig, new OnConnListener() {
    @Override
    public void onConnectSuccess() {
        // TODO: SDK 已连接
    }

    @Override
    public void onConnectFailed(int code, String error) {
        // TODO: 处理连接失败
    }

    @Override
    public void onUserTokenExpired() {
        // TODO: 处理登录票据过期
    }
});
```

## 相关类型
