`SetAppBackgroundStatus` 来自 `open_im_sdk/init_login.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                           |
| -------- | ------------------------------------------------------------------------------ |
| 方法     | `SetAppBackgroundStatus(callback Base, operationID string, isBackground bool)` |
| 所属能力 | 初始化与登录                                                                   |
| 源码位置 | `open_im_sdk/init_login.go`                                                    |

## 作用

通知 SDK 应用进入前台或后台，便于 SDK 调整连接、同步和推送相关行为。

## 参数说明

`isBackground` 为 `true` 表示进入后台，`false` 表示回到前台。

## 返回与事件

通过 `Base` 回调返回处理结果。

## 使用边界

初始化类方法应在应用进程内按需调用；用户态方法通常要求 `InitSDK` 已成功执行，并通过 `operationID` 串联日志。
