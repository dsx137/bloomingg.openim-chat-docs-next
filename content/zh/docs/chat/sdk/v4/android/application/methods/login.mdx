`Login` 来自 `open_im_sdk/init_login.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                                                    |
| -------- | ----------------------------------------------------------------------- |
| 方法     | `Login(callback Base, operationID string, userID string, token string)` |
| 所属能力 | 初始化与登录                                                            |
| 源码位置 | `open_im_sdk/init_login.go`                                             |

## 作用

使用业务服务端签发的 token 登录指定用户。登录成功后才能可靠调用会话、消息、群组、好友等用户态接口。

## 参数说明

`userID` 是 OpenIM 用户 ID；`token` 必须与该用户匹配；`operationID` 建议每次登录唯一。

## 返回与事件

成功或失败通过 `Base` 回调返回。

## 使用边界

初始化类方法应在应用进程内按需调用；用户态方法通常要求 `InitSDK` 已成功执行，并通过 `operationID` 串联日志。
