`GetSdkVersion` 来自 `open_im_sdk/init_login.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                        |
| -------- | --------------------------- |
| 方法     | `GetSdkVersion() string`    |
| 所属能力 | 初始化与登录                |
| 源码位置 | `open_im_sdk/init_login.go` |

## 作用

获取当前 core SDK 的版本号，适合在启动日志、问题反馈和兼容性判断中记录。

## 参数说明

无入参。

## 返回与事件

直接返回版本字符串。

## 使用边界

初始化类方法应在应用进程内按需调用；用户态方法通常要求 `InitSDK` 已成功执行，并通过 `operationID` 串联日志。
