`UnInitSDK` 来自 `open_im_sdk/init_login.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                            |
| -------- | ------------------------------- |
| 方法     | `UnInitSDK(operationID string)` |
| 所属能力 | 初始化与登录                    |
| 源码位置 | `open_im_sdk/init_login.go`     |

## 作用

释放当前 SDK 全局实例，适合应用完全退出 IM 能力或测试环境重建实例时使用。

## 参数说明

`operationID` 用于日志追踪。

## 返回与事件

无业务数据返回；未初始化时只打印状态。

## 使用边界

初始化类方法应在应用进程内按需调用；用户态方法通常要求 `InitSDK` 已成功执行，并通过 `operationID` 串联日志。
