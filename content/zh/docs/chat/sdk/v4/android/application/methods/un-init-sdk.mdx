## 功能介绍

释放当前 SDK 全局实例，适合应用完全退出 IM 能力或测试环境重建实例时使用。

初始化类方法应在应用进程内按需调用；用户态方法通常要求 `InitSDK` 已成功执行，并通过 `operationID` 串联日志。

## Core Binding 方法

| 项目     | 说明                            |
| -------- | ------------------------------- |
| 方法签名 | `UnInitSDK(operationID string)` |
| 所属能力 | 初始化与登录                    |
| 源码位置 | `open_im_sdk/init_login.go`     |
