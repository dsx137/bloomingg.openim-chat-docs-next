## 功能介绍

通知 SDK 网络状态已变化，当前实现会触发长连接关闭并进入后续重连处理。

初始化类方法应在应用进程内按需调用；用户态方法通常要求 `InitSDK` 已成功执行，并通过 `operationID` 串联日志。

## Core Binding 方法

| 项目     | 说明                                                      |
| -------- | --------------------------------------------------------- |
| 方法签名 | `NetworkStatusChanged(callback Base, operationID string)` |
| 所属能力 | 初始化与登录                                              |
| 源码位置 | `open_im_sdk/init_login.go`                               |
