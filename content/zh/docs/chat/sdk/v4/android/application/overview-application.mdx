初始化与登录方法来自 `open_im_sdk/init_login.go`。初始化成功后，SDK 会创建全局 `UserForSDK`；后续用户、群组、会话、消息和第三方能力都依赖这个实例。

本页是初始化与登录能力的方法目录。每个方法的签名、参数、返回和使用边界都放在独立页面中。

## 方法索引

| 方法                                                                                                | 说明                                                                                                |
| --------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------- |
| [`GetSdkVersion`](/docs/chat/sdk/v4/android/application/methods/get-sdk-version)                    | 获取当前 core SDK 的版本号，适合在启动日志、问题反馈和兼容性判断中记录。                            |
| [`InitSDK`](/docs/chat/sdk/v4/android/application/methods/init-sdk)                                 | 初始化 SDK 全局实例、日志、长连接管理、本地数据目录和基础资源。应用进程内通常只调用一次。           |
| [`UnInitSDK`](/docs/chat/sdk/v4/android/application/methods/un-init-sdk)                            | 释放当前 SDK 全局实例，适合应用完全退出 IM 能力或测试环境重建实例时使用。                           |
| [`Login`](/docs/chat/sdk/v4/android/application/methods/login)                                      | 使用业务服务端签发的 token 登录指定用户。登录成功后才能可靠调用会话、消息、群组、好友等用户态接口。 |
| [`Logout`](/docs/chat/sdk/v4/android/application/methods/logout)                                    | 退出当前登录用户，断开当前用户态会话并清理登录相关状态。它不等同于销毁 SDK 全局实例。               |
| [`SetAppBackgroundStatus`](/docs/chat/sdk/v4/android/application/methods/set-app-background-status) | 通知 SDK 应用进入前台或后台，便于 SDK 调整连接、同步和推送相关行为。                                |
| [`NetworkStatusChanged`](/docs/chat/sdk/v4/android/application/methods/network-status-changed)      | 通知 SDK 网络状态已变化，当前实现会触发长连接关闭并进入后续重连处理。                               |
| [`GetLoginStatus`](/docs/chat/sdk/v4/android/application/methods/get-login-status)                  | 查询 SDK 当前登录状态，用于业务层判断是否可以继续调用用户态 API。                                   |
| [`GetLoginUserID`](/docs/chat/sdk/v4/android/application/methods/get-login-user-id)                 | 获取当前 SDK 实例记录的登录用户 ID，常用于日志、会话归属和账号切换判断。                            |
