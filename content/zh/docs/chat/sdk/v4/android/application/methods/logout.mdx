`Logout` 来自 `open_im_sdk/init_login.go`。本页只说明当前 Android core binding 暴露的真实方法，不包含示例代码。

## 方法签名

| 项目     | 说明                                        |
| -------- | ------------------------------------------- |
| 方法     | `Logout(callback Base, operationID string)` |
| 所属能力 | 初始化与登录                                |
| 源码位置 | `open_im_sdk/init_login.go`                 |

## 作用

退出当前登录用户，断开当前用户态会话并清理登录相关状态。它不等同于销毁 SDK 全局实例。

## 参数说明

`operationID` 用于串联退出登录日志。

## 返回与事件

通过 `Base` 回调返回退出结果。

## 使用边界

初始化类方法应在应用进程内按需调用；用户态方法通常要求 `InitSDK` 已成功执行，并通过 `operationID` 串联日志。
