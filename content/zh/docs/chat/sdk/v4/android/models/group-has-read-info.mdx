`GroupHasReadInfo` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.GroupHasReadInfo`。表示一条群消息在某次查询或回调中的已读、未读人数快照。

## 字段说明

| 字段名称     | Java 类型 | 描述                                                                       |
| ------------ | --------- | -------------------------------------------------------------------------- |
| hasReadCount | int       | 已对目标群消息产生有效已读记录的成员数量。                                 |
| unreadCount  | int       | 目标群消息当前仍未读的成员数量；成员变动或新回执到达后，该值可能继续变化。 |

该类型只有人数统计，不包含具体成员列表；需要成员明细时应读取 [`GroupMessageReadInfo`](/docs/chat/sdk/v4/android/models/group-message-read-info) 中的已读成员字段。

## 关联类型

- [群消息已读信息](/docs/chat/sdk/v4/android/models/group-message-read-info)（`GroupMessageReadInfo`）：按消息 ID 提供群消息已读统计和成员明细。
