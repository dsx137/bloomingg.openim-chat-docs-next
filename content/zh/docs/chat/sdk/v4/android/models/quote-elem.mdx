`QuoteElem` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.QuoteElem`。引用消息元素。

## 字段说明

| 字段名称     | Java 类型                                             | 描述                                            |
| ------------ | ----------------------------------------------------- | ----------------------------------------------- |
| text         | String                                                | 当前回复消息的正文。                            |
| quoteMessage | [`Message`](/docs/chat/sdk/v4/android/models/message) | 被回复消息的快照；core 会避免继续嵌套引用消息。 |

## 关联类型

- [消息对象](/docs/chat/sdk/v4/android/models/message)（`Message`）：消息对象，包含消息 ID、发送者、会话、内容元素和状态等信息。
