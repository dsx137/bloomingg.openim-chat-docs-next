`Platform` 是 Android SDK 的枚举常量，完整类名为 `io.openim.android.sdk.enums.Platform`。客户端平台类型常量。

## 取值说明

| 常量        | 值  | 描述                                                                    |
| ----------- | --- | ----------------------------------------------------------------------- |
| IOS         | 1   | iPhone 等 iOS 客户端。                                                  |
| ANDROID     | 2   | Android 手机客户端；Android SDK 的 `InitConfig.platformID` 固定为此值。 |
| WIN         | 3   | Windows 桌面客户端。                                                    |
| XOS         | 4   | macOS（历史命名为 OSX）桌面客户端。                                     |
| WEB         | 5   | Web 浏览器客户端。                                                      |
| MINI_WEB    | 6   | 小程序等轻量 Web 运行环境。                                             |
| LINUX       | 7   | Linux 桌面或原生客户端。                                                |
| ANDROID_PAD | 8   | Android 平板客户端。                                                    |
| IPAD        | 9   | iPad 客户端。                                                           |
