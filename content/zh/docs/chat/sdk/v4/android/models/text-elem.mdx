`TextElem` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.TextElem`。文本消息元素。

## 字段说明

| 字段名称 | Java 类型 | 描述                                             |
| -------- | --------- | ------------------------------------------------ |
| content  | String    | 文本消息正文，创建消息时传入，接收端按原文读取。 |
