`OnMsgSendCallback` 是 Android SDK 的回调接口，完整类名为 `io.openim.android.sdk.listener.OnMsgSendCallback`。发送消息时使用的进度、成功和失败回调。

## 类型说明

消息发送监听

## 回调方法

| 方法       | 参数                   | 描述     |
| ---------- | ---------------------- | -------- |
| onError    | int code；String error | 发送失败 |
| onProgress | long progress          | 上传进度 |
| onSuccess  | Message s              | 发送成功 |

## 关联类型

- [通用回调](/docs/chat/sdk/v4/android/models/on-base)（`OnBase`）：Android SDK 异步方法的通用成功和失败回调。
- [消息对象](/docs/chat/sdk/v4/android/models/message)（`Message`）：消息对象，包含消息 ID、发送者、会话、内容元素和状态等信息。
