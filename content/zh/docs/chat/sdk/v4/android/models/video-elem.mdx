`VideoElem` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.VideoElem`。视频消息元素，同时保存发送前的本地文件信息和上传后的远程资源信息。

## 字段说明

| 字段名称       | Java 类型 | 描述                                                                            |
| -------------- | --------- | ------------------------------------------------------------------------------- |
| videoPath      | String    | 发送前视频文件的本地路径；使用 URL 创建消息时可以为空，接收端也不应依赖该路径。 |
| videoUUID      | String    | 视频资源标识，用于上传、缓存或资源去重；通过 URL 创建消息时由调用方提供。       |
| videoUrl       | String    | 上传完成后的远程视频 URL；本地创建消息后由发送流程补齐。                        |
| videoType      | String    | 视频 MIME 类型或业务约定的格式字符串，例如 `video/mp4`。                        |
| videoSize      | long      | 视频文件大小，单位为字节。通过本地文件创建时由 core 读取文件属性。              |
| duration       | long      | 视频时长，单位为秒，由调用方在创建消息时传入。                                  |
| snapshotPath   | String    | 视频封面图的本地路径；没有封面时为空。                                          |
| snapshotUUID   | String    | 视频封面资源标识，用于上传、缓存或资源去重。                                    |
| snapshotSize   | long      | 视频封面文件大小，单位为字节。                                                  |
| snapshotUrl    | String    | 上传完成后的远程封面图 URL。                                                    |
| snapshotWidth  | int       | 封面图像素宽度。通过本地封面创建时由 core 读取。                                |
| snapshotHeight | int       | 封面图像素高度。通过本地封面创建时由 core 读取。                                |
