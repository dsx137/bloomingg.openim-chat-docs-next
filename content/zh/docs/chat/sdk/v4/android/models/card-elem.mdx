`CardElem` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.CardElem`。个人名片消息元素。

## 字段说明

| 字段名称 | Java 类型 | 描述                                         |
| -------- | --------- | -------------------------------------------- |
| userID   | String    | 名片所代表用户的 ID。                        |
| nickname | String    | 发送名片时写入的用户昵称快照。               |
| faceURL  | String    | 发送名片时写入的用户头像 URL 快照。          |
| ex       | String    | 名片业务扩展数据，格式由发送方和接收方约定。 |
