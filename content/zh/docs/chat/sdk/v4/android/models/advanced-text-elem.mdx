`AdvancedTextElem` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.AdvancedTextElem`。高级文本消息的内容元素，用于描述富文本正文和片段。

## 字段说明

| 字段名称          | Java 类型                                                                    | 描述                                                                         |
| ----------------- | ---------------------------------------------------------------------------- | ---------------------------------------------------------------------------- |
| text              | String                                                                       | 完整的高级文本正文；`messageEntityList` 中的偏移量和长度都以该字符串为基准。 |
| messageEntityList | `List<`[`MessageEntity`](/docs/chat/sdk/v4/android/models/message-entity)`>` | 正文中的富文本片段列表，每项描述片段类型、位置、长度、URL 和扩展数据。       |

## 关联类型

- [高级消息片段](/docs/chat/sdk/v4/android/models/message-entity)（`MessageEntity`）：富文本消息中的片段信息。
