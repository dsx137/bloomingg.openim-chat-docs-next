`C2CReadReceiptInfo` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.C2CReadReceiptInfo`。单聊消息已读回执信息。

## 字段说明

| 字段名称    | Java 类型      | 描述                                                                                                                                                            |
| ----------- | -------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| userID      | String         | 执行已读操作的用户 ID。收到对端已读回执时，可用它确认是哪位用户读取了消息。                                                                                     |
| groupID     | String         | 群 ID 兼容字段；单聊已读回执通常为空。                                                                                                                          |
| msgIDList   | `List<String>` | 已被该用户读取的客户端消息 ID 列表，对应 [`Message.clientMsgID`](/docs/chat/sdk/v4/android/models/message)。                                                    |
| readTime    | long           | 已读时间，当前 core 使用 Unix 毫秒时间戳。                                                                                                                      |
| msgFrom     | int            | 原消息来源：`100` 表示用户消息，`200` 表示系统消息。                                                                                                            |
| contentType | int            | 原消息内容类型，取值见 [`MessageType`](/docs/chat/sdk/v4/android/models/message-type)。不要只按 Android 类注释中列出的 `101–113` 判断，后续消息类型也可能出现。 |
| sessionType | int            | 原消息所属会话类型，取值见 [`ConversationType`](/docs/chat/sdk/v4/android/models/conversation-type)。                                                           |

## 关联类型

- [会话类型](/docs/chat/sdk/v4/android/models/conversation-type)（`ConversationType`）：会话类型常量，用于区分单聊、群聊和通知会话。
- [消息类型](/docs/chat/sdk/v4/android/models/message-type)（`MessageType`）：消息内容类型常量。
