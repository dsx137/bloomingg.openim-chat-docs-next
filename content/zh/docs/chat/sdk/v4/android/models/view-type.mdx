`ViewType` 是 Android SDK 的枚举常量，完整类名为 `io.openim.android.sdk.enums.ViewType`。消息列表查询视图类型。

## 取值说明

| 常量    | 值  | 描述                                                 |
| ------- | --- | ---------------------------------------------------- |
| History | 0   | 普通聊天记录浏览，按历史消息分页语义处理结果和游标。 |
| Search  | 1   | 搜索结果浏览，允许查询链路按搜索视图处理消息定位。   |

Android Manager 会调用 `getValue()` 传入底层 core。不要使用 `ordinal()` 代替公开数值，避免枚举顺序变化后产生错误参数。
