`GetFriendApplicationListAsRecipientReq` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.GetFriendApplicationListAsRecipientReq`。查询当前用户收到的好友申请时使用的分页和状态过滤条件。

## 字段说明

| 字段名称      | Java 类型 | 描述                                                                                                              |
| ------------- | --------- | ----------------------------------------------------------------------------------------------------------------- |
| offset        | int       | 从第几条好友申请开始返回，首条记录的偏移量为 `0`。                                                                |
| count         | int       | 本页最多返回的好友申请数量。                                                                                      |
| handleResults | int[]     | 只返回指定处理状态的申请。当前 Go core 使用 `-1` 表示拒绝、`0` 表示未处理、`1` 表示同意；空数组表示不按状态过滤。 |
