`GroupVerification` 是 Android SDK 的枚举常量，完整类名为 `io.openim.android.sdk.enums.GroupVerification`。入群验证方式常量。

## 取值说明

| 常量                                    | 值  | 描述                                                                             |
| --------------------------------------- | --- | -------------------------------------------------------------------------------- |
| APPLY_NEED_VERIFICATION_INVITE_DIRECTLY | 0   | 用户主动申请需要群主或管理员同意；由有权限成员发出的邀请可直接入群。             |
| ALL_NEED_VERIFICATION                   | 1   | 主动申请和普通成员邀请都需要验证；群主或管理员邀请的处理仍以服务端权限策略为准。 |
| DIRECTLY                                | 2   | 主动申请或邀请无需审核即可加入群组。                                             |
