`PublicUserInfo` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.PublicUserInfo`。公开用户资料。

## 字段说明

| 字段名称   | Java 类型 | 描述                                                       |
| ---------- | --------- | ---------------------------------------------------------- |
| userID     | String    | 用户唯一 ID。                                              |
| nickname   | String    | 用户公开昵称。                                             |
| faceURL    | String    | 用户公开头像 URL。                                         |
| ex         | String    | 用户公开业务扩展字符串；业务在解析前应确认双方约定的格式。 |
| createTime | Long      | 用户资料创建时间，当前 core 使用 Unix 毫秒时间戳。         |

该类型只承载客户端可见的公开资料，不包含好友备注、黑名单状态或管理员侧私有字段。
