`UserExInfo` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.UserExInfo`。用户资料中的扩展信息。

## 字段说明

| 字段名称 | Java 类型 | 描述                                                                             |
| -------- | --------- | -------------------------------------------------------------------------------- |
| userID   | String    | 与这条用户级扩展信息关联的用户 ID。                                              |
| ex       | String    | 该用户在当前消息扩展项中的业务扩展内容；格式由写入方约定，SDK 不解析其内部结构。 |
