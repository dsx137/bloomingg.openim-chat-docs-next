`Message` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.Message`。消息对象，包含消息 ID、发送者、会话、内容元素和状态等信息。

## 字段说明

| 字段名称             | Java 类型        | 描述                                                                                                                                                                                                                                |
| -------------------- | ---------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| clientMsgID          | String           | 消息唯一ID                                                                                                                                                                                                                          |
| serverMsgID          | String           | 消息服务器ID，暂时不使用                                                                                                                                                                                                            |
| createTime           | long             | 消息创建时间，单位纳秒                                                                                                                                                                                                              |
| sendTime             | long             | 消息发送时间，单位ms                                                                                                                                                                                                                |
| sessionType          | int              | 会话类型 1:单聊 2:群聊 ConversationType                                                                                                                                                                                             |
| sendID               | String           | 发送者ID                                                                                                                                                                                                                            |
| recvID               | String           | 接收者ID                                                                                                                                                                                                                            |
| msgFrom              | int              | 标识消息是用户级别还是系统级别 100:用户 200:系统                                                                                                                                                                                    |
| contentType          | int              | MessageType 消息类型：； 101:文本消息； 102:图片消息； 103:语音消息； 104:视频消息； 105:文件消息； 106:@消息； 107:合并消息； 108:转发消息； 109:位置消息； 110:自定义消息； 111:撤回消息回执； 112:C2C已读回执； 113:正在输入状态 |
| platformID           | int              | Platform 平台类型 1:ios 2:android 3:windows 4:osx 5:web 6:mini 7:linux                                                                                                                                                              |
| senderNickname       | String           | 发送者昵称                                                                                                                                                                                                                          |
| senderFaceUrl        | String           | 发送者头像                                                                                                                                                                                                                          |
| groupID              | String           | 群聊ID                                                                                                                                                                                                                              |
| seq                  | int              | 消息唯一序列号                                                                                                                                                                                                                      |
| isRead               | boolean          | 是否已读                                                                                                                                                                                                                            |
| status               | int              | MessageStatus 消息状态 1:发送中 2:发送成功 3:发送失败 4:已删除 5:已撤回                                                                                                                                                             |
| isReact              | boolean          | SDK 返回或提交的字段。                                                                                                                                                                                                              |
| isExternalExtensions | boolean          | SDK 返回或提交的字段。                                                                                                                                                                                                              |
| offlinePush          | OfflinePushInfo  | 离线消息推送内容                                                                                                                                                                                                                    |
| attachedInfo         | String           | 附加信息                                                                                                                                                                                                                            |
| ext                  | Object           | android 附加字段                                                                                                                                                                                                                    |
| ex                   | Object           | go 附加字段                                                                                                                                                                                                                         |
| localEx              | Object           | 本地消息拓展信息                                                                                                                                                                                                                    |
| pictureElem          | PictureElem      | 图片信息                                                                                                                                                                                                                            |
| soundElem            | SoundElem        | 语音信息                                                                                                                                                                                                                            |
| videoElem            | VideoElem        | 视频信息                                                                                                                                                                                                                            |
| fileElem             | FileElem         | 文件信息                                                                                                                                                                                                                            |
| atTextElem           | AtTextElem       | \_@信息                                                                                                                                                                                                                             |
| locationElem         | LocationElem     | 位置信息                                                                                                                                                                                                                            |
| customElem           | CustomElem       | 自定义信息                                                                                                                                                                                                                          |
| quoteElem            | QuoteElem        | 引用消息                                                                                                                                                                                                                            |
| mergeElem            | MergeElem        | 合并信息                                                                                                                                                                                                                            |
| notificationElem     | NotificationElem | 通知                                                                                                                                                                                                                                |
| faceElem             | FaceElem         | 自定义表情                                                                                                                                                                                                                          |
| attachedInfoElem     | AttachedInfoElem | 附加信息 如：群消息已读                                                                                                                                                                                                             |
| textElem             | TextElem         | SDK 返回或提交的字段。                                                                                                                                                                                                              |
| cardElem             | CardElem         | 个人名片                                                                                                                                                                                                                            |
| advancedTextElem     | AdvancedTextElem | SDK 返回或提交的字段。                                                                                                                                                                                                              |
| typingElem           | TypingElem       | SDK 返回或提交的字段。                                                                                                                                                                                                              |
