`RichMessage` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.RichMessage`。高级文本消息中的富文本片段。

## 字段说明

| 字段名称 | Java 类型 | 描述                                                                                       |
| -------- | --------- | ------------------------------------------------------------------------------------------ |
| type     | String    | 富文本片段类型，例如链接、提及或其他业务自定义类型；SDK 原样保存，不限定取值集合。         |
| offset   | int       | 片段在完整 `text` 中的起始偏移量，不是分页偏移量；发送端和渲染端须使用相同的字符计数规则。 |
| length   | int       | 从 `offset` 开始覆盖的文本长度。                                                           |
| url      | String    | 片段关联的 URL；非链接类片段可留空。                                                       |
| info     | String    | 片段的业务扩展数据，用于保存提及用户、样式或其他渲染信息，格式由业务双方约定。             |
