`JoinSource` 是 Android SDK 的枚举常量，完整类名为 `io.openim.android.sdk.enums.JoinSource`。入群来源枚举。

## 取值说明

| 常量       | 值  | 描述                                                                        |
| ---------- | --- | --------------------------------------------------------------------------- |
| Invitation | 2   | 由现有群成员邀请加入。申请或成员记录中的 `inviterUserID` 通常会保存邀请人。 |
| Search     | 3   | 用户通过群 ID、群名称等搜索入口主动申请或加入。                             |
| QrCode     | 4   | 用户通过群二维码入口申请或加入。                                            |

调用加入群组方法时应传枚举的数值，不要使用数组下标或 Java `ordinal()`；后者分别是 `0/1/2`，与协议值不同。
