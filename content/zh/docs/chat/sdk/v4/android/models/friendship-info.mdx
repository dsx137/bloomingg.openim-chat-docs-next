`FriendshipInfo` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.FriendshipInfo`。好友关系校验结果。

## 字段说明

| 字段名称 | Java 类型 | 描述                                                                                                                  |
| -------- | --------- | --------------------------------------------------------------------------------------------------------------------- |
| userID   | String    | 本条校验结果对应的目标用户 ID。                                                                                       |
| result   | int       | 关系结果：`1` 表示双方是好友且目标用户不在当前用户黑名单中，`0` 表示条件不成立；仅凭 `0` 不能区分“非好友”和“已拉黑”。 |
