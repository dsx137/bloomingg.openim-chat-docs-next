`SingleTypeKeyInfoSum` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.SingleTypeKeyInfoSum`。消息扩展中单个类型和 key 的统计信息。

## 字段说明

| 字段名称      | Java 类型        | 描述                   |
| ------------- | ---------------- | ---------------------- |
| typeKey       | String           | SDK 返回或提交的字段。 |
| counter       | int              | SDK 返回或提交的字段。 |
| infoList      | List<UserExInfo> | SDK 返回或提交的字段。 |
| isContainSelf | boolean          | SDK 返回或提交的字段。 |

## 关联类型
