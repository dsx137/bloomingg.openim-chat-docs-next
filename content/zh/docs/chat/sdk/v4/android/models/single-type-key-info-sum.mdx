`SingleTypeKeyInfoSum` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.SingleTypeKeyInfoSum`。消息扩展中单个类型和 key 的统计信息。

## 字段说明

| 字段名称      | Java 类型                                                               | 描述                                                                                   |
| ------------- | ----------------------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| typeKey       | String                                                                  | 消息扩展的类型 key，用于区分不同的聚合项，例如不同表态或业务自定义类型。               |
| counter       | int                                                                     | 当前 `typeKey` 对应的累计数量。                                                        |
| infoList      | `List<`[`UserExInfo`](/docs/chat/sdk/v4/android/models/user-ex-info)`>` | 参与该 `typeKey` 的用户及用户级扩展数据列表。                                          |
| isContainSelf | boolean                                                                 | 当前登录用户是否包含在 `infoList` 所代表的参与者中；可直接用于渲染当前用户的选中状态。 |

## 关联类型

- [用户扩展信息](/docs/chat/sdk/v4/android/models/user-ex-info)（`UserExInfo`）：用户资料中的扩展信息。
