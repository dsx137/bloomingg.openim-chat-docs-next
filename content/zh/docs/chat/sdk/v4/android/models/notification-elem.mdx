`NotificationElem` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.NotificationElem`。通知消息元素。

## 字段说明

| 字段名称    | Java 类型 | 描述                                                                                                                 |
| ----------- | --------- | -------------------------------------------------------------------------------------------------------------------- |
| detail      | String    | 通知的结构化或自定义详情。自定义业务通知会把该字段原样传给 `onRecvCustomBusinessMessage`；内部格式由通知发送方约定。 |
| defaultTips | String    | 无需解析 `detail` 时可直接展示的默认提示文本；是否展示及本地化方式由业务层决定。                                     |
