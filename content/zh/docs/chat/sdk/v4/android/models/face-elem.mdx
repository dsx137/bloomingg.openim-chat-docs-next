`FaceElem` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.FaceElem`。自定义表情消息元素。

## 字段说明

| 字段名称 | Java 类型 | 描述                                    |
| -------- | --------- | --------------------------------------- |
| index    | int       | app内嵌表情包，根据index 匹配显示表情图 |
| data     | String    | 其他表情，如url表情，直接加载url即可    |
