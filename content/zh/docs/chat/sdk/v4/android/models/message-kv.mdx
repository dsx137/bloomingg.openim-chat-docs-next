`MessageKv` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.MessageKv`。消息扩展键值信息。

## 字段说明

| 字段名称      | Java 类型                                                                                     | 描述                                                                                           |
| ------------- | --------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------- |
| clientMsgID   | String                                                                                        | 发生 KV 聚合信息变化的消息客户端 ID，用于定位本地消息。                                        |
| changedKvList | `List<`[`SingleTypeKeyInfoSum`](/docs/chat/sdk/v4/android/models/single-type-key-info-sum)`>` | 该消息发生变化后的 KV 聚合结果；每项对应一个 `typeKey`，包含计数、用户明细及当前用户是否参与。 |

## 关联类型

- [单类型扩展统计](/docs/chat/sdk/v4/android/models/single-type-key-info-sum)（`SingleTypeKeyInfoSum`）：消息扩展中单个类型和 key 的统计信息。
