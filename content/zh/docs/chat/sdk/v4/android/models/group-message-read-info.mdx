`GroupMessageReadInfo` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.GroupMessageReadInfo`。群消息已读与未读统计信息。

## 字段说明

| 字段名称     | Java 类型                                                                           | 描述                                                                                   |
| ------------ | ----------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| clientMsgID  | String                                                                              | 已读统计所属消息的客户端消息 ID。                                                      |
| hasReadCount | int                                                                                 | 当前已经阅读该消息的群成员数量。                                                       |
| unreadCount  | int                                                                                 | 当前尚未阅读该消息的群成员数量。                                                       |
| readMembers  | `List<`[`GroupMembersInfo`](/docs/chat/sdk/v4/android/models/group-members-info)`>` | 本次回执中可返回的已读群成员资料；可使用 `userID` 标识成员，列表内容以服务端回执为准。 |

## 关联类型

- [群成员信息](/docs/chat/sdk/v4/android/models/group-members-info)（`GroupMembersInfo`）：群成员资料，包含群内昵称、角色、入群来源和禁言时间。
