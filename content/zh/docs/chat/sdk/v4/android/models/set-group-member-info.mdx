`SetGroupMemberInfo` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.SetGroupMemberInfo`。设置群成员资料时使用的请求对象。

## 字段说明

| 字段名称  | Java 类型 | 描述                                                                                                                                                                           |
| --------- | --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| groupID   | String    | 要修改成员所属的群 ID，必填。                                                                                                                                                  |
| userID    | String    | 要修改的群成员用户 ID，必填。                                                                                                                                                  |
| nickname  | String    | 新的群内昵称；为 `null` 时不修改，传空字符串表示清空。                                                                                                                         |
| faceURL   | String    | 新的群成员头像 URL；为 `null` 时不修改。                                                                                                                                       |
| roleLevel | Integer   | 新的群内角色，取值见 [`GroupRole`](/docs/chat/sdk/v4/android/models/group-role)：`20` 普通成员、`60` 管理员、`100` 群主。Android 源码注释中的 `1/2/3` 与当前 core 常量不一致。 |
| ex        | String    | 群成员级业务扩展字符串；为 `null` 时不修改。                                                                                                                                   |

Android 使用 `toStringWithoutNull` 序列化此对象，因此可选字段保持 `null` 才表示“不更新”；权限及角色变更是否合法由服务端校验。

## 关联类型

- [群成员角色](/docs/chat/sdk/v4/android/models/group-role)（`GroupRole`）：群主、管理员和普通成员的角色值。
