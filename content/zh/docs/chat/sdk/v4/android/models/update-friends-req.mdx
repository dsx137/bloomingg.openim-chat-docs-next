`UpdateFriendsReq` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.UpdateFriendsReq`。批量更新好友备注或扩展信息的请求对象。

## 字段说明

| 字段名称      | Java 类型 | 描述           |
| ------------- | --------- | -------------- |
| ownerUserID   | String    | 操作者Uid      |
| friendUserIDs | String[]  | 待更新的用户id |
| isPinned      | Boolean   | 是否置顶       |
| remark        | String    | 备注名         |
| ex            | String    | 扩展参数       |
