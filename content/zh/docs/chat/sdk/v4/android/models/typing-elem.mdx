`TypingElem` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.TypingElem`。正在输入状态消息元素。

## 字段说明

| 字段名称 | Java 类型 | 描述                                                                                                                      |
| -------- | --------- | ------------------------------------------------------------------------------------------------------------------------- |
| msgTips  | String    | 输入状态提示值。`changeInputStates` 使用 `"yes"` 表示开始输入、`"no"` 表示停止输入；`typingStatusUpdate` 原样发送传入值。 |
