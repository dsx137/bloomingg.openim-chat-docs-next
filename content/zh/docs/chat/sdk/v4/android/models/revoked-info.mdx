`RevokedInfo` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.RevokedInfo`。消息撤回事件中的撤回详情。

## 字段说明

| 字段名称                    | Java 类型 | 描述                                                                                                      |
| --------------------------- | --------- | --------------------------------------------------------------------------------------------------------- |
| revokerID                   | String    | 执行撤回操作的用户 ID。群管理员撤回他人消息时，该值与原发送者不同。                                       |
| revokerRole                 | int       | 撤回者在群内的角色，取值见 [`GroupRole`](/docs/chat/sdk/v4/android/models/group-role)；单聊时通常为 `0`。 |
| revokerNickname             | String    | core 在触发回调前补齐的撤回者昵称；无法查询到群成员时可能为 `unknown`。                                   |
| clientMsgID                 | String    | 被撤回原消息的客户端消息 ID，可用于在当前消息列表中定位并替换展示。                                       |
| revokeTime                  | long      | 撤回操作时间，当前 core 使用 Unix 毫秒时间戳。                                                            |
| sourceMessageSendTime       | long      | 原消息发送时间，当前 core 使用 Unix 毫秒时间戳。                                                          |
| sourceMessageSendID         | String    | 原消息发送者用户 ID。                                                                                     |
| sourceMessageSenderNickname | String    | 原消息发送者发送时的昵称快照。                                                                            |
| sessionType                 | int       | 原消息所属会话类型，取值见 [`ConversationType`](/docs/chat/sdk/v4/android/models/conversation-type)。     |
