`RevokedInfo` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.RevokedInfo`。消息撤回事件中的撤回详情。

## 字段说明

| 字段名称                    | Java 类型 | 描述                      |
| --------------------------- | --------- | ------------------------- |
| revokerID                   | String    | 撤回者ID                  |
| revokerRole                 | int       | 撤回者群角色 GroupRole    |
| revokerNickname             | String    | 撤回者昵称                |
| clientMsgID                 | String    | 消息id                    |
| revokeTime                  | long      | 撤回时间                  |
| sourceMessageSendTime       | long      | 消息发送时间              |
| sourceMessageSendID         | String    | 消息发送者                |
| sourceMessageSenderNickname | String    | 消息发送者昵称            |
| sessionType                 | int       | 会话类型 ConversationType |

## 关联类型

- [会话类型](/docs/chat/sdk/v4/android/models/conversation-type)（`ConversationType`）：会话类型常量，用于区分单聊、群聊和通知会话。
- [群成员角色](/docs/chat/sdk/v4/android/models/group-role)（`GroupRole`）：群成员角色常量。
