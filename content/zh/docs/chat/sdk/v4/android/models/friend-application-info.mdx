`FriendApplicationInfo` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.FriendApplicationInfo`。好友申请记录，包含申请人、接收人、处理状态和备注。

## 字段说明

| 字段名称      | Java 类型 | 描述                                                                                                   |
| ------------- | --------- | ------------------------------------------------------------------------------------------------------ |
| fromUserID    | String    | 发起好友申请的用户 ID。                                                                                |
| fromNickname  | String    | 申请发起方的昵称快照。                                                                                 |
| fromFaceURL   | String    | 申请发起方的头像 URL 快照。                                                                            |
| fromGender    | int       | 发起方性别兼容字段：`1` 男、`2` 女、`0` 或其他值未设置。当前提供的 core 转换未填充该字段，通常为 `0`。 |
| toUserID      | String    | 好友申请接收方的用户 ID。                                                                              |
| toNickname    | String    | 申请接收方的昵称快照。                                                                                 |
| toFaceURL     | String    | 申请接收方的头像 URL 快照。                                                                            |
| toGender      | int       | 接收方性别兼容字段：`1` 男、`2` 女、`0` 或其他值未设置。当前提供的 core 转换未填充该字段，通常为 `0`。 |
| handleResult  | int       | 处理状态：`-1` 已拒绝、`0` 未处理、`1` 已同意。Android 类注释把拒绝写成 `2`，与当前 core 常量不一致。  |
| reqMsg        | String    | 申请发起方填写的验证信息或附言。                                                                       |
| createTime    | long      | 申请创建时间，当前 core 使用 Unix 毫秒时间戳。                                                         |
| handlerUserID | String    | 执行同意或拒绝操作的用户 ID，通常是申请接收方；未处理时为空。                                          |
| handleMsg     | String    | 处理申请时填写的回复说明；未处理时通常为空。                                                           |
| handleTime    | long      | 申请处理时间，当前 core 使用 Unix 毫秒时间戳；未处理时通常为 `0`。                                     |
| ex            | String    | 好友申请记录的业务扩展字符串。                                                                         |
