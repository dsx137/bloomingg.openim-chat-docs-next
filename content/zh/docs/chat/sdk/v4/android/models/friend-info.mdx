`FriendInfo` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.FriendInfo`。好友资料，包含昵称、头像、备注和添加来源等信息。

## 类型说明

好友信息

## 字段说明

| 字段名称       | Java 类型 | 描述       |
| -------------- | --------- | ---------- |
| userID         | String    | 好友id     |
| nickname       | String    | 好友昵称   |
| faceURL        | String    | 好友头像   |
| gender         | int       | 性别       |
| phoneNumber    | String    | 手机号     |
| birth          | long      | 出生日期   |
| email          | String    | 邮箱       |
| remark         | String    | 好友备注名 |
| ex             | String    | 扩展字段   |
| createTime     | long      | 创建时间   |
| addSource      | int       | 添加方式   |
| operatorUserID | String    | 操作者id   |
