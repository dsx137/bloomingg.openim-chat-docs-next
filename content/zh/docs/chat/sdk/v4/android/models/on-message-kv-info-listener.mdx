`OnMessageKvInfoListener` 是 Android SDK 的监听器，完整类名为 `io.openim.android.sdk.listener.OnMessageKvInfoListener`。消息扩展键值变化监听器。

## 回调方法

| 方法                   | 参数                 | 描述             |
| ---------------------- | -------------------- | ---------------- |
| onMessageKvInfoChanged | List<MessageKv> list | 事件触发时回调。 |

## 关联类型

- [消息扩展键值](/docs/chat/sdk/v4/android/models/message-kv)（`MessageKv`）：消息扩展键值信息。
