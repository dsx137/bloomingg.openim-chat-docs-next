`UsersOnlineStatus` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.UsersOnlineStatus`。用户在线状态信息。

## 字段说明

| 字段名称    | Java 类型       | 描述                                                                                                                           |
| ----------- | --------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| userID      | String          | 在线状态所属的用户 ID。                                                                                                        |
| status      | int             | 汇总在线状态：`0` 表示离线，`1` 表示在线。core 根据 `platformIDs` 是否为空生成该值。                                           |
| platformIDs | `List<Integer>` | 用户当前在线的平台 ID 列表，取值对应 [`Platform`](/docs/chat/sdk/v4/android/models/platform)；空列表表示用户在所有平台均离线。 |
