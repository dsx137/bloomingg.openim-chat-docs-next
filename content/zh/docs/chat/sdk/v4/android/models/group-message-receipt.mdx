`GroupMessageReceipt` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.GroupMessageReceipt`。群消息已读回执。

## 字段说明

| 字段名称             | Java 类型                                                                                    | 描述                                                                                              |
| -------------------- | -------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------- |
| conversationID       | String                                                                                       | 已读回执所属的群会话 ID。一个回调对象只包含同一会话内的消息。                                     |
| groupMessageReadInfo | `List<`[`GroupMessageReadInfo`](/docs/chat/sdk/v4/android/models/group-message-read-info)`>` | 本次已读数据发生变化的消息列表；每项通过 `clientMsgID` 标识消息，并给出最新已读、未读统计和成员。 |

## 关联类型

- [群消息已读信息](/docs/chat/sdk/v4/android/models/group-message-read-info)（`GroupMessageReadInfo`）：群消息已读与未读统计信息。
