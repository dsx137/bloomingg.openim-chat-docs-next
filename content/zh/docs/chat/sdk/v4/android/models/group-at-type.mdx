`GroupAtType` 是 Android SDK 的枚举常量，完整类名为 `io.openim.android.sdk.enums.GroupAtType`。群会话强提醒类型常量。

## 取值说明

| 常量               | 值  | 描述                                                                        |
| ------------------ | --- | --------------------------------------------------------------------------- |
| AT_NORMAL          | 0   | 当前会话没有待展示的 @ 或群公告强提醒，通常在用户读取相关消息后恢复为该值。 |
| AT_ME              | 1   | 存在明确提及当前用户的未读 @ 消息。                                         |
| AT_ALL             | 2   | 存在 @ 所有人的未读消息，但没有单独 @ 当前用户。                            |
| AT_ALL_AT_ME       | 3   | 同时存在 @ 所有人和单独 @ 当前用户的未读提醒。                              |
| GROUP_NOTIFICATION | 4   | 存在需要在会话列表提示的群公告更新。                                        |
