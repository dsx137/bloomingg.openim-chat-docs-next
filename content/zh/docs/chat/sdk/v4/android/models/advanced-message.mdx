`AdvancedMessage` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.AdvancedMessage`。历史消息分页查询返回的结果对象，包含错误信息、分页游标和消息列表。

## 字段说明

| 字段名称    | Java 类型                                                       | 描述                                                                                       |
| ----------- | --------------------------------------------------------------- | ------------------------------------------------------------------------------------------ |
| errCode     | int                                                             | 本次历史消息查询的错误码；`0` 表示查询成功。                                               |
| errMsg      | String                                                          | 查询失败时的错误信息；成功时通常为空。                                                     |
| lastMinSeq  | int                                                             | 本页结果覆盖到的最小消息序列号，可作为继续向更早消息分页时的边界游标。                     |
| isEnd       | boolean                                                         | 是否已经到达当前查询方向的历史消息末端；为 `true` 时继续使用相同方向查询不会得到更早结果。 |
| messageList | `List<`[`Message`](/docs/chat/sdk/v4/android/models/message)`>` | 本页返回的消息对象列表；空列表表示当前分页范围内没有可返回的消息。                         |

## 关联类型

- [消息对象](/docs/chat/sdk/v4/android/models/message)（`Message`）：消息对象，包含消息 ID、发送者、会话、内容元素和状态等信息。
