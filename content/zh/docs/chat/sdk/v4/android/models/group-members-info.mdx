`GroupMembersInfo` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.GroupMembersInfo`。群成员资料，包含群内昵称、角色、入群来源和禁言时间。

## 字段说明

| 字段名称       | Java 类型 | 描述                                               |
| -------------- | --------- | -------------------------------------------------- |
| groupID        | String    | 群id                                               |
| userID         | String    | 用户id                                             |
| nickname       | String    | 群内昵称                                           |
| faceURL        | String    | 头像                                               |
| roleLevel      | int       | 群角色                                             |
| joinTime       | long      | 入群时间                                           |
| joinSource     | int       | 加入方式 2：邀请加入 3：搜索加入 4：通过二维码加入 |
| operatorUserID | String    | 操作者id                                           |
| ex             | String    | 扩展字段                                           |
| muteEndTime    | long      | 禁言结束时间                                       |
| inviterUserID  | String    | 邀请人id                                           |
