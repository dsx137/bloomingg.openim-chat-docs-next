`PictureInfo` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.PictureInfo`。图片资源信息，包含 UUID、URL、尺寸和大小。

## 字段说明

| 字段名称 | Java 类型 | 描述                                            |
| -------- | --------- | ----------------------------------------------- |
| uuid     | String    | 图片资源标识，用于上传、缓存或资源去重。        |
| type     | String    | 图片 MIME 类型或格式字符串，例如 `image/jpeg`。 |
| size     | long      | 图片文件大小，单位为字节。                      |
| width    | int       | 图片像素宽度。                                  |
| height   | int       | 图片像素高度。                                  |
| url      | String    | 图片上传后的远程 URL；不限定为某一家 OSS 服务。 |
