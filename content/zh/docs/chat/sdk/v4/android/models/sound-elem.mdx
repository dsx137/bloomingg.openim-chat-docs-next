`SoundElem` 是 Android SDK 的数据模型，完整类名为 `io.openim.android.sdk.models.SoundElem`。语音消息元素，包含发送前本地路径和上传后的远程地址。

## 字段说明

| 字段名称  | Java 类型 | 描述                                                                          |
| --------- | --------- | ----------------------------------------------------------------------------- |
| uuid      | String    | 语音资源标识，用于上传、缓存或资源去重；使用 URL 创建消息时由调用方提供。     |
| soundPath | String    | 发送前语音文件的本地路径；使用 URL 创建消息时可以为空，接收端不应依赖该路径。 |
| sourceUrl | String    | 上传完成后的远程语音 URL；本地创建消息后由发送流程补齐。                      |
| dataSize  | long      | 语音文件大小，单位为字节。                                                    |
| duration  | long      | 语音时长，单位为秒，由调用方创建消息时传入。                                  |
