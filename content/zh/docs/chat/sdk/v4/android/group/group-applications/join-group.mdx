申请加入指定群组。是否直接入群取决于群设置和服务端审核策略。

## 功能介绍

申请加入指定群组。是否直接入群取决于群设置和服务端审核策略。

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。

## Core Binding 方法

| 项目     | 说明                                                                                                       |
| -------- | ---------------------------------------------------------------------------------------------------------- |
| 方法签名 | `JoinGroup(callback Base, operationID string, groupID string, reqMsg string, joinSource int32, ex string)` |
| 所属能力 | 群组                                                                                                       |
| 源码位置 | `open_im_sdk/group.go`                                                                                     |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void joinGroup(OnBase<String> base, String gid, String reason, int joinSource)
public void joinGroup(OnBase<String> base, String gid, String reason, int joinSource, String ex)
```

## 输入参数

| 参数名称   | 参数类型                                             | 是否必填 | 描述                                                                  |
| ---------- | ---------------------------------------------------- | -------- | --------------------------------------------------------------------- |
| callBack   | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base) | 是       | 回调接口                                                              |
| gid        | String                                               | 是       | 群ID                                                                  |
| reason     | String                                               | 否       | 请求信息                                                              |
| joinSource | int                                                  | 是       | [`JoinSource`](/docs/chat/sdk/v4/android/models/join-source) 请求来源 |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().groupManager.joinGroup(new OnBase<String>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(String data) {
        // TODO: 处理成功结果
    }
}, gid, reason, joinSource);
```
