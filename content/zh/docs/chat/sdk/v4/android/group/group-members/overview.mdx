群成员用于邀请或移出成员、维护成员资料，以及按条件查询和搜索群成员。

## 方法索引

| 方法                                                                                                          | 说明                                                                     |
| ------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------ |
| [邀请用户入群](/docs/chat/sdk/v4/android/group/group-members/invite-user-to-group)                            | 邀请一批用户加入群组。是否直接加入取决于群设置和服务端策略。             |
| [移出群成员](/docs/chat/sdk/v4/android/group/group-members/kick-group-member)                                 | 将一批成员移出群组，通常要求操作者具备管理权限。                         |
| [更新群成员资料](/docs/chat/sdk/v4/android/group/group-members/set-group-member-info)                         | 更新群成员资料，例如群昵称、角色相关扩展或其他成员字段。                 |
| [获取群成员列表](/docs/chat/sdk/v4/android/group/group-members/get-group-member-list)                         | 按成员类型筛选并分页获取群成员列表。                                     |
| [按入群时间筛选成员](/docs/chat/sdk/v4/android/group/group-members/get-group-member-list-by-join-time-filter) | 按入群时间范围和用户过滤条件获取群成员，适合审计、成员管理和增量查看。   |
| [获取群主和管理员](/docs/chat/sdk/v4/android/group/group-members/get-group-member-owner-and-admin)            | 获取指定群的群主和管理员列表，适合管理页、权限判断和群成员列表顶部展示。 |
| [获取指定群成员资料](/docs/chat/sdk/v4/android/group/group-members/get-specified-group-members-info)          | 获取指定群内指定用户的群成员资料，适合查看成员详情或刷新少量成员信息。   |
| [搜索群成员](/docs/chat/sdk/v4/android/group/group-members/search-group-members)                              | 搜索群成员，适合按用户 ID 或群昵称查找成员。                             |

## 相关类型

- [群成员信息](/docs/chat/sdk/v4/android/models/group-members-info)（`GroupMembersInfo`）：群成员资料对象，包含角色、入群时间和群昵称等信息。
- [群成员设置请求](/docs/chat/sdk/v4/android/models/set-group-member-info)（`SetGroupMemberInfo`）：更新群成员资料时使用的请求对象。
- [群成员筛选类型](/docs/chat/sdk/v4/android/models/group-member-filter)（`GroupMemberFilter`）：获取群成员列表时使用的成员筛选类型。
