更新群成员资料，例如群昵称、角色相关扩展或其他成员字段。

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。

## Core Binding 方法

| 项目     | 说明                                                                            |
| -------- | ------------------------------------------------------------------------------- |
| 方法签名 | `SetGroupMemberInfo(callback Base, operationID string, groupMemberInfo string)` |
| 所属能力 | 群组                                                                            |
| 源码位置 | `open_im_sdk/group.go`                                                          |

## Android 调用参考

Android SDK v3.8.3 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void setGroupMemberInfo(SetGroupMemberInfo setGroupMemberInfo, OnBase<String> base)
```

## 输入参数

| 参数名称           | Java 类型                                                                      | 是否必填 | 描述                     |
| ------------------ | ------------------------------------------------------------------------------ | -------- | ------------------------ |
| setGroupMemberInfo | [`SetGroupMemberInfo`](/docs/chat/sdk/v4/android/models/set-group-member-info) | 是       | 要修改的群成员字段。     |
| base               | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)`<String>`                 | 是       | 群成员资料更新结果回调。 |

## 返回结果

`onSuccess` 表示非空成员字段已通过服务端校验并完成本地成员同步，回调字符串不含业务数据；权限不足、角色值无效或目标成员不存在时调用 `onError`。

## 代码示例

以下示例展示 Android SDK v3.8.3 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().groupManager.setGroupMemberInfo(setGroupMemberInfo, new OnBase<String>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(String data) {
        // TODO: 处理成功结果
    }
});
```
