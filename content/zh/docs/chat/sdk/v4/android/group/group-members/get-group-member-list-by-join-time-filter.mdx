按入群时间范围和用户过滤条件获取群成员，适合审计、成员管理和增量查看。

## 功能介绍

按入群时间范围和用户过滤条件获取群成员，适合审计、成员管理和增量查看。

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。

## Core Binding 方法

| 项目     | 说明                                                                                                                                                                                |
| -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 方法签名 | `GetGroupMemberListByJoinTimeFilter(callback Base, operationID string, groupID string, offset int32, count int32, joinTimeBegin int64, joinTimeEnd int64, filterUserIDList string)` |
| 所属能力 | 群组                                                                                                                                                                                |
| 源码位置 | `open_im_sdk/group.go`                                                                                                                                                              |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void getGroupMemberListByJoinTime(OnBase<List<GroupMembersInfo>> base, String groupID,
                                         int offset, int count, long joinTimeBegin,
                                         long joinTimeEnd, List<String> excludeUserIDList)
```

## 输入参数

| 参数名称          | 参数类型                                                                                                                                          | 是否必填 | 描述                  |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- | -------- | --------------------- |
| callBack          | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)&lt;List&lt;[`GroupMembersInfo`](/docs/chat/sdk/v4/android/models/group-members-info)&gt;&gt; | 是       | 回调接口              |
| groupID           | String                                                                                                                                            | 是       | 群ID                  |
| offset            | int                                                                                                                                               | 是       | 开始下标              |
| count             | int                                                                                                                                               | 是       | 总数                  |
| joinTimeBegin     | int                                                                                                                                               | 是       | 加入开始时间          |
| joinTimeEnd       | int                                                                                                                                               | 是       | 加入结束时间          |
| excludeUserIDList | List<String>                                                                                                                                      | 否       | 剔除不被查询的成员 ID |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().groupManager.getGroupMemberListByJoinTime(
    new OnBase<List<GroupMembersInfo>>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(List<GroupMembersInfo> data) {
        // TODO: 处理群成员列表
    }
},
    groupID,
    offset,
    count,
    joinTimeBegin,
    joinTimeEnd,
    excludeUserIDList
);
```
