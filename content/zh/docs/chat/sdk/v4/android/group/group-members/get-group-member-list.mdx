按成员类型筛选并分页获取群成员列表。

## 功能介绍

按成员类型筛选并分页获取群成员列表。

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。

## Core Binding 方法

| 项目     | 说明                                                                                                             |
| -------- | ---------------------------------------------------------------------------------------------------------------- |
| 方法签名 | `GetGroupMemberList(callback Base, operationID string, groupID string, filter int32, offset int32, count int32)` |
| 所属能力 | 群组                                                                                                             |
| 源码位置 | `open_im_sdk/group.go`                                                                                           |

## Android 调用参考

Android SDK v3.8.3.5 的 Java 方法原型如下，示例代码也基于该调用方式。Core binding 方法签名用于说明底层 `open_im_sdk` 能力和参数映射。

```java showLineNumbers
public void getGroupMemberList(OnBase<List<GroupMembersInfo>> base, String groupId, int filter, int offset, int count)
```

## 输入参数

| 参数名称 | 参数类型                                                                                                                                          | 是否必填 | 描述                                                                                 |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------- | -------- | ------------------------------------------------------------------------------------ |
| callBack | [`OnBase`](/docs/chat/sdk/v4/android/models/on-base)&lt;List&lt;[`GroupMembersInfo`](/docs/chat/sdk/v4/android/models/group-members-info)&gt;&gt; | 是       | 回调接口                                                                             |
| groupId  | String                                                                                                                                            | 是       | 群 ID                                                                                |
| filter   | int                                                                                                                                               | 否       | [`GroupMemberFilter`](/docs/chat/sdk/v4/android/models/group-member-filter) 过滤成员 |
| offset   | int                                                                                                                                               | 是       | 偏移量                                                                               |
| count    | int                                                                                                                                               | 是       | 每页数量                                                                             |

## 返回结果

通过回调返回结果：成功走 `OnSuccess`，失败走 `OnError`；发送或上传类方法还可能通过进度回调返回进度。

## 代码示例

以下示例展示 Android SDK v3.8.3.5 的调用方式。

```java showLineNumbers
OpenIMClient.getInstance().groupManager.getGroupMemberList(
    new OnBase<List<GroupMembersInfo>>() {
    @Override
    public void onError(int code, String error) {
        // TODO: 处理错误
    }

    @Override
    public void onSuccess(List<GroupMembersInfo> data) {
        // TODO: 处理群成员列表
    }
},
    groupId,
    filter,
    offset,
    count
);
```

## 相关类型

- [底层通用回调](/docs/chat/sdk/v4/android/models/base)（`Base`）：core binding 异步结果回调，Android 调用层通常映射为 OnBase。
- [通用回调](/docs/chat/sdk/v4/android/models/on-base)（`OnBase`）：Android SDK 异步方法的通用成功和失败回调。
- [群成员筛选类型](/docs/chat/sdk/v4/android/models/group-member-filter)（`GroupMemberFilter`）：群成员列表筛选类型。
- [群成员信息](/docs/chat/sdk/v4/android/models/group-members-info)（`GroupMembersInfo`）：群成员资料，包含群内昵称、角色、入群来源和禁言时间。
