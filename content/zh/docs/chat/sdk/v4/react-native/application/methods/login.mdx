使用业务服务端签发的 token 登录指定用户。登录成功后才能可靠调用会话、消息、群组、好友等用户态接口。

## 功能介绍

使用业务服务端签发的 token 登录指定用户。登录成功后才能可靠调用会话、消息、群组、好友等用户态接口。

初始化类方法应在应用进程内按需调用；用户态方法通常要求 `InitSDK` 已成功执行，并通过 `operationID` 串联日志。

## Core Binding 方法

| 项目     | 说明                                                             |
| -------- | ---------------------------------------------------------------- |
| 方法签名 | `Login(callback Base, operationID string, userID, token string)` |
| 所属能力 | 初始化与登录                                                     |
| 源码位置 | `open_im_sdk/init_login.go`                                      |

## React Native 调用参考

React Native SDK v3.8.3-patch.12.4 的 TypeScript 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`src/sdk.ts`。

```ts showLineNumbers
login(params: LoginParams, operationID: string = id())
```

## 输入参数

| 参数名称 | 参数类型    | 是否必填 | 描述                                   |
| -------- | ----------- | -------- | -------------------------------------- |
| params   | LoginParams | 是       | 请求对象，字段以对应平台类型定义为准。 |

## 返回结果

返回 `Promise`。调用失败时 Promise reject，错误会被封装为 OpenIMApiError。

## 代码示例

以下示例展示 React Native SDK v3.8.3-patch.12.4 的调用骨架，请按业务上下文补充真实参数、错误处理和 UI 状态更新。

```ts showLineNumbers
import OpenIMSDK from '@openim/rn-client-sdk';

const result = await OpenIMSDK.login({});
```
