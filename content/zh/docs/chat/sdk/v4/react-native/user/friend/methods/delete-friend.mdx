删除指定好友关系，适合用户主动解除好友。

## 功能介绍

删除指定好友关系，适合用户主动解除好友。

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。

## Core Binding 方法

| 项目     | 说明                                                                   |
| -------- | ---------------------------------------------------------------------- |
| 方法签名 | `DeleteFriend(callback Base, operationID string, friendUserID string)` |
| 所属能力 | 好友                                                                   |
| 源码位置 | `open_im_sdk/relation.go`                                              |

## React Native 调用参考

React Native SDK v3.8.3-patch.12.4 的 TypeScript 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`src/sdk.ts`。

```ts showLineNumbers
deleteFriend(friendUserID: string, operationID: string = id())
```

## 输入参数

| 参数名称     | 参数类型 | 是否必填 | 描述                     |
| ------------ | -------- | -------- | ------------------------ |
| friendUserID | string   | 是       | 用户 ID 或用户 ID 列表。 |

## 返回结果

返回 `Promise`。调用失败时 Promise reject，错误会被封装为 OpenIMApiError。

## 代码示例

以下示例展示 React Native SDK v3.8.3-patch.12.4 的调用骨架，请按业务上下文补充真实参数、错误处理和 UI 状态更新。

```ts showLineNumbers
import OpenIMSDK from '@openim/rn-client-sdk';

const result = await OpenIMSDK.deleteFriend('friendUserID');
```
