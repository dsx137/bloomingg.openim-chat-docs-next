## 功能介绍

获取当前用户发出的好友申请列表，适合查看申请进度或历史申请。

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。

## Core Binding 方法

| 方法签名 | `GetFriendApplicationListAsApplicant(callback Base, operationID, req string)` |
| 所属能力 | 好友                                                                          |
| 源码位置 | `open_im_sdk/relation.go`                                                     |

返回 `Promise`。调用失败时 Promise reject，错误会被封装为 OpenIMApiError。
