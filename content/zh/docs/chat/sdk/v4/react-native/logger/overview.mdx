日志页面覆盖客户端主动写日志和上传 SDK 日志。离线推送和文件上传已拆分到独立模块。

## 使用边界

| 场景 | 说明 |
| ---- | ---- |
| 初始化日志配置 | 初始化 SDK 时配置日志级别和日志目录。 |
| 写入客户端日志 | 业务层可把关键 IM 操作写入 SDK 日志链路，便于排查问题。 |
| 上传日志 | 用户反馈或线上问题定位时上传 SDK 日志，并用 operationID 串联服务端日志。 |
