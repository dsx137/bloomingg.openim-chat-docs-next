删除指定会话及其消息，适合彻底移除某个会话记录。

## 功能介绍

删除指定会话及其消息，适合彻底移除某个会话记录。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## Core Binding 方法

| 项目     | 说明                                                                                          |
| -------- | --------------------------------------------------------------------------------------------- |
| 方法签名 | `DeleteConversationAndDeleteAllMsg(callback Base, operationID string, conversationID string)` |
| 所属能力 | 会话与消息                                                                                    |
| 源码位置 | `open_im_sdk/conversation_msg.go`                                                             |

## React Native 调用参考

React Native SDK v3.8.3-patch.12.4 的 TypeScript 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`src/sdk.ts`。

```ts showLineNumbers
deleteConversationAndDeleteAllMsg(conversationID: string, operationID: string = id())
```

## 输入参数

| 参数名称       | 参数类型 | 是否必填 | 描述                     |
| -------------- | -------- | -------- | ------------------------ |
| conversationID | string   | 是       | 会话 ID 或会话 ID 列表。 |

## 返回结果

返回 `Promise`。调用失败时 Promise reject，错误会被封装为 OpenIMApiError。
