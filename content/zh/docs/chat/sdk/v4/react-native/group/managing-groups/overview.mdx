管理群组用于创建群组、维护群资料、查询群资料、搜索群组、转让群主和解散群组。

## 方法索引

| 方法                                                                                               | 说明                                                                   |
| -------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| [创建群组](/docs/chat/sdk/v4/react-native/group/managing-groups/create-group)                      | 创建一个 OpenIM 群组。适合发起群聊、创建项目群或从业务组织关系生成群。 |
| [更新群资料](/docs/chat/sdk/v4/react-native/group/managing-groups/set-group-info)                  | 更新群资料，例如群名称、头像、公告或扩展字段。                         |
| [获取指定群组资料](/docs/chat/sdk/v4/react-native/group/managing-groups/get-specified-groups-info) | 批量获取指定群组资料，适合会话列表补齐群信息或刷新部分群资料。         |
| [搜索群组](/docs/chat/sdk/v4/react-native/group/managing-groups/search-groups)                     | 按关键字搜索群组，可按群 ID、群名称等字段搜索。                        |
| [转让群主](/docs/chat/sdk/v4/react-native/group/managing-groups/transfer-group-owner)              | 将群主转让给指定成员，通常只允许当前群主执行。                         |
| [解散群组](/docs/chat/sdk/v4/react-native/group/managing-groups/dismiss-group)                     | 解散指定群组，通常要求当前用户具备群主或管理权限。                     |

## 相关类型

- [群组信息](/docs/chat/sdk/v4/react-native/models/group-info)（`GroupInfo`）：群组资料对象，包含群名、公告、群主、成员数和入群设置。
