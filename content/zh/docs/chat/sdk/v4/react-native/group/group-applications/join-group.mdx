申请加入指定群组。是否直接入群取决于群设置和服务端审核策略。

## 功能介绍

申请加入指定群组。是否直接入群取决于群设置和服务端审核策略。

群组方法通常要求当前用户已登录，且群主、管理员、成员权限由服务端校验；群组变化通过 `SetGroupListener` 接收。

## Core Binding 方法

| 项目     | 说明                                                                                                       |
| -------- | ---------------------------------------------------------------------------------------------------------- |
| 方法签名 | `JoinGroup(callback Base, operationID string, groupID string, reqMsg string, joinSource int32, ex string)` |
| 所属能力 | 群组                                                                                                       |
| 源码位置 | `open_im_sdk/group.go`                                                                                     |

## React Native 调用参考

React Native SDK v3.8.3-patch.12.4 的 TypeScript 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

源码位置：`src/sdk.ts`。

```ts showLineNumbers
joinGroup(params: JoinGroupParams, operationID: string = id())
```

## 输入参数

| 参数名称 | 参数类型        | 是否必填 | 描述                                   |
| -------- | --------------- | -------- | -------------------------------------- |
| params   | JoinGroupParams | 是       | 请求对象，字段以对应平台类型定义为准。 |

## 返回结果

返回 `Promise`。调用失败时 Promise reject，错误会被封装为 OpenIMApiError。
