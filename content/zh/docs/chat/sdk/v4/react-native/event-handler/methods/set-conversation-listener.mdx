设置会话事件监听器，用于接收服务端同步进度、新会话、会话变更、总未读数和输入状态变化。

## 功能介绍

设置会话事件监听器，用于接收服务端同步进度、新会话、会话变更、总未读数和输入状态变化。

`Set*Listener` 会替换当前 SDK 保存的同类监听器；如果多个模块都要消费事件，应在业务层封装统一分发。

## Core Binding 方法

| 项目     | 说明                                                       |
| -------- | ---------------------------------------------------------- |
| 方法签名 | `SetConversationListener(listener OnConversationListener)` |
| 所属能力 | 监听器                                                     |
| 源码位置 | `open_im_sdk/listener.go`                                  |

## React Native 调用参考

React Native SDK v3.8.3-patch.12.4 的 TypeScript 方法原型如下。Core binding 方法签名用于说明底层能力和参数映射。

```ts showLineNumbers
setConversationListener(/* ... */);
```

| 参数名称 | 参数类型   | 是否必填   | 描述                 |

## 返回结果

返回 `Promise`。调用失败时 Promise reject，错误会被封装为 OpenIMApiError。
