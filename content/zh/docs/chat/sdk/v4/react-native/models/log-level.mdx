`LogLevel` 是 React Native SDK 的枚举常量，完整类名为 `io.openim.react-native.sdk.enums.LogLevel`。SDK 日志等级常量。

## 取值说明

| 常量         | 值  | 描述                           |
| ------------ | --- | ------------------------------ |
| Fatal        | 0   | Fatal 对应的 SDK 常量。        |
| Panic        | 1   | Panic 对应的 SDK 常量。        |
| Error        | 2   | Error 对应的 SDK 常量。        |
| Warn         | 3   | Warn 对应的 SDK 常量。         |
| Info         | 4   | Info 对应的 SDK 常量。         |
| Debug        | 5   | Debug 对应的 SDK 常量。        |
| DebugWithSQL | 6   | DebugWithSQL 对应的 SDK 常量。 |
