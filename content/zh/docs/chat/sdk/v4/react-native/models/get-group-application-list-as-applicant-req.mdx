`GetGroupApplicationListAsApplicantReq` 是 React Native SDK 的数据模型，完整类名为 `io.openim.react-native.sdk.models.GetGroupApplicationListAsApplicantReq`。查询当前用户发出的入群申请时使用的分页和过滤条件。

## 字段说明

| 字段名称      | TypeScript 类型 | 描述                   |
| ------------- | --------------- | ---------------------- |
| offset        | int             | 分页偏移量。           |
| count         | int             | 分页数量。             |
| groupIDs      | String[]        | SDK 返回或提交的字段。 |
| handleResults | int[]           | SDK 返回或提交的字段。 |
