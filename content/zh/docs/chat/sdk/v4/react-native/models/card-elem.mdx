`CardElem` 是 React Native SDK 的数据模型，完整类名为 `io.openim.react-native.sdk.models.CardElem`。个人名片消息元素。

## 字段说明

| 字段名称 | TypeScript 类型 | 描述                   |
| -------- | --------------- | ---------------------- |
| userID   | String          | SDK 返回或提交的字段。 |
| nickname | String          | SDK 返回或提交的字段。 |
| faceURL  | String          | SDK 返回或提交的字段。 |
| ex       | String          | 扩展字段。             |
