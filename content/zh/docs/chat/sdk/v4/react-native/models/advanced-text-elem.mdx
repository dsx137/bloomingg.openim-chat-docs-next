`AdvancedTextElem` 是 React Native SDK 的数据模型，完整类名为 `io.openim.react-native.sdk.models.AdvancedTextElem`。高级文本消息的内容元素，用于描述富文本正文和片段。

## 字段说明

| 字段名称          | TypeScript 类型     | 描述                   |
| ----------------- | ------------------- | ---------------------- |
| text              | String              | SDK 返回或提交的字段。 |
| messageEntityList | List<MessageEntity> | SDK 返回或提交的字段。 |

## 关联类型

- [高级消息片段](/docs/chat/sdk/v4/react-native/models/message-entity)（`MessageEntity`）：富文本消息中的片段信息。
