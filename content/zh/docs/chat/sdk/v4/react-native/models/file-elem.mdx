`FileElem` 是 React Native SDK 的数据模型，完整类名为 `io.openim.react-native.sdk.models.FileElem`。文件消息元素，用于本地文件消息或 URL 文件消息。

## 字段说明

| 字段名称  | TypeScript 类型 | 描述             |
| --------- | --------------- | ---------------- |
| filePath  | String          | 文件本地资源地址 |
| uuid      | String          | id               |
| sourceUrl | String          | oss地址          |
| fileName  | String          | 文件名称         |
| fileSize  | long            | 文件大小         |
