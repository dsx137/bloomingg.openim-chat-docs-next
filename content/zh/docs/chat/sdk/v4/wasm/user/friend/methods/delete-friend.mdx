删除指定好友关系，适合用户主动解除好友。

## 功能介绍

删除指定好友关系，适合用户主动解除好友。

好友关系变化通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；涉及列表和请求对象的字符串参数应传入对应结构的 JSON。

## 底层能力

| 项目     | 说明                                                                   |
| -------- | ---------------------------------------------------------------------- |
| 方法签名 | `DeleteFriend(callback Base, operationID string, friendUserID string)` |
| 所属能力 | 好友                                                                   |
| 源码位置 | `open_im_sdk/relation.go`                                              |

## WASM 调用参考

WASM SDK v3.8.3 暴露以下平台方法原型。底层能力表用于说明 OpenIM core 能力和参数映射。

源码位置：`src/sdk/index.ts`。

```ts showLineNumbers
deleteFriend(data: string, operationID = uuidv4())
```

## 输入参数

| 参数名称    | 参数类型   | 是否必填 | 描述                               |
| ----------- | ---------- | -------- | ---------------------------------- |
| data        | string     | 是       | 按平台类型定义传入。               |
| operationID | As defined | 否       | 链路追踪 ID；不传时 SDK 自动生成。 |

## 返回结果

返回 `Promise` 或注册回调结果。调用失败时 Promise reject；建议在业务层使用 `try/catch` 或 `.catch()` 处理错误。
