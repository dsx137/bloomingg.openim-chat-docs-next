获取当前登录用户自己的资料，适合进入个人资料页、启动后刷新当前用户信息或账号切换后校验本地缓存。

## 功能介绍

获取当前登录用户自己的资料，适合进入个人资料页、启动后刷新当前用户信息或账号切换后校验本地缓存。

用户资料和在线状态方法通常要求当前用户已登录；在线状态变化通过 `SetUserListener` 设置的监听器接收。

## 底层能力

| 项目     | 说明                                                 |

| 方法签名 | `GetSelfUserInfo(callback Base, operationID string)` |

## WASM 调用参考

WASM SDK v3.8.3 暴露以下平台方法原型。底层能力表用于说明 OpenIM core 能力和参数映射。

源码位置：`src/sdk/index.ts`。

```ts showLineNumbers
getSelfUserInfo((operationID = uuidv4()));
```

## 输入参数

| 参数名称    | 参数类型   | 是否必填 | 描述                               |
| ----------- | ---------- | -------- | ---------------------------------- |
| operationID | As defined | 否       | 链路追踪 ID；不传时 SDK 自动生成。 |

## 返回结果

返回 `Promise` 或注册回调结果。调用失败时 Promise reject；建议在业务层使用 `try/catch` 或 `.catch()` 处理错误。
