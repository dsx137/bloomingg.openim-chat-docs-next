## 功能介绍

将指定用户加入黑名单，适合屏蔽对方消息或限制关系互动。具体业务影响由 SDK 和服务端策略决定。

黑名单新增和移除通过 `SetFriendListener` 设置的 `OnFriendshipListener` 接收；屏蔽后的业务效果以当前服务端策略为准。

| 源码位置 | `open_im_sdk/relation.go`                                                    |

## WASM 调用参考

WASM SDK v3.8.3 暴露以下平台方法原型。底层能力表用于说明 OpenIM core 能力和参数映射。

源码位置：`src/sdk/index.ts`。

```ts showLineNumbers
addBlack(data: AddBlackParams, operationID = uuidv4())
```

## 输入参数

| 参数名称    | 参数类型       | 是否必填 | 描述                               |
| ----------- | -------------- | -------- | ---------------------------------- |
| data        | AddBlackParams | 是       | 按平台类型定义传入。               |
| operationID | As defined     | 否       | 链路追踪 ID；不传时 SDK 自动生成。 |

## 返回结果

返回 `Promise` 或注册回调结果。调用失败时 Promise reject；建议在业务层使用 `try/catch` 或 `.catch()` 处理错误。
