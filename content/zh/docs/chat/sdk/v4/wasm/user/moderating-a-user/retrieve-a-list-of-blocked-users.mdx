OpenIM 的黑名单是当前登录用户视角下的联系人关系。调用 `getBlackList()` 可以获取当前用户已经拉黑的用户，返回项使用 `BlackUserItem` 表示，适合黑名单设置页、用户资料卡关系状态和聊天入口限制。

黑名单不是群组管理状态。群成员禁言、踢出或角色调整应使用群成员相关 API；`getBlackList()` 只回答“当前用户拉黑了哪些用户”。

## 获取黑名单用户

完成 SDK 初始化和 `login()` 后，调用 `getBlackList()` 读取当前用户的黑名单。空数组表示当前没有已拉黑用户，不应当作请求失败。

```ts
async function loadBlockedUsers() {
  const operationID = crypto.randomUUID();
  const { data, errCode, errMsg } = await OpenIM.getBlackList(operationID);

  if (errCode !== 0) {
    console.error('getBlackList failed', {
      errCode,
      errMsg,
      operationID,
    });
    throw new Error(errMsg || 'Failed to load blocked users');
  }

  return data;
}

const blockedUsers = await loadBlockedUsers();
replaceBlockedUsers(blockedUsers);
```

资料卡、会话操作菜单或联系人列表通常只需要判断某个 `userID` 是否在黑名单中。建议用 `userID` 建立集合，不要用昵称、头像或备注判断关系状态。

```ts
const blockedUserIDs = new Set(blockedUsers.map((user) => user.userID));

function isBlocked(userID: string) {
  return blockedUserIDs.has(userID);
}
```

## 黑名单用户属性

`getBlackList()` 返回 `BlackUserItem[]`。渲染列表时优先使用 `userID` 作为稳定 key，其他字段只用于展示。

| 字段       | 说明                                     |
| ---------- | ---------------------------------------- |
| `userID`   | 被当前用户拉黑的目标用户 ID。            |
| `nickname` | 目标用户昵称，用于列表展示。             |
| `faceURL`  | 目标用户头像地址。                       |
| `gender`   | 目标用户性别字段，按你的产品规则展示。   |
| `ex`       | 扩展字段，只解析你的业务已经约定的内容。 |

如果黑名单页同时展示公开资料或好友备注，需要按 `userID` 合并多份数据，并清楚区分 `BlackUserItem`、`FriendUserItem` 和 `PublicUserItem` 的来源。

## 同步黑名单变化

黑名单可能在当前页面操作后变化，也可能由同一账号的其他已登录端触发。需要保持关系列表同步时，监听 `CbEvents.OnBlackAdded` 和 `CbEvents.OnBlackDeleted`。

```ts
function handleBlackAdded({ data }) {
  upsertBlockedUser(data);
}

function handleBlackDeleted({ data }) {
  removeBlockedUser(data.userID);
}

OpenIM.on(CbEvents.OnBlackAdded, handleBlackAdded);
OpenIM.on(CbEvents.OnBlackDeleted, handleBlackDeleted);

// 页面卸载、退出登录或切换账号时移除。
OpenIM.off(CbEvents.OnBlackAdded, handleBlackAdded);
OpenIM.off(CbEvents.OnBlackDeleted, handleBlackDeleted);
```

跨页面、刷新浏览器、断线重连或怀疑本地缓存不一致时，重新调用 `getBlackList()`，用完整结果替换本地黑名单缓存。拉黑和取消拉黑操作见 [拉黑和取消拉黑其他成员](/docs/chat/sdk/v4/wasm/user/moderating-a-user/block-and-unblock-other-members)。
