`UploadLogProgress` 是 WASM SDK 的回调接口，完整类名为 `open_im_sdk_callback.UploadLogProgress`。日志上传进度回调接口。

## 回调方法

| 方法       | 参数                      | 描述                                                        |
| ---------- | ------------------------- | ----------------------------------------------------------- |
| onProgress | current: long；size: long | 日志上传进度，`current` 表示已上传大小，`size` 表示总大小。 |
