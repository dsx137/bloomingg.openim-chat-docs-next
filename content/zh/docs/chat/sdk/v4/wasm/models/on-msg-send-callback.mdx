`OnMsgSendCallback` 是 WASM SDK 的回调接口，完整类名为 `io.openim.react-native.sdk.listener.OnMsgSendCallback`。发送消息时使用的进度、成功和失败回调。

## 类型说明

消息发送监听

## 回调方法

| 方法       | 参数                   | 描述     |
| ---------- | ---------------------- | -------- |
| catch      | int code；String error | 发送失败 |
| onProgress | long progress          | 上传进度 |
| then       | Message s              | 发送成功 |

## 关联类型

- [通用回调](/docs/chat/sdk/v4/wasm/models/on-base)（`Promise`）：WASM SDK 异步方法的通用成功和失败回调。
