`FriendshipInfo` 是 WASM SDK 的数据模型，完整类名为 `io.openim.react-native.sdk.models.FriendshipInfo`。好友关系校验结果。

| -------- | --------------- | --------------------------- |
| userID   | String          | 用户id                      |
| result   | int             | 1表示好友（并且不是黑名单） |
