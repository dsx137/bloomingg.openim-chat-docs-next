`SendMsgCallBack` 是 WASM SDK 文档中展示 OpenIM core 方法签名时使用的回调类型，完整类名为 `open_im_sdk_callback.SendMsgCallBack`。OpenIM core 消息发送回调，WASM 调用层通常映射为 OnMsgSendCallback。

## 回调方法

| 方法       | 参数                           | 描述                                   |
| ---------- | ------------------------------ | -------------------------------------- |
| OnError    | errCode: int32；errMsg: string | 发送失败时返回错误码和错误信息。       |
| OnSuccess  | data: string                   | 发送成功时返回服务端确认后的消息数据。 |
| OnProgress | progress: int                  | 发送或上传过程中的进度回调。           |
