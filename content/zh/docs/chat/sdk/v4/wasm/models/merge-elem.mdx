`MergeElem` 是 WASM SDK 的数据模型，完整类名为 `io.openim.react-native.sdk.models.MergeElem`。合并消息元素，用于多条消息合并转发。

## 字段说明

| 字段名称     | TypeScript 类型 | 描述                   |
| ------------ | --------------- | ---------------------- |
| title        | String          | 标题                   |
| abstractList | List<String>    | 摘要                   |
| multiMessage | List<Message>   | 具体选择合并的消息列表 |

## 关联类型

- [消息对象](/docs/chat/sdk/v4/wasm/models/message)（`Message`）：消息对象，包含消息 ID、发送者、会话、内容元素和状态等信息。
