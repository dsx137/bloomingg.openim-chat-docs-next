`Base` 是 WASM SDK 文档中展示 OpenIM core 方法签名时使用的回调类型，完整类名为 `open_im_sdk_callback.Base`。OpenIM core 异步结果回调，WASM 调用层通常映射为 Promise。

## 回调方法

| 方法      | 参数                           | 描述                             |
| --------- | ------------------------------ | -------------------------------- |
| OnError   | errCode: int32；errMsg: string | 操作失败时返回错误码和错误信息。 |
| OnSuccess | data: string                   | 操作成功时返回字符串数据。       |
