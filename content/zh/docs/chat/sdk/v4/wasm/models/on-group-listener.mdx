`OnGroupListener` 是 WASM SDK 的监听器，完整类名为 `io.openim.react-native.sdk.listener.OnGroupListener`。群组、入群申请和群成员变化监听器。

## 类型说明

群组监听

## 回调方法

| 方法                       | 参数                      | 描述             |
| -------------------------- | ------------------------- | ---------------- |
| onGroupApplicationAccepted | GroupApplicationInfo info | 事件触发时回调。 |
| onGroupApplicationAdded    | GroupApplicationInfo info | 事件触发时回调。 |
| onGroupApplicationDeleted  | GroupApplicationInfo info | 事件触发时回调。 |
| onGroupApplicationRejected | GroupApplicationInfo info | 事件触发时回调。 |
| onGroupDismissed           | GroupInfo info            | 事件触发时回调。 |
| onGroupInfoChanged         | GroupInfo info            | 事件触发时回调。 |
| onGroupMemberAdded         | GroupMembersInfo info     | 事件触发时回调。 |
| onGroupMemberDeleted       | GroupMembersInfo info     | 事件触发时回调。 |
| onGroupMemberInfoChanged   | GroupMembersInfo info     | 事件触发时回调。 |
| onJoinedGroupAdded         | GroupInfo info            | 事件触发时回调。 |
| onJoinedGroupDeleted       | GroupInfo info            | 事件触发时回调。 |

## 关联类型

- [群成员信息](/docs/chat/sdk/v4/wasm/models/group-members-info)（`GroupMembersInfo`）：群成员资料，包含群内昵称、角色、入群来源和禁言时间。
- [群申请信息](/docs/chat/sdk/v4/wasm/models/group-application-info)（`GroupApplicationInfo`）：入群申请记录，包含群资料、申请人、处理状态和来源。
- [群组信息](/docs/chat/sdk/v4/wasm/models/group-info)（`GroupInfo`）：群组资料对象，包含群名、公告、群主、成员数和入群设置。
