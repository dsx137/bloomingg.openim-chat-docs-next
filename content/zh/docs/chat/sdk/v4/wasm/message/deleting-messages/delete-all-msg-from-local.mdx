仅删除当前用户本地全部消息数据，适合本地缓存清理。

## 功能介绍

仅删除当前用户本地全部消息数据，适合本地缓存清理。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## 底层能力

| 项目     | 说明                                                       |
| -------- | ---------------------------------------------------------- |
| 方法签名 | `DeleteAllMsgFromLocal(callback Base, operationID string)` |
| 所属能力 | 会话与消息                                                 |
| 源码位置 | `open_im_sdk/conversation_msg.go`                          |
