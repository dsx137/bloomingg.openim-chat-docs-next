OpenIM WASM SDK 不提供把消息直接复制到目标频道的单一 `copyMessage()` 方法。复制消息时，先基于已有 `MessageItem` 创建一条新的转发消息，再调用 `sendMessage()` 发送到目标单聊或群聊。复制后的消息会生成新的消息 ID 和发送状态，不会修改原消息。

## 用户消息

对于文本、引用、表情等普通用户消息，使用 `createForwardMessage()` 创建转发消息。发送到单聊时传入目标用户 ID，并将 `groupID` 设为空字符串。

```ts
const operationID = crypto.randomUUID();

const {
  data: forwardMessage,
  errCode,
  errMsg,
} = await OpenIM.createForwardMessage(sourceMessage, operationID);

if (errCode !== 0) {
  throw new Error(errMsg || 'Failed to create forward message');
}

const { data: sentMessage } = await OpenIM.sendMessage(
  {
    recvID: targetUserID,
    groupID: '',
    message: forwardMessage,
  },
  operationID,
);

appendMessage(sentMessage);
```

## 文件消息

文件消息同样可以作为 `createForwardMessage()` 的输入。转发文件消息时不要复用原消息的 `clientMsgID`，也不要把原消息对象直接插入目标会话；应使用 SDK 返回的新 `MessageItem`。

```ts
const { data: forwardFileMessage } = await OpenIM.createForwardMessage(sourceFileMessage);

await OpenIM.sendMessage({
  recvID: '',
  groupID: targetGroupID,
  message: forwardFileMessage,
});
```

如果产品需要重新上传本地文件，请使用发送文件消息的流程；复制已有文件消息适用于把已经存在于消息列表或历史查询结果中的 `MessageItem` 转发到另一个会话。

## 多条消息

多选转发或聊天记录分享可以使用 `createMergerMessage()`。`messageList` 是要合并的原始消息列表，`title` 是合并消息标题，`summaryList` 用于展示合并卡片摘要。

```ts
const summaryList = selectedMessages.map((message) => message.content || '[消息]');

const {
  data: mergerMessage,
  errCode,
  errMsg,
} = await OpenIM.createMergerMessage({
  messageList: selectedMessages,
  title: '聊天记录',
  summaryList,
});

if (errCode !== 0) {
  throw new Error(errMsg || 'Failed to create merger message');
}

await OpenIM.sendMessage({
  recvID: '',
  groupID: targetGroupID,
  message: mergerMessage,
});
```

## 参数列表

| 参数                | 类型            | 说明                                                         |
| ------------------- | --------------- | ------------------------------------------------------------ |
| `sourceMessage`     | `MessageItem`   | 要复制的原始消息，可来自消息列表、历史消息查询或新消息事件。 |
| `sourceFileMessage` | `MessageItem`   | 要复制的文件消息。                                           |
| `selectedMessages`  | `MessageItem[]` | 需要合并转发的多条消息。                                     |
| `targetUserID`      | `string`        | 单聊目标用户 ID。发送到群聊时传空字符串。                    |
| `targetGroupID`     | `string`        | 群聊目标群组 ID。发送到单聊时传空字符串。                    |
| `operationID`       | `string`        | 可选的调用追踪 ID，便于排查发送失败。                        |

`createForwardMessage()` 和 `createMergerMessage()` 只生成待发送消息；目标会话里的在线成员会在 `sendMessage()` 成功后通过 `CbEvents.OnRecvNewMessages` 收到新消息。失败时记录 `errCode`、`errMsg` 和 `operationID`，并确认当前用户具备目标群聊的发言权限。
