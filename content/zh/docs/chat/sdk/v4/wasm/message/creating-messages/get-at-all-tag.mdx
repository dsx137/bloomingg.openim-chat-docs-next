获取群聊 @ 所有人使用的特殊标记。发送 @ 所有人消息时，应使用该标记而不是业务侧随意拼接字符串。

## 功能介绍

获取群聊 @ 所有人使用的特殊标记。发送 @ 所有人消息时，应使用该标记而不是业务侧随意拼接字符串。

创建消息类方法同步返回消息 JSON；发送、查询、删除、已读和本地写入方法通过回调返回。消息与会话事件分别通过消息监听器和会话监听器接收。

## 底层能力

| 项目     | 说明                                     |
| -------- | ---------------------------------------- |
| 方法签名 | `GetAtAllTag(operationID string) string` |
| 所属能力 | 会话与消息                               |
| 源码位置 | `open_im_sdk/conversation_msg.go`        |
