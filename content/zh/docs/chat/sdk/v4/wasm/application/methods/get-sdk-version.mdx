## 功能介绍

获取当前 core SDK 的版本号，适合在启动日志、问题反馈和兼容性判断中记录。

初始化类方法应在应用进程内按需调用；用户态方法通常要求 `InitSDK` 已成功执行，并通过 `operationID` 串联日志。

## 底层能力

| 项目     | 说明                        |
| -------- | --------------------------- |
| 方法签名 | `GetSdkVersion() string`    |
| 所属能力 | SDK 生命周期                |
| 源码位置 | `open_im_sdk/init_login.go` |

## WASM 调用参考

WASM SDK v3.8.3 当前未暴露 `getSdkVersion` 这个独立平台方法。此页保留底层能力说明，避免把其他平台或旧封装层的调用方式误写成当前 WASM API。

## 输入参数

当前平台未暴露独立方法，因此没有可调用的平台参数。

## 返回结果

当前平台未暴露独立方法，因此没有平台返回值。
