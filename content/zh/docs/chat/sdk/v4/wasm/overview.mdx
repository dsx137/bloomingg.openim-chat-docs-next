OpenIM WASM SDK 用于在浏览器应用中接入 OpenIM Chat。浏览器包由 `@openim/wasm-client-sdk` 提供，依赖 `openIM.wasm`、`sql-wasm.wasm` 和 `wasm_exec.js`，并通过 `getSDK()` 创建一个在当前浏览器会话中复用的 SDK 实例。

从这里开始时，通常先完成 SDK 安装、WASM 资源发布、用户登录和第一条消息发送，再按产品功能进入消息、群组、用户、事件和诊断页面。
