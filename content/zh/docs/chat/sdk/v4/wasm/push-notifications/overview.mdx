| 方法 | 说明 |
| ---- | ---- |
| [更新 FCM Token](/docs/chat/sdk/v4/wasm/push-notifications/methods/update-fcm-token) | 更新当前设备的 FCM token，用于服务端向设备发送离线推送。 |
| [设置应用角标](/docs/chat/sdk/v4/wasm/push-notifications/methods/set-app-badge) | 设置应用角标未读数，适合未读数变化后同步系统角标。 |
