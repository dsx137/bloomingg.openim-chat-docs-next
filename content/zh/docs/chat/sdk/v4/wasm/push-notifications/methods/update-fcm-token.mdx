更新当前设备的 FCM token，用于服务端向 WASM 设备发送离线推送。

## 功能介绍

更新当前设备的 FCM token，用于服务端向 WASM 设备发送离线推送。

第三方能力至少要求 SDK 已初始化；日志和文件上传的进度通过对应 progress callback 返回。

## 底层能力

| 项目     | 说明                                                                            |
| -------- | ------------------------------------------------------------------------------- |
| 方法签名 | `UpdateFcmToken(callback Base, operationID, fcmToken string, expireTime int64)` |
| 所属能力 | 推送通知                                                                |
| 源码位置 | `open_im_sdk/third.go`                                                          |
